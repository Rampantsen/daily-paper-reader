## 摘要
三维高斯溅射（3D-GS）能够实现实时三维场景重建，但缺乏用于对象移除、提取和重新着色等编辑任务的鲁棒分割。现有将二维分割提升到三维域的方法存在视图不一致和粗糙掩码的问题。在本文中，我们提出了一种新颖的框架，利用高质量分割任意模型（SAM-HQ）生成精确的二维掩码，解决了标准SAM在边界保真度和精细结构保留方面的局限性。为了对给定场景中的任意目标对象实现鲁棒的三维分割，我们引入了一种先验引导的标签重分配方法，通过强制多视图一致性和学习到的先验，为三维高斯分配标签。我们的方法实现了最先进的分割精度，并支持交互式实时对象编辑，同时保持高视觉保真度。定性结果表明，在虚拟现实（VR）和机器人技术中，具有优越的边界保留性和实际应用价值，推动了三维场景编辑的发展。

## Abstract
3D Gaussian Splatting (3D-GS) enables real-time 3D scene reconstruction but lacks robust segmentation for editing tasks such as object removal, extraction, and recoloring. Existing approaches that lift 2D segmentations to the 3D domain suffer from view inconsistencies and coarse masks. In this paper, we propose a novel framework that leverages the Segment Anything Model High Quality (SAM-HQ) to generate accurate 2D masks, addressing the limitations of the standard SAM in boundary fidelity and fine-structure preservation. To achieve robust 3D segmentation of any target object in a given scene, we introduce a prior-guided label reassignment method that assigns labels to 3D Gaussians by enforcing multiview consistency with learned priors. Our approach achieves state-of-the-art segmentation accuracy and enables interactive, real-time object editing while maintaining high visual fidelity. Qualitative results demonstrate superior boundary preservation and practical utility in Virtual Reality (VR) and robotics, advancing 3D scene editing.

*   **研究动机与背景**

    *   本研究的核心问题是：**标准 3D-GS 缺乏原生的、鲁棒的对象级语义分割能力**。这意味着用户无法直接对场景中的特定物体（如移除一把椅子、更改一个玩具的颜色）进行交互式编辑。
    *   现有主流方法试图将从二维图像（如 SAM 模型）中获取的分割掩码“提升”到三维高斯基元上。但这些方法普遍存在两大缺陷：

        2.  **启发式偏差**：在将二维掩码信息分配给三维高斯时，许多方法依赖于手工设定的背景偏差等启发式规则，这在处理复杂遮挡和精细结构时鲁棒性差。

*   **整体含义**
    *   本文旨在解决上述核心痛点，构建一个**可编辑的 3D-GS 框架**。其核心意义在于，通过提供一种**鲁棒、高精度的三维场景分割方法**，让用户能够通过简单的点击交互，就能对 3D-GS 重建出的场景进行实时、视觉上无缝的对象移除、提取和重新着色等编辑操作。

### 2. 论文提出的方法论

该方法的核心思想是：“高质量二维监督 + 三维特征联合学习 + 先验引导的最优标签重分配”。

*   **整体框架流程**
    1.  **A. 高质量多视角掩码生成**：对输入的多视角图像，使用 SAM-HQ 生成精细掩码，并通过 DEVA（解耦视频分割）保证多视角标签一致性。
    2.  **B. 三维特征联合训练**：在训练 3D-GS 的同时，为每个高斯球学习一个与视角无关的 16 维物体特征向量，并使用步骤 A 生成的掩码作为监督信号。
    3.  **C. 先验引导的标签重分配（核心创新）**：针对训练后仍可能存在的噪声标签，提出一种结合“多视角投票”与“学习先验”的线性规划方法，为每个高斯球最终分配一个最优标签。

*   **关键技术细节**
    *   **高质量掩码预处理 (Section 2.1)**
        *   使用 **SAM-HQ** 替代标准 SAM，以获得更清晰的对象边界和更精细的结构。
        *   使用 **DEVA** 模型，将多视角图像序列视为视频流，通过双向时间传播来统一不同图像上同一对象的 ID 标签，解决跨视图一致性问题。
        *   应用 **噪声滤除** 后处理：通过形态学闭运算平滑边缘，利用连通分量分析移除面积小于阈值（默认 500 像素）的碎片化区域，并将其重新分配给周围占主导的标签。
    *   **物体特征驱动的 3D-GS (Section 2.2)**
        *   **特征增强**：为每个 3D 高斯球增加一个 16 维的、视角不变（球谐系数度数设为 0）的特征向量 `f_o`。
        *   **可微分渲染**：使用与 RGB 颜色相同的 α-混合渲染管线，将高维特征向量 `f_o` 渲染到像素平面，得到像素级特征 `F_o(u)`。
        *   **分类与监督**：将渲染出的像素特征 `F_o` 通过一个可训练的线性层 `φ` 映射为 256 类的 Logits，再经 Softmax 得到每个像素属于各物体 ID 的概率。使用**交叉熵损失 `L_obj`**，以预处理得到的高质量掩码为真值，监督特征和线性层的训练。总损失 `L_total = L_rgb + γ_obj * L_obj`。
    *   **先验引导的标签重分配 (Section 2.3)**
        *   **问题建模**：将此步骤转化为一个线性规划问题，旨在为每个高斯球找到最佳标签。
        *   **核心公式**：重新定义了高斯球 `i` 对标签 `l` 的贡献分数 `A^new_l,i`：
            `A^new_l,i = A_l,i + γ_p · p_{i,l_i} · I{l_i = l}`
            *   `A_l,i` 是传统多数投票的贡献分，基于高斯球在所有训练视角下，投影到标签为 `l` 的掩码像素上的 α 值与透光率累积。
            *   `p_{i,l_i}` 是**先验分数**，取自上一阶段训练好的 Softmax 层输出的概率，代表了网络学到的该高斯球属于其当前标签 `l_i`的置信度。
            *   `γ_p` 是控制先验影响力的权重系数。
        *   **最终分配**：最终的标签 `P^new_i = argmax_l (A^new_l,i)`。该方法通过加入可学习的、数据驱动的先验，消除了对启发式背景偏差的依赖，提升了多视角一致性和鲁棒性。

### 8. 不足与局限

*   **比较对象较少**：实验主要与同属最优水平的 FlashSplat 进行对比，缺乏与更多近期方法（如 Gaussian Grouping, SAGD 等）的横向比较，难以全面确立其在领域内的绝对领先地位。

*   **泛化性风险**：虽然在一个标准数据集（NVOS）上进行了测试，但其余所有实验主要基于其自建的、由 SAM-HQ 处理过的数据集。这种训练与测试数据同源的处理方式可能存在过拟合于特定掩码风格的风险，在其数据域之外的泛化性能有待更广泛的检验。
