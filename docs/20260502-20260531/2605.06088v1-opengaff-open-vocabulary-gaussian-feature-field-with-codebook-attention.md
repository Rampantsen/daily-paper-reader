## 摘要
由于多视角观测中语义预测的碎片化和空间不一致性，基于高斯表示的开放词汇三维场景理解仍然具有挑战性。本文提出了OpenGaFF，一个基于三维高斯泼溅的开放词汇三维场景理解新框架。我们方法的核心是高斯特征场，它将语义建模为高斯几何和外观的连续函数。通过显式地以几何结构为条件进行语义预测，这一形式强化了几何与语义之间的耦合，从而提高了三维空间中相似结构的空间一致性。为了进一步强化物体级别的语义一致性，我们引入了一个结构化码本，作为一组共享的语义基元。此外，我们提出了一种码本引导的注意力机制，通过查询嵌入与学习到的码本条目的相似性匹配来检索语言特征，实现了鲁棒的开放词汇推理，同时减少了物体内部的特征方差。在标准的二维和三维开放词汇基准上的大量实验表明，我们的方法持续优于先前的方法，获得了更优的分割质量、更强的三维语义一致性和一个具有语义可解释性的码本，为学习到的表示提供了见解。

## Abstract
Understanding open-vocabulary 3D scenes with Gaussian-based representations remains challenging due to fragmented and spatially inconsistent semantic predictions across multi-view observations. In this paper, we present OpenGaFF, a novel framework for open-vocabulary 3D scene understanding built upon 3D Gaussian Splatting. At the core of our method is a Gaussian Feature Field that models semantics as a continuous function of Gaussian geometry and appearance. By explicitly conditioning semantic predictions on geometric structure, this formulation strengthens the coupling between geometry and semantics, leading to improved spatial coherence across similar structures in 3D space. To further enforce object-level semantic consistency, we introduce a structured codebook that serves as a set of shared semantic primitives. Furthermore, a codebook-guided attention mechanism is proposed to retrieve language features via similarity matching between query embeddings and learned codebook entries, enabling robust open-vocabulary reasoning while reducing intra-object feature variance. Extensive experiments on standard 2D and 3D open-vocabulary benchmarks demonstrate that our method consistently outperforms prior approaches, achieving improved segmentation quality, stronger 3D semantic consistency and a semantically interpretable codebook that provides insight into the learned representation.

- **核心问题**：在基于 3D Gaussian Splatting (3DGS) 的开放词汇三维场景理解中，由于多视图观测下语义预测碎片化、空间不一致，导致难以获得既表达丰富又几何一致的语义表示。
- **研究动机**：现有方法大多在点或基元级别独立学习语义特征，未能将几何与语义有效耦合，造成跨视图语义断裂、物体内部特征方差大、对透明物体等语义归属不可靠。本文希望构建一个能将语义与几何结构紧密关联、并在物体级别保持语义一致性的框架，以提升三维开放词汇分割的完整性和鲁棒性。

## 2. 方法论

### 2.1 核心思想
- **高斯特征场（Gaussian Feature Field）**：不再为每个高斯独立分配可学习语义特征，而是用一个 MLP 将高斯的位置（经傅里叶编码）和颜色映射为低维语义特征 \( \mathbf{m}_i = \mathcal{F}(\varphi(\boldsymbol{\mu}_i), \mathbf{c}_i) \)。该做法隐式共享参数，促使几何相近的高斯拥有相近的语义，从而增强几何‑语义耦合。
- **结构化语言码本（Structured Language Codebook）**：对从 CLIP 提取的多视图语言特征进行聚类，自动确定码本大小 \(N_c\)，并用簇心初始化码本 \( \mathbf{S} \in \mathbb{R}^{N_c \times D} \)。码本作为共享语义基元，鼓励同一物体的区域映射到共同码本条目，减少类内特征方差。
- **码本引导注意力机制**：将渲染出的低维特征图 \( \hat{\mathbf{M}} \) 作为 Query，码本作为 Key/Value，通过缩放点积注意力检索高维语言特征 \( \hat{\mathbf{L}} \)。注意力权重反映 Query 与各码本条目之间的相似度，并可通过熵正则化鼓励每个 Query 聚焦单个条目，强化语义解耦与一致性。
- **分离的语义不透明度**：为每个高斯额外学习一个语义不透明度 \( \alpha_i^s \)，与外观不透明度解耦，专门用于语义渲染，以缓解透明物体等特殊外观对语义融合的不利影响。

### 2.2 算法流程（两阶段训练）
1. **预处理**：利用 SAM 提取分割掩码，CLIP 提取语言特征并重分配给每个像素的 2D 语言特征图；用 PCA 将高维语言特征降维得到低维特征图，用作第一阶段监督信号；通过聚类初始化结构化码本。
2. **阶段一**：联合优化 3DGS 外观分支和高斯特征场，使用光度损失和低维语义特征损失（余弦距离 + L1）训练。
3. **阶段二**：冻结高斯特征场，训练码本和注意力模块，仅使用语言特征损失（余弦距离 + L1）和监督熵正则化损失，使网络学习从低维特征到高维语言特征的检索映射。

- 高斯特征场通过将语义建模为位置与颜色的连续函数，显著提升了 3D 语义的空间一致性，使得分割结果更完整、几何更对齐。

## 7. 优点

- **几何‑语义强耦合**：高斯特征场利用空间连续函数显式关联几何与语义，优于逐高斯独立学习的方案。
- **解耦且可解释的语义码本**：数据驱动的码本初始化与熵正则化使每个条目聚焦于单一语义概念，便于理解和分析。
- **稳健的开放词汇推理**：基于注意力的特征检索能应对多视图特征的不一致，减少物体内部语义方差。
- **高效**：训练时间和显存需求低于 GALA 和 LangSplatV2，推理速度可达 35 fps。
- **统一的 2D/3D 框架**：同一方法在 2D 和 3D 任务上均能取得领先，避免了为不同维度单独设计模型。

## 8. 不足与局限

- **对稀疏出现物体的敏感性**：码本基于聚类初始化，若某个物体在训练视图中出现极少，其对应的语义可能在码本中得不到充分表示，导致 2D 小物体分割的 mIoU 略有下降（如 MipNeRF360 上的某些场景）。
- **熵正则化的平衡**：过大的熵系数会使码本过度特化，可能损害语义组合性，对罕见物体学习不利；系数选择需要权衡。
- **依赖预处理质量**：码本初始化和低维特征监督依赖于 SAM 和 CLIP 的提取质量，相关预处理误差会向上传递。
- **尚未探索动态场景**：本文仅在静态场景上验证，对于动态物体或实时变化的环境尚未测试。
- **3D 任务上的绝对 mIoU 仍有提升空间**：尽管相对提升显著，在 ScanNet-v2 19 类上的 mIoU 为 36.55%，仍有长尾类别和细粒度分割方面的挑战。
