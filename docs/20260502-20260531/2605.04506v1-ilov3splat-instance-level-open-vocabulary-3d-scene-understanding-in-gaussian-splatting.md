## 摘要
我们提出Ilov3Splat，一个基于3D高斯泼溅（3D-GS）的新型实例级开放词汇3D场景理解框架。现有方法大多依赖基于2D渲染的匹配或点级语义关联，这削弱了跨视图一致性，缺乏连贯的实例级推理，并限制了下游3D任务的精度。为解决这些局限，我们的方法通过为高斯泼溅体增加视图一致的特征场，联合优化场景几何与语义表示。具体来说，我们利用多分辨率哈希嵌入来高效编码语言对齐的CLIP特征，从而在3D空间中实现密集且连贯的语言基础。我们进一步使用基于SAM掩码的对比损失训练实例特征场，支持跨视图的细粒度对象区分。在推理时，CLIP编码的查询与学习到的特征进行匹配，然后通过两阶段的3D聚类来检索相关的高斯组。这使得我们的框架能够基于自然语言描述识别3D场景中的任意对象，无需类别监督或手动标注。在标准基准上的实验表明，Ilov3Splat在对象选择和实例分割方面均优于先前的开放词汇3D-GS方法，为语言驱动的3D场景理解提供了灵活且精确的解决方案。项目页面：https://csiro-robotics.github.io/Ilov3Splat。

## Abstract
We introduce Ilov3Splat, a novel framework for instance-level open-vocabulary 3D scene understanding built on 3D Gaussian Splatting (3D-GS). Most prior work depends on 2D rendering-based matching or point-level semantic association, which undermines cross-view consistency, lacks coherent instance-level reasoning, and limits precision in downstream 3D tasks. To address these limitations, our method jointly optimizes scene geometry and semantic representations by augmenting Gaussian splats with view-consistent feature fields. Specifically, we leverage multi-resolution hash embedding to efficiently encode language-aligned CLIP features, enabling dense and coherent language grounding in 3D space. We further train an instance feature field using contrastive loss over SAM masks, supporting fine-grained object distinction across views. At inference time, CLIP-encoded queries are matched against the learned features, followed by two-stage 3D clustering to retrieve relevant Gaussian groups. This enables our framework to identify arbitrary objects in 3D scenes based on natural language descriptions, without requiring category supervision or manual annotations. Experiments on standard benchmarks demonstrate that Ilov3Splat outperforms prior open-vocabulary 3D-GS methods in both object selection and instance segmentation, offering a flexible and accurate solution for language-driven 3D scene understanding. ### 论文总结

#### 1. 核心问题与研究动机
- 当前基于3D高斯泼溅（3D-GS）的开放词汇3D场景理解方法存在三个主要局限：
  - **缺乏跨视图一致性**：依赖2D渲染匹配或点级语义关联，无法保持同一物体在不同视角下的语义连贯性。
  - **缺乏实例级推理**：仅提供点级语义引用，无法明确分离同一类别下的多个物体实例，导致语义响应的背景泄漏或噪声激活。
  - **几何与语义解耦**：多数方法将几何重建和语义学习分成独立阶段，引发言语义漂移和对齐误差。
- 这些局限阻碍了下游任务（如目标导航、操作）所需的精确对象级推理。
- 本文旨在提出一个统一的、实例级、开放词汇的3D场景理解框架，直接基于3D高斯泼溅实现语言驱动的任意物体检索与分割，无需类别监督或手动标注。

#### 2. 方法论
- **核心思想**：为每个3D高斯泼溅体增加视图一致、可学习的语义属性，联合优化场景几何与语义特征场，实现实例感知的语言定位。
- **语言特征场**：
  - 利用多分辨率哈希编码（MHE）映射高斯中心位置 `x_i` 到紧凑语言嵌入 `f_lang_i`，再通过轻量MLP投影出CLIP对齐特征 `f_CLIP_i` 和标量 `scale_i`。
  - 通过α混合将3D特征渲染为2D语言特征图，用多尺度CLIP裁剪构建的伪真值进行Huber损失监督。
  - 实现密集而连贯的3D语言基础，避免几何-语义分离造成的错位。
- **实例特征场**：
  - 为每个高斯学习实例嵌入 `f_inst_i`，使用对比损失训练。
  - 对于每一视图的SAM实例掩码，计算掩码内渲染实例特征的平均代表特征，通过正样本损失拉近同实例特征，通过margin损失推远不同实例特征。
  - 总实例损失：`L_instance = L_pos + w_neg * L_neg`。
- **两阶段3D实例聚类**（训练后一次性执行）：
  - 第一阶段（语义聚类）：在实例特征空间用HDBSCAN聚类，保留与文本查询CLIP相似度超过阈值（结合预测 scale 值自适应调节）的语义簇。
  - 第二阶段（空间细化）：对每个语义簇内的高斯，按3D空间坐标执行DBSCAN，分割出空间连通子簇，保留所有有效子簇以区分同类不同实例，剔除噪声点。
- **联合训练目标**：总损失 `L_total = L_RGB + λ_lang L_CLIP + λ_inst L_instance + λ_reg L_DINO`，其中DINO正则化增强物体边界一致性。
- 设计亮点：无需场景特定预处理、固定码本或预先指定实例数量，实现了连续的实例特征学习和密度聚类解码。

#### 6. 主要结论与发现
- Ilov3Splat首次在3D-GS框架下实现了无需类别监督的实例级开放词汇对象识别与分割。

- 两阶段聚类在语义空间和3D空间结合，有效滤除噪声并保留同类多实例，在对象选择（mAcc提升，mIoU竞争）和实例分割（mAcc显著优于所有对比方法，mIoU接近使用深度的Sam3D）上均取得领先或极具竞争力的性能。

#### 7. 优点
- **统一框架**：将几何重建与开放词汇语义学习无缝集成，避免分离式训练导致的错位。
- **实例感知**：通过对比损失和两阶段聚类，首次在3D-GS中显式建模实例级连贯性，解决了同类多实例分离的难题。
- **灵活高效**：无需预定义类别、场景特定码本或深度输入，仅靠RGB和SAM掩码监督，兼容任意文本查询。
- **性能优越**：在多个基准上超越现有开放词汇3D-GS方法，尤其在mAcc指标上表现突出。

#### 8. 不足与局限
- **场景泛化性**：实验仅在室内场景（LERF、ScanNet）进行，未评估室外大规模、动态或稀疏点云场景。
