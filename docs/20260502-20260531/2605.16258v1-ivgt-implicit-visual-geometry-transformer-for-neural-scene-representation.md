## 摘要
从未定位的多视图图像重建连贯的3D几何和外观是计算机视觉中一个基本但具有挑战性的问题。大多数现有的视觉几何基础模型通过回归像素对齐的点图来预测显式几何，通常存在冗余和有限的几何连续性。我们提出了IVGT，一个隐式视觉几何Transformer，能够从无姿态的多视图图像中隐式建模连续且连贯的几何。该方案在规范坐标系中学习连续的神经场景表示，并支持在任何3D位置进行连续空间查询，利用轻量级解码器检索局部特征来预测带符号距离（SDF）值和颜色。它允许直接提取连续且连贯的表面几何，能够从任意视点渲染RGB图像、深度图和表面法线图。我们通过多数据集联合优化，结合2D监督和3D几何正则化来训练IVGT。IVGT展现了跨场景的泛化能力，并在多项任务上取得了强劲性能，包括网格和点云重建、新视图合成、深度和表面法线估计以及相机姿态估计。

## Abstract
Reconstructing coherent 3D geometry and appearance from unposed multi-view images is a fundamental yet challenging problem in computer vision. Most existing visual geometry foundation models predict explicit geometry by regressing pixel-aligned pointmaps, often suffering from redundancy and limited geometric continuity. We propose IVGT, an Implicit Visual Geometry Transformer that implicitly models continuous and coherent geometry from pose-free multi-view images. This formulation learns a continuous neural scene representation in a canonical coordinate system and supports continuous spatial queries at any 3D positions, retrieving local features to predict signed distance (SDF) values and colors using lightweight decoders. It allows direct extraction of continuous and coherent surface geometry, enabling rendering of RGB images, depth maps, and surface normal maps from arbitrary viewpoints. We train IVGT via multi-dataset joint optimization with 2D supervision and 3D geometric regularization. IVGT demonstrates generalization across scenes and achieves strong performance on various tasks, including mesh and point cloud reconstruction, novel view synthesis, depth and surface normal estimation, and camera pose estimation.

- **现有显式模型的局限**：当前主流视觉几何基础模型（如 DUSt3R、VGGT 等）回归**像素对齐的点图**，输出的是离散、视图索引的 3D 点集合。这种做法存在两个根本缺陷：
  - **表示离散**：仅在像素位置定义几何，无法查询任意 3D 位置，也难以直接提取连续表面。
  - **几何冗余与不一致**：同一物体点可能在多个视图中被重复预测，且不同视图的点图之间缺乏连续约束，导致几何连续性差。

## 二、论文提出的方法论

### 2.1 核心思想：隐式连续场景表示

IVGT 将无姿态多视图图像映射为一个全局的神经场景场 \(\mathbf{F}\)，该场可在**规范坐标系**中任意 3D 点 \(\mathbf{x}\) 查询，预测有符号距离（SDF）\(\hat{s}(\mathbf{x})\) 和视角相关颜色 \(\hat{c}(\mathbf{x}, \mathbf{v})\)。场景表面即 SDF 的零水平集，天然具有连续性和视图独立性，避免了显式点图的离散冗余。

### 2.2 模型架构与关键技术细节

1. **图像编码与全局特征提取**  
   - 使用 DINO 预训练模型将每张输入图像转换为 token。  
   - Transformer 骨干网络交替执行**帧内自注意力**和**跨帧自注意力**，隐式对齐多视图观测，无需显式输入相机姿态。  
   - 骨干网络同时预测辅助输出：每视图的深度图 \(D_i\) 和相机参数 \(g_i\)，为后续学习提供几何线索。

2. **连续 3D 查询与多视图特征聚合**  
   - 对于规范坐标系中任意点 \(\mathbf{x}\)，将其投影到所有有效视图，采样像素对齐特征并求和，得到图像特征 \(\mathbf{z}_f\)。  
   - 为避免同一射线上不同深度的点共享相同图像特征，引入**光线深度编码**：用轻量 MLP 编码该点在每视图的光线深度 \(d_i(\mathbf{x})\)，并聚合为 \(\mathbf{z}_d\)。  
   - 将 \(\mathbf{z}_f\) 与 \(\mathbf{z}_d\) 拼接作为该点的局部空间特征 \(\mathbf{z}\)。该设计利用视一致的光线深度替代绝对 3D 坐标编码，解决了参考帧依赖导致的歧义。

3. **级联解码器：SDF 与颜色**  
   - 几何解码器（8 层 MLP）：\((\hat{s}, \hat{\mathbf{z}}) = f_\theta(\mathbf{z})\)，输出 SDF 和中间外观特征。  
   - 外观解码器（2 层 MLP）：\(\hat{\mathbf{c}} = c_\theta(\hat{\mathbf{z}}, \hat{\mathbf{n}}, \gamma(\mathbf{v}))\)，其中 \(\hat{\mathbf{n}} = \nabla \hat{s}\) 为表面法向，\(\gamma(\mathbf{v})\) 为视角方向的 NeRF 式位置编码。

4. **可微渲染与表面提取**  
   - 使用 VolSDF 的 SDF‑to‑density 变换计算采样点密度，通过体积渲染积分得到每条射线的预测颜色 \(\hat{C}\)、深度 \(\hat{D}\) 和法向 \(\hat{N}\)。  
   - 推理时，先由输入视图的预测深度和相机姿态得到场景点云，确定空间包围盒；在该区域进行 64³ 粗采样筛选有效区域，上采样至 512³ 精查询 SDF，最终通过 Marching Cubes 提取连续网格，并沿法向方向解码顶点颜色。

### 2.3 两阶段训练策略

- **第一阶段**：仅使用 2D 监督，损失包括：  
  - 渲染 RGB 的 L1 损失 \(L_{\text{rgb}}\)  
  - 渲染深度的 L1 损失 \(L_{\text{depth-render}}\)  
  - 渲染法向的混合损失 \(L_{\text{normal}}\)（L1 + 角度一致）  
  - 相机参数 Huber 损失 \(L_{\text{camera}}\)  
  - 总损失：\(L_{\text{stage1}} = L_{\text{rgb}} + \lambda_1 L_{\text{depth-render}} + \lambda_2 L_{\text{normal}} + \lambda_3 L_{\text{camera}}\)

- **第二阶段**：在保留第一阶段全部损失的基础上，加入：  
  - Eikonal 正则项 \(L_{\text{eikonal}}\) 强制 SDF 梯度模长接近 1  
  - 光滑项 \(L_{\text{smooth}}\) 促进表面局部平滑  
  - 深度解码损失 \(L_{\text{depth}}\)（带预测不确定度，来自 VGGT 的设计）  
  - 总损失：\(L_{\text{stage2}} = L_{\text{stage1}} + \lambda_4 L_{\text{eikonal}} + \lambda_5 L_{\text{smooth}} + \lambda_6 L_{\text{depth}}\)

- **训练数据**：仅需 RGB‑D 序列和相机姿态，法向使用 DSine 从 RGB 预测获得，从而无需昂贵的人工标注。数据集涵盖 ARKitScenes、CO3Dv2、HyperSim、MegaDepth、OmniObject3D、ScanNet、ScanNet++、Unreal4K、WildRGBD 等，覆盖室内场景和物体、真实与合成数据。  
- **优化设置**：使用 AdamW，学习率余弦调度，最大 2e‑4；每迭代渲染 8 个视点（4 个上下文、4 个新视点），每条射线采样 1024 点，采用误差界采样策略。

## 六、论文的主要结论与发现

1. **隐式连续表示优于显式点图**：SDF 场天然保证表面连续、无冗余，且支持直接网格提取与新视图渲染，解决了现有显式模型离散、不连贯等问题。  
2. **一次前馈实现高质量表面重建**：IVGT 在 ScanNet 上的网格重建质量超越多数逐场景优化方法（如 NeuS、VolSDF），而无需任何测试时优化，显著提升了效率。  
3. **多任务统一性**：同一隐式表示可以端到端地支持 RGB 渲染、深度估计、法线估计和网格提取，且各任务指标均达到或接近先进水平，验证了表示的通用性。  
4. **光线深度编码的有效性**：消融实验表明绝对坐标编码会引入参考帧歧义，而基于视一致光线深度的嵌入是连续查询成功的关键。  
5. **两阶段训练策略的价值**：仅靠 2D 监督只能获得粗糙几何，后期加入 Eikonal 和光滑正则化对形成光滑表面至关重要。

- **高频细节的渲染能力不足**：在新视图合成任务中，IVGT的PSNR与SSIM具有竞争力，但LPIPS指标偏高，表明对复杂纹理和高频几何细节的重建质量不及一些专用渲染方法或显式高斯泼溅模型。
- **训练依赖RGB‑D序列**：两阶段训练均需要深度图与相机姿态作为监督信号，且法向由预训练模型提供，本质上仍依赖多视图RGB‑D数据，一定程度上限制了训练数据的易得性和场景多样性扩展。
- **推理效率偏低**：查询时每条射线需采样1024点并通过多层MLP推理SDF与颜色，体积渲染开销较大，难以实现实时光栅化模型的响应速度，限制了交互式应用场景。
- **仅限于静态场景**：当前模型仅针对静态场景建模，尚未扩展到动态场景，无法处理运动物体或非刚性形变。
- **对室外开放场景的泛化验证不足**：实验以室内场景和特定物体为主，虽然训练数据包含MegaDepth等部分室外数据，但缺少在大型、复杂室外开放场景（如城市街道全景）上的定量评估，鲁棒性有待进一步检验。
- **稀疏视图下的性能未系统研究**：尽管模型支持无姿态多视图输入，但对极端稀疏视图（如2~3帧）下的重建质量与视图关联性缺乏系统的定量分析。
