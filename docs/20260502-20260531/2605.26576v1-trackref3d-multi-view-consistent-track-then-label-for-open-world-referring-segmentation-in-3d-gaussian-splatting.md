## 摘要
基于自然语言进行3D对象分割的指代3D高斯溅射已成为具身人工智能的一项关键能力。然而，现有方法通常依赖昂贵的逐场景人工标注和逐视图伪掩码生成，这会导致多视图不一致且难以泛化到不同查询粒度。为此，我们提出TrackRef3D，这是一个无需人工标注、在3D高斯溅射中实现开放世界指代分割的全自动流程，其核心是引入多视图一致的“先跟踪后标注”范式，从根本上去耦合对象发现与语义定位。具体来说，我们设计了一个轨迹感知语义共识模块，通过同义聚类和轨迹感知投票聚合跨视图预测，建立规范的语义标识，从而确保多视图一致性。此外，我们采用可见性感知的描述生成策略来缓解歧义，并提出了一种混合训练策略，利用多正例对比目标联合优化粗粒度类别语义和细粒度指代线索，以确保在变化查询粒度下的鲁棒性。在基准数据集上的大量实验表明，TrackRef3D达到了最先进的性能。

## Abstract
Referring 3D Gaussian Splatting (R3DGS), which utilizes natural language for 3D object segmentation, has emerged as a crucial capability for embodied AI. However, existing methods typically rely on expensive per-scene manual annotation and per-view pseudo mask generation, which suffer from multi-view inconsistency and poor generalization to varying query specificities. To address this, we present TrackRef3D, a fully automatic pipeline that achieves open-world referring segmentation in 3D Gaussian Splatting (3DGS) without manual annotation by introducing a multi-view consistent track-then-label paradigm that fundamentally decouples object discovery from semantic grounding. Specifically, we propose a Trajectory-Aware Semantic Consensus Module (TSCM) which aggregates cross-view predictions via synonymous clustering and trajectory-aware voting to establish a canonical semantic identity, thereby ensuring multi-view consistency. Furthermore, we employ a visibility-aware description generation strategy to mitigate ambiguity and propose a Hybrid Training Strategy (HTS) that jointly optimizes coarse category semantics and fine-grained referential cues to ensure robustness under varying query specificities using a multi-positive contrastive objective. Extensive experiments on benchmarks demonstrate that TrackRef3D achieves state-of-the-art performance.

### **2. 论文提出的方法论**

TrackRef3D 的核心创新在于其**多视图一致的“先跟踪后标注”**范式，将“对象发现”与“语义接地”彻底解耦。整个流程分为三个主要阶段：

*   **初始对象发现**
    *   **目的**：从单视图图片中尽可能多地召回潜在对象。
    *   **方法**：利用开集检测模型（Florence-2）和分割模型（SAM-2）的组合，为每张图片生成实例掩码和初步的、可能带有噪声或不一致的类别标签。此阶段不进行过滤，保留所有预测结果，将一致性问题留待后续解决。

*   **轨迹感知语义共识模块**
    *   **核心思想**：将标注焦点从“单视图”转移到“相机轨迹”，认为同一物体在整个轨迹中应保持一致的语义身份。
    *   **关键技术细节**：
        1.  **视频追踪**：
            *   使用视频追踪器（DEVA），根据视觉特征和空间位置，将不同视图中的孤立掩码关联成时域连贯的物体轨迹 `T`。
            *   此过程即使在物体被遮挡后重现时也能维持其身份 ID，这是单视图方法无法做到的。
        2.  **同义聚类**：
            *   解决同一概念（如“垃圾箱”、“垃圾桶”）的文本标签差异问题。
            *   利用句子嵌入模型获取所有标签的文本嵌入，通过预设阈值 `τ_sem` 进行凝聚聚类，合并语义相近的标签（如统一为“垃圾桶”）。
        3.  **轨迹感知投票**：
            *   在每条轨迹 `Ti` 上，统计经过同义聚类后的身份标签出现频率。
            *   公式表示为：\(c_i^* = \text{arg max}_{c \in C_i} \sum_{t \in T_i} \mathbb{I}[\hat{c}_t = c]\)，其中 \(\hat{c}_t\) 是聚类后的身份，\(c_i^*\) 是最频繁的标签。
            *   将胜出的规范语义身份 \(c_i^*\) 反向传播到该轨迹的**所有视图**，确保每个物体实例在所有视角下都获得完全一致的监督标签。
        4.  **可见性感知的描述生成**：
            *   解决在物体被遮挡时生成模糊描述的问题。
            *   **关键帧选择**：提出一种可见性评分机制，为每条轨迹选择一个物体可见且具有清晰空间上下文的“黄金视角”。评分公式权衡了掩码面积和与中值面积的偏差：\(v_t = A_t \cdot \exp\left(-\frac{(\sqrt{A_t} - \sqrt{A_{\text{med}}})^2}{2\sigma^2}\right)\)。
            *   在该关键帧上利用视觉-语言模型（CogVLM-2）生成多粒度的指代描述，确保描述既包含物体属性，又包含空间关系。

*   **混合训练策略**
    *   **核心思想**：解决模型因仅接触长描述而无法泛化到短查询的问题。
    *   **关键技术细节**：
        *   将 TSCM 模块生成的“规范语义标识”（粗粒度类别）和“指代性描述”（细粒度）**全部视为正样本**。
        *   设计一个**多正例对比学习损失函数**：
            \[
            L_{\text{con}} = -\frac{1}{|P(c_i^*)|} \sum_{f_e^+ \in P(c_i^*)} \log \frac{\exp(f_g \cdot f_e^+ / \tau)}{\sum_{f_e \in D} \exp(f_g \cdot f_e / \tau)}
            \]
            其中，\(f_g\) 是预测区域的 3D 高斯嵌入，\(f_e^+\) 是正样本文本嵌入（包括类别名和所有指代描述），\(D\) 是批次中所有描述的集合（包括其他物体的描述作为负样本）。
        *   总损失 \(L = L_{\text{seg}} + \lambda L_{\text{con}}\)，其中 \(L_{\text{seg}}\) 是标准的掩码二元交叉熵损失。
