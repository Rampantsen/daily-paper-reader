## 摘要
3D高斯泼溅（3DGS）利用各向异性高斯核提供了高质量场景重建的高效方法。近期，基于3DGS的方法在实现实时性能的同时，显著提升了人体头像的渲染质量。然而，现有方法存在基于图像和基于3DMM生成的高斯核数量级不匹配的问题。这种差异导致重建的表情缺乏细粒度细节。在本文中，我们提出了一种从单张图像重建可驱动头部头像的新方法。我们设计了一个图分裂网络，利用自回归架构逐步从粗到细生成高斯核。为解决由分裂高斯核引起的图不一致性，我们采用网格拓扑扩展方法，使图神经网络（GNN）的连通性与增加的高斯核数量对齐。此外，我们引入了一种新颖的密度控制方法，其中包括一个门控机制，为高斯核生成软掩码，以防止分裂操作后的过度稠密化。这使得能够动态控制不同面部区域的高斯密度。为了平稳快速的训练，我们采用延迟过滤策略，避免在训练过程中重新计算图拓扑。实验结果表明，我们的自回归结构通过逐步分裂高斯核，有效提升了表情表示能力。这一由GNN引导的分裂过程合成了更精确的面部细节，并实现了更高的重建质量。

## Abstract
3D Gaussian Splatting (3DGS) provides an efficient method for high-quality scene reconstruction using anisotropic Gaussians. Recently, 3DGS-based methods have significantly improved the rendering quality of human avatars while enabling real-time performance. However, existing methods suffer from a magnitude mismatch in the number of Gaussians generated by image-based and 3DMM-based approaches. This discrepancy results in reconstructed expressions that lack fine-grained detail. In this paper, we introduce a novel method for reconstructing an animatable head avatar from a single image. We propose a Graph splitting network to progressively generate Gaussians from coarse to fine using an autoregressive architecture. To address the graph inconsistency caused by split Gaussians, we employ a mesh topology extension method to align the GNN's connectivity with the increased Gaussian count. Furthermore, we introduce a novel density control method that includes a gating mechanism that generates soft masks for Gaussians, preventing over-densification after the splitting operation. This allows for dynamic control over Gaussian density across different facial regions. For smooth and rapid training, we employ a delayed filtering strategy to avoid re-computing the graph topology during training. Experimental results demonstrate that our autoregressive structure effectively improves expression representation ability by progressively splitting Gaussians. This process, enabled by the GNN-guided splitting, synthesizes more precise facial details and achieves higher reconstruction quality.

*   **整体含义**：本文旨在解决上述数量级不匹配问题，通过提出一种新颖的“自回归高斯分裂”框架，将稀疏的表情高斯核渐进式地、可控地“分裂”为稠密的高斯核，从而在不牺牲身份一致性的前提下，显著提升表情驱动的细节保真度和重建质量。该工作为单样本高保真虚拟化身生成提供了一条高效且可控的新途径。

### 2. 论文提出的方法论

*   **核心思想**：将表情高斯的生成视为一个从粗到细的序列预测任务。利用一个基于图神经网络（GNN）的自回归模型，逐层对初始稀疏的高斯核进行分裂，每分裂一次，高斯核数量成倍增加，同时预测其更精细的属性，最终达到与身份高斯核相匹配的尺度。
*   **关键技术细节**：
    *   **双分支架构**：
        1.  **身份分支**：利用冻结的DINOv2模型提取源图像特征，通过高斯特征适配器（Gaussian Feature Adapter）生成稠密的“身份高斯核”，用于重建人物基本外貌。
        2.  **表情分支**：使用3DMM估计器从驱动图像中获取FLAME参数，以此为锚点生成初始的稀疏“表情高斯核”（共5023个）。该分支同样受源图像身份特征的条件约束。
    *   **自回归图分裂网络（Autoregressive GSN）**：
        *   **定义**：将表情高斯核定义为一个包含 \(L+1\) 层的序列 \(G_{exp} = (G_0, ..., G_L)\)。\(G_0\)是初始高斯，后续每一层\(G_{i+1}\)由上一层\(G_i\)通过高斯分裂网络（GSN）预测得到。
        *   **GSN核心**：一个基于图注意力网络（GAT）的模块。它将上一层高斯的属性（位置、旋转、缩放、颜色等）通过嵌入层映射为256维特征\(F_i\)，然后输入GAT进行图节点间的信息传播，最后由解码器D输出当前层的高斯属性\(G_{i+1}。\)
        *   **公式化**：\(F_i = E(G_i)\)，\(G_{i+1} = \lambda D(N_{GSN}(F_i, E_i) + F_i)\)，其中\(E_i\)是第\(i\)层的图邻接矩阵，\(\lambda\)是各属性的尺度因子。残差连接用于防止梯度消失。
    *   **网格拓扑扩展**：为解决因高斯分裂导致的数量激增与初始图拓扑不匹配的问题，论文提出动态构建每层的图结构\(E_i\)，它由两部分组成：
        1.  **拓扑继承边（\(E_i^{topo}\)）**：在由同一个父顶点分裂出的\(k\)个副本内，保留FLAME模型原始的拓扑结构。
        2.  **内部连接边（\(E_i^{conn}\)）**：将由同一个父顶点分裂出的\(k\)个子顶点进行全连接，以保证信息在分裂后的小组内有效传播。总边集\(E_i = E_i^{topo} \cup E_i^{conn}\)。
    *   **高斯门控机制（Gating Mechanism）**：
        *   **动机**：多次分裂会导致高斯冗余和过度稠密化。
        *   **方法**：引入一个软掩码网络\(N_{mask}\)，该网络根据上一层特征\(F_i\)预测一个介于0到1之间的软掩码\(m_{i+1}\)，用于动态过滤冗余高斯。
        *   **延迟过滤策略**：训练时不立即移除被屏蔽的高斯，而是将掩码层层累积（逻辑与操作），仅在最终渲染时忽略它们，避免了训练中频繁重构图拓扑的开销。
        *   **梯度传播**：使用直通估计器（STE）技术解决了掩码二值化过程中的梯度截断问题。
