## 摘要
实时多相机三维重建是沉浸式媒体、远程交互和空间计算的关键基础。尽管同步相机阵列被广泛采用，但实现几何一致且可扩展的实时重建仍具挑战。一个关键挑战是外参标定、多视图融合和全局优化之间的紧密耦合，这导致重建结果波动、累积误差和系统扩展性差。我们提出了一种用于标定与无状态实时多视图点云融合的解耦框架（FUSE-Flow），该框架包含两个协同组件：几何对齐的多视图外参标定（GMAC）和可靠性引导的多视图点云融合（FUSE）。这种分离设计避免了相互冲突的优化目标，从而实现有针对性的改进。GMAC模块通过几何约束和多视图重建变换器优化相机外参，无需标定目标、密集图像或全局光束平差即可实现精确稀疏视图标定。FUSE模块集成了置信度加权和自适应空间哈希，以实现无状态融合，确保线性的时间和内存消耗。两个模块相互增强：准确的相机位姿提高融合精度，置信度感知的融合校正标定偏差。在公开数据集和真实相机设置上的验证表明，FUSE-Flow在视觉效果、动态稳定性和可扩展性方面优于主流的实时重建方法，为大规模实时三维重建提供了一种实用解决方案。

## Abstract
Real-time multi-camera 3D reconstruction is a key foundation for immersive media, remote interaction and spatial computing. While synchronized camera arrays are widely adopted, achieving geometrically consistent and scalable real-time reconstruction remains challenging. A key challenge is the close linkage among extrinsic calibration, multi-view fusion and global optimization, which causes fluctuating reconstruction results, cumulative errors and poor system expandability. We propose a decoupled framework for calibration and stateless real-time multi-view point cloud fusion (FUSE-Flow), a framework with two collaborative components: geometry-aligned multi-view extrinsic calibration (GMAC) and reliability-guided multi-view point cloud fusion (FUSE). This split design avoids conflicting optimization objectives for targeted improvement. The GMAC module refines camera extrinsics via geometric constraints and multi-view reconstruction transformers, enabling accurate sparse-view calibration without calibration targets, dense images or global bundle adjustment. The FUSE module integrates confidence weighting and adaptive spatial hashing for stateless fusion, ensuring linear time and memory consumption. The two modules mutually reinforce each other: accurate camera poses boost fusion accuracy, and confidence-aware fusion corrects calibration biases. Validated on public datasets and real camera setups, FUSE-Flow outperforms mainstream real-time reconstruction methods in visual effect, dynamic stability and scalability, providing a practical solution for large-scale real-time 3D reconstruction.

### 1. 论文的核心问题与整体含义

- **核心问题**：实时多相机三维重建（MRR）面临三个相互关联的瓶颈：
  - **外参标定不稳定**：传统方法在机械振动、热漂移或动态场景下失效，在线方法依赖全局光束平差，相机数量增大时计算成本高。
  - **多视图融合不可靠**：深度数据受遮挡、高光、噪声影响，全局体素方法对配准误差敏感，简单点聚合忽略观测可靠性。
  - **可扩展性差**：基于 TSDF、SLAM、神经隐式的方法因全局状态或迭代优化导致超线性复杂度，难以支持大规模实时使用。
- **整体含义**：核心难点在于标定、融合与优化之间的紧密耦合，导致跨模块误差传递和系统扩展受限。本文提出解耦框架 FUSE‑Flow，将重建拆分为 GMAC（外参标定）和 FUSE（点云融合）两个独立模块，以减少任务间干扰，实现组件级优化与缩放。

### 2. 论文提出的方法论

- **总体架构**：解耦流水线，GMAC 输出度量级外参，FUSE 接收外参和深度图生成融合点云，无帧间状态。
- **GMAC（Geometry‑Aligned Multi‑view Extrinsic Calibration）**
  - **核心思想**：利用预训练多视图重建 Transformer（如 VGGT）的全局几何先验，将其尺度模糊的相对位姿通过几何约束提升为物理世界的度量尺度外参。
  - **关键技术细节**：
    1. 冻结主干网络，提取跨视图耦合的潜在特征和初始相对位姿。
    2. 采用**图耦合度量尺度提升**：假设所有相机共享单一全局尺度因子 s，通过深度引导的**重投影一致性**建立跨图边的深度残差。
    3. 构造加权最小二乘目标函数，权重综合考虑图重叠度、潜在特征余弦相似度和深度有效性掩码，得到闭式解 s*。
    4. 利用**三视图循环一致性**校验并迭代剔除异常边，确保全局几何一致性。
    5. 最终输出度量一致的相机外参。
- **FUSE（Reliability‑Guided Multi‑View Point Cloud Fusion）**
  - **核心思想**：每帧独立、无状态地融合多视图深度点，通过置信度建模和自适应空间哈希，实现线性复杂度且对残差外参稳健。
  - **关键技术细节**：
    1. **测量置信度建模**：基于深度归一化后的局部梯度大小和方差计算单点置信度。
    2. **跨视图几何一致性**：将点投影到预计算好的邻接相机，计算相对深度残差，集成为一致性权重；最终与单视图置信度通过**调和平均**结合为融合权重，实现“双门控”——要求单视图可靠且多视图一致。
    3. **两层自适应空间哈希与代表点选择**：
       - 粗网格估计密度，细网格尺寸随局部密度自适应调整。
       - 在每个细胞中选取置信度滤波后的候选点，再用**置信度加权的 farthest point sampling** 选出代表点，避免平均造成的模糊。
    4. **复杂度分析**：通过预计算相机图和自适应哈希，整体复杂度为 O(NHW)，随相机数线性增长，且无全局状态。
- **FUSE‑Flow 组合**：顺序耦合，GMAC 的外参提升降低融合中的不一致性，FUSE 的置信度设计反过来吸收标定误差，实现协同增益。

### 6. 论文的主要结论与发现

- 解耦设计有效减少任务间干扰，GMAC 与 FUSE 各自独立优化且可组合，无需端到端联合训练。

- 在不同算力平台上，FUSE‑Flow 保持近似线性的帧率下降，证实其工程可扩展性。

### 7. 优点（方法或实验设计上的亮点）

- **解耦架构创新**：将标定与融合分离，避免联合优化中的冲突，使各模块可独立部署、调试和升级。
- **无靶标度量标定**：GMAC 仅需少数图像，无需棋盘格或全局平差，利用闭式解实现高效标定，实用性强。
- **线性复杂度的无状态融合**：FUSE 彻底抛弃全局状态，通过置信度调和与自适应空间哈希实现线性缩放，这在实时多相机系统中极为少见。
- **双门控融合权重**：调和平均结合单视图置信和跨视图一致性，能同时过滤噪声、遮挡和匹配错误，无需显式遮挡检测。
- **实验全面扎实**：涵盖多种数据集、多主干网络、静态与动态、多相机数量和不同硬件，消融和联合评估逻辑清晰，结论可信。

### 8. 不足与局限

- **GMAC 的限制**：
  - 依赖可靠几何对应，在纹理稀疏、重叠少、运动模糊和强反射场景下性能下降。
  - 使用单一全局尺度因子，难以处理大规模相机阵列中可能存在的局部尺度不一致。
  - 主干网络的潜在特征不一致性可能影响标定质量。
- **FUSE 的限制**：
  - 需要有效深度输入和多视图一致性，导致薄结构、远处小物体或深度缺失区域重建不完整。
  - 无状态的逐帧处理在快速运动时可能引入微小的帧间不连续。
- **应用边界**：框架整体有意避免全局优化以保持线性复杂度和实时性，因此不适合亚毫米级工业计量等超高精度任务，定位在沉浸式多媒体和大规模交互应用。
- **未述及针对极端光照、完全无重叠等情况的鲁棒性，泛化能力有待进一步验证**。
