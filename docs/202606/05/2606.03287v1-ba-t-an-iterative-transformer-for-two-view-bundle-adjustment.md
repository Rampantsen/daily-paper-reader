## 摘要
用于三维重建的前馈模型通过深度跨视图注意力在图像间交换信息，已取得强劲性能。然而，这些方法通常依赖庞大的解码器堆栈，并缺乏结构化的几何优化机制，导致多视图一致性较差。我们借鉴经典光束平差（BA）的思想来解决这一问题，该过程可被视为位姿与局部几何之间的迭代信息传播。受BA启发，我们提出了BA-T，一种迭代Transformer，它在隐式令牌空间中将BA风格的结构化更新实现为可重复的层。BA-T不依赖深度注意力堆栈，而是通过单个轻量层基于潜在残差来优化预测。实验表明，BA-T在迭代过程中逐步提升位姿和重建精度，比传统解码器实现了更强的跨视图一致性，并且在仅使用解码器16%参数的情况下，匹配甚至超越了规模大得多的模型。BA-T为深度注意力提供了一种紧凑、高效且结构化的替代方案，能够在轻量架构中实现精确的三维重建。代码将公开于https://github.com/zhangganlin/BA-T。

## Abstract
Feed-forward models for 3D reconstruction have achieved strong performance using deep cross-view attention to exchange information across images. However, these approaches often depend on heavy decoder stacks and lack a structured mechanism for geometry refinement, resulting in poor multi-view consistency. We address this by drawing inspiration from classical bundle adjustment (BA), which can be viewed as an iterative information propagation process between poses and local geometry. Inspired by BA, we propose BA-T, an iterative Transformer that implements BA-style structured updates as a repeatable layer in implicit token space. Instead of relying on deep attention stacks, BA-T refines predictions based on latent residual by a single lightweight layer. Experiments demonstrate that BA-T progressively improves pose and reconstruction accuracy across iterations, achieves stronger cross-view consistency than conventional decoders, and matches or surpasses substantially larger models while using only 16% of their decoder parameters. BA-T provides a compact, efficient, and structural alternative to depth-heavy attention, enabling accurate 3D reconstruction within a lightweight architecture. The code will be made publicly at https://github.com/zhangganlin/BA-T.

*   **研究动机**：经典的光束平差（Bundle Adjustment, BA）通过显式迭代优化相机位姿和三维结构来保证几何一致性。然而，传统BA依赖于显式的几何表示（如点云、特征轨迹）和求解器（如高斯-牛顿法），难以直接集成到现代基于隐式token表示的学习框架中。该工作旨在结合两者的优势——在端到端学习的隐式空间内，引入BA的“位姿-几何”迭代信息传播模式，以更紧凑、更结构化的方式实现高精度、高一致性的三维重建。

### 论文提出的方法论
BA-T的核心思想是将经典BA中的位姿与几何间的迭代更新过程，建模为一个可重复使用的、结构化的Transformer层，直接在隐式token空间进行操作。

*   **整体流程**:
    *   输入：一对图像经编码器（ViT）提取的特征，作为初始“局部几何token”和共享的“相机token”。
    *   迭代精化：将初始几何token和相机token输入同一个轻量级BA-T层，进行多次迭代更新。每次迭代，BA-T层会预测一组隐式残差，并相应地更新相机token和几何token。
*   **关键技术细节**:
    *   **相机条件化几何变换 (Camera-Conditioned Geometry Transform)**: 为了在不同视图的局部坐标系下比较几何token，该模块利用相机token信息，通过一个adaln条件化块，将一个视图的几何token变换到另一视图的坐标空间。`π_c(g_a) := Condition(g_a, c)`
    *   **Token级对应匹配 (Token-level Correspondence Matching)**: 用交叉注意力机制为变换后的几何token在目标视图中寻找对应token。注意力分数作为柔性对应权重，通过对所有目标几何token的加权求和得到匹配结果。`g_q = Attention(π_c(g_a), g_b)`
    *   **隐式属性与残差函数 (Latent Property and Residual Function)**: 将匹配后的token对通过一个MLP，直接计算其在隐空间的“残差token” `r`，这模拟了经典BA中的重投影误差。`r = MLP( f(g_q) - f(π_c(g_a)) )`
    *   **相机更新 (Camera Update)**: 遵循BA中首先求解位姿更新的信息流模式，使用一个单对多的交叉注意力，让相机token从所有相关的残差token中聚合信息，从而产生更新量 `Δc`。`Δc = CrossAttn(c, r)`
    *   **局部几何更新 (Local Geometry Update)**: 在相机token更新后，再利用交叉注意力，基于另一视图的、经新相机参数变换后的几何token来更新本视图的几何token，实现跨视图几何精化。`Δg_a = CrossAttn(g_a, π_c'(g_b))`
*   **公式化描述**: 整个迭代过程可描述为 `T^{(k+1)} = T^{(k)} + BA-T(T^{(k)})`，其中 `T` 是包含相机和几何token的集合。

### 论文的主要结论与发现
*   **高效且精确**: BA-T以仅38M的解码器参数，在双视图位姿估计上超越了所有同规模及部分大模型，并在几何重建上表现相当，实现了精度与效率的最佳平衡。
*   **结构化迭代精化的有效性**: 相比简单堆叠交叉注意力块并进行迭代训练的基线（ViSTA†），BA-T结构化的迭代更新能更有效地提升几何一致性和位姿精度。
*   **组件贡献明确**: 消融实验证实，相机条件化几何变换、token级对应匹配以及迭代监督对最终性能均有正向贡献。
*   **多视图扩展潜力**: BA-T的架构天然支持多视图扩展，仅通过改变token间交互的图结构即可实现，且初步结果证明了其在多视图场景下的有效性。

### 优点
*   **方法设计优雅**: 将经典BA的优化模式与深度学习的隐式表征巧妙结合，是一种思维上的跨越。
*   **性能与效率俱佳**: 在显著降低模型尺寸和计算开销（参数量、内存、推理时间）的同时，达到了SOTA级别的精度。
*   **结构化设计可解释性强**: BA-T内部的各个模块（变换、匹配、残差计算、更新）职能清晰，其行为（如残差图变化、几何变换）在实验中被可视化，证明了其设计的合理性。
*   **可扩展性**: 层可重复使用，迭代次数可变，并且展现出向多视图增量的潜力。

### 不足与局限
*   **两视图训练为主**: 当前实现和训练主要聚焦于两视图重建，多视图场景虽展示了概念验证，但未进行大规模训练和深入优化，其性能可能与专门为多视图设计的SOTA大模型仍有差距。
*   **几何提升相对有限**: 文中指出，由于几何重建是在每个视图本地进行预测后组合，迭代精化对几何绝对指标的提升不如位姿提升显著。几何一致性的改进更多是间接通过精化的位姿体现。
*   **动态与无纹理场景鲁棒性未知**: 方法继承了学习类前馈模型的特点，但未专门评估其在高度动态或缺乏纹理等传统SfM/SLAM挑战场景下的表现，其泛化能力可能仍受限于训练数据分布。
*   **依赖初始预测**: 作为一个端到端精化网络，其最终输出质量依赖于编码器提供的初始几何和相机token质量。如果初始值偏差过大，BA-T是否仍能有效收敛至全局最优，论文未深入探讨。
