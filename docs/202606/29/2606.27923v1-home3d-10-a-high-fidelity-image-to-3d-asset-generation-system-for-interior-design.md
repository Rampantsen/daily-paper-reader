## 摘要
我们提出了Home3D 1.0，一个模块化的图像到3D生成系统，可从单张参考图像生成高质量的3D资产，面向室内设计和电子商务应用。给定一张家具或装饰品的照片，系统输出具有基于物理的渲染（PBR）材质的网格，并且该网格可以分解为特定材质的组件。该管线由四个紧密耦合的模块组成：几何模块通过潜在SDF建模，利用几何VAE和从粗到细的流匹配DiT重建水密网格；纹理模块预测多视图反照率观测，将其重投影到网格上，并用3D纹理场补全不可见表面区域；材质模块利用MatWeaver通过基于视频的分割和UV空间投票获得组件掩码，然后通过分层多模态匹配从精选材质库中检索并烘焙PBR贴图；部件模块使用PartVAE和PartDiT生成材质可编辑的语义部件网格，一次性解码多头部件特定的SDF场。每个模块都通过专门的度量进行独立评估，突显了当前系统的能力以及向更广泛部署过程中尚存的差距。

## Abstract
We present Home3D 1.0, a modular image-to-3D generation system that produces high-quality 3D assets from a single reference image, targeting interior design and e-commerce applications. Given a photograph of a furniture or decor item, the system outputs a mesh with physically-based rendering (PBR) materials, and the mesh can be decomposed into material-specific components. The pipeline is organized into four tightly coupled modules: Geometry reconstructs a watertight mesh through latent SDF modelling with a geometry VAE and a coarse-to-fine flow-matching DiT; Texture predicts multiview albedo observations, reprojects them onto the mesh, and completes unseen surface regions with a 3D texture field; Material uses MatWeaver to obtain component masks through video-based segmentation and UV-space voting, then retrieves and bakes PBR maps from a curated material library through hierarchical multi-modal matching; and Parts generates material-editable semantic part meshes with a PartVAE and PartDiT, decoding multi-head part-specific SDF fields in one pass. Each module is evaluated independently with dedicated metrics, highlighting both the current system capability and the remaining gaps toward broader deployment.

- **研究动机**：室内设计和电子商务领域需要从单张商品照片自动生成可编辑的 3D 家具资产。通用图像到 3D 生成仅要求类别合理，而家具生成必须精确保持轮廓、比例、风格、构造逻辑与材质质感，对几何稳定性、纹理细节和 PBR 物理属性要求极高。
- **现状与痛点**：现有方法生成的网格常出现噪声、缺失结构、对称性差、薄结构破损，纹理模糊、材质不一致，部件分解也多为功能导向，无法满足设计师按材料（织物、木材、金属等）编辑的需要。
- **整体含义**：Home3D 1.0 将问题定义为产品建模任务，提出一个模块化管线，分几何、纹理、材质和部件四个阶段，各自解决家具生成中的一类失效模式，最终产出水密网格、全纹理覆盖、材质可编辑的部件化资产，为工业级 3D 内容生产提供基准。

## 2. 方法论

- **核心思想**：将单图到 3D 的生成分解为四个专门模块，按依赖关系流水线执行（几何→纹理→材质；部件模块可并行在几何之后），避免单个黑盒模型的局限性。
- **几何模块（Geometry）**：
  - 使用几何 VAE 学习连续形状的潜在表示。编码器以点云（位置 + 法线 + 尖锐边缘指示）为输入，通过 Perceiver 风格的潜在查询交叉注意力提取紧凑潜在码 Z。
  - 解码器将潜在码与查询点 x 送入隐式函数 s(x)=fθ(x,Z)，预测带符号距离场（SDF），再通过 Marching Cubes 提取水密网格。
  - 图像条件生成采用从粗到细的流匹配 DiT：粗阶段 DiT 从噪声生成全局一致的潜在码，解码后体素化得到空间先验 Sc；细阶段 DiT 结合图像特征和 Sc 生成精细潜在码，恢复尖锐边界和局部细节。
  - 损失函数：VAE 使用 SDF 回归损失与 KL 散度；DiT 使用流匹配目标 LFM = Et, Z0,Z1[||vψ(Zt,t,C) – (Z1-Z0)||²₂]，Zt = (1-t)Z0 + tZ1。
- **纹理模块（Texture）**：
  - 多视图反照率预测：输入参考图、网格及几何图像，联合预测多个规范视角的反照率贴图，保持外观特征和跨视图一致性。
  - 将预测的反照率根据相机几何重投影到网格表面，形成部分纹理网格。
  - 3D 纹理补全模块：在 3D 表面空间建模连续颜色场，补全因遮挡和视角有限造成的缺失区域，不受 UV 拓扑限制。
  - 训练采用扩散损失、跨视图一致性损失、图像重建损失和梯度正则化项的组合，并引入光照增强以解耦固有色与光照。
- **材质模块（Material）**：
  - MatWeaver 部件分割：将 3D 网格部件分割转化为视频跟踪任务。通过超像素聚类和边缘感知采样生成点提示，利用 SAM2 在合成视频序列上追踪部件掩码；用 mIoU 合并相同部件，再通过 UV-Face 映射和投票将分割结果烘焙到 3D 面，得到语义一致的材质区域。
  - 分层多模态材质检索：首先通过 RoMA 特征匹配和 P4Pf 求解进行 2D‑3D 对齐，将部件掩码投影到输入图像上。然后利用 VLM 为每个部件生成描述和材质类别，使用 Qwen3‑VL 跨模态嵌入在材质库内检索 top-5 候选；最后用 VLM 作为专家裁判对候选重排序，选出最匹配的材质。
  - 物理烘焙：从检索到的材质参考中提取金属度、粗糙度、法线等 PBR 属性，按部件分配到独立材质槽，合成为引擎就绪的贴图，并通过 Draco 压缩可降低 20 倍尺寸。
- **部件模块（Parts）**：
  - 采用 PartVAE 学习固定大小的部件感知潜在空间。编码器接收带部件 ID 的点云，通过部件嵌入与位置法线融合，输出共享潜在码 z。
  - 解码器为多头 SDF：一个潜在变换器和交叉注意力解码器一次输出多通道的部件 SDF 场（如 32 头），每个通道对应一个部件，用 Marching Cubes 分别提取部件网格，避免逐部件重复解码。
  - PartDiT 以 DINOv2 图像特征和完整几何潜在码为条件，从噪声生成潜在码，再解码为部件网格。
  - 训练损失：多头部 SDF 重建损失、远场抑制损失（避免部件间伪影）和 KL 正则化；DiT 使用流匹配，分大规模预训练、强化训练和精调三步走。

## 6. 主要结论与发现

- Home3D 1.0 在几何重建上取得最低 CD 0.4936×10⁻³、最低 EMD 5.174×10⁻² 和最高 F1@0.01 0.6329，优于所有闭源基线；在外观上 CLIP‑I 最高 0.9370、LPIPS 最低 0.07438，表现出高保真度和多视图一致性。
- 材质检索与视频分割方案能生成准确的材料区域和物理正确的 PBR 贴图，避免了直接生成 PBR 的不稳定性。
- 部件模块的 PartVAE 以共享潜在码和多头 SDF 解码实现了固定解码成本下的语义部件分离，解码时间比逐部件方法更高效（如 32 部件 24.048s），且生成的部件支持材料级编辑。
- 系统验证了“分而治之”的模块化策略对工业级家具资产生成的有效性，各模块可独立优化与评估。

## 7. 优点

- **模块化设计**：每个模块针对家具生成中的特定失败模式，职责清晰，便于独立改进和替换。
- **高保真几何**：粗到细的流匹配 DiT 结合感知器风格的 VAE，能恢复薄结构、尖锐边界和全局对称性。
- **材质稳定性**：基于检索而非生成的材质获取方式，确保了 PBR 贴图的物理可信度和一致性，适合电商场景。
- **高效部件解码**：PartVAE 的多头 SDF 设计使解码时间不随部件数量线性增长，支持一步生成完整部件集合。
- **无标注分割**：基于视频跟踪与 UV 投票的 MatWeaver 无需人工标注即可获得部件掩码，具有较好的自动化潜力。
- **评价多元**：采用 CD、EMD、F1、CLIP‑I、LPIPS 等多维度指标，并包含解码延迟等工程指标，评估较为全面。
