## 摘要
多模态融合与多模型集成被广泛用于提升三维语义分割的性能。尽管效果显著，但这些方法要么依赖辅助输入信号，要么面临高昂的计算开销。为了在不引入难以承受成本的前提下有效增强分割性能，我们提出通过知识蒸馏将多模态模型（即点云和图像）及多个模型专家的丰富知识迁移到基于点云的网络。具体而言，我们提出了面向信息的异构蒸馏（IHD），帮助单模态模型吸收多模态教师的互补知识。我们设计了面向信息的过滤（IOF）策略，从连续图像序列中筛选信息量丰富的图像用于多模态融合。这一做法能提升多模态教师的性能，从而有利于学生的学习。此外，与需要单独训练每个专家的普通模型集成不同，我们提出了擅长快照蒸馏（ASD）。ASD 将训练阶段自然产生的模型快照视为多个专家，从而显著降低了模型集成的训练开销。对每个专家教师，它仅在自己擅长的类别上为学生提供监督。由此产生的异构与擅长快照知识蒸馏（HAS-KD）在 ScanNetV2 和 S3DIS 数据集上取得了最优结果。HAS-KD 可无缝融入现有的三维分割算法，并在不引入额外推理负担的情况下带来可观的性能增益。代码将在论文发表后公开。

## Abstract
Multi-modal fusion and multi-model ensembling are prevalent in enhancing the performance of 3D semantic segmentation. Despite the impressive performance, these methods either rely on auxiliary input signals or suffer from costly computational expense. To efficaciously enhance the segmentation performance without introducing intolerable costs, we propose to transfer the rich knowledge from the multi-modal model (i.e., point clouds and images) and multiple model experts to the point-cloudbased network through knowledge distillation. Specifically, we present Information-oriented Heterogeneous Distillation (IHD) to help the uni-modal model absorb the complementary knowledge from the multi-modal teacher. We design the Information-Oriented Filtering (IOF) strategy to select informative images from the continuous image sequence for multi-modal fusion. This practice can boost the performance of the multi-modal teacher, thus benefiting the learning of the student. Besides, as opposed to vanilla model ensembling that requires the separate training of each expert, we propose Adept Snapshot Distillation (ASD). ASD treats the freely available model snapshots generated during the training phase as multiple experts, which significantly reduces the training cost for model ensembling. For each expert teacher, it only provides supervision to the student in the class where it is adept. The resulting Heterogeneous and Adept Snapshot Knowledge Distillation, dubbed HAS-KD, attains state-of-the-art results on ScanNetV2 and S3DIS datasets. HAS-KD can be seamlessly integrated into contemporary 3D segmentation algorithms and bring considerable gains without introducing extra inference burdens. The code will be made publicly available upon publication.

## 方法论

### 1. 信息导向的异构蒸馏（Information‑oriented Heterogeneous Distillation, IHD）
- **目标**：让单模态点云学生网络模仿多模态教师网络的融合特征，从而在推理时丢弃图像输入也能获得性能提升。
- **关键组件——信息导向过滤（Information‑Oriented Filtering, IOF）**：
  - 观察：同一物体在连续图像序列中会出现多次，但随机选择视图可能无法捕捉到物体完整、无遮挡的纹理信息。
  - 做法：利用训练时可得的地面真值实例掩码（或离线实例分割模型如 SAM3D 的预测），为每个物体计算其在各视图的**可见点比例（语义丰度 %）**。
    - 将物体点云投影到第 \(j\) 个视图，筛选落在视口内且投影深度与深度图差异小于 2 cm 的点，获得可见点数 \(M\)，则语义丰度 \(A_j^i = M/N\)。
    - 选择对每个物体拥有最大 \(A_j^i\) 的视图作为其“最佳视角”，用于多模态融合。
  - 效果：提升多模态教师的性能，从而蒸馏出更强的学生。
- **蒸馏方式**：用多模态教师输出的融合特征 \(F_{\text{fusion}}\) 与学生单模态特征 \(F_{\text{student}}\) 计算 MSE 损失：
  \[
  L_{\text{IHD}} = \mathbb{E}[\|F_{\text{student}} - F_{\text{fusion}}\|^2]
  \]

### 2. 擅长快照蒸馏（Adept Snapshot Distillation, ASD）
- **目标**：利用多模型集成智慧，但避免额外训练代价，并避免向学生传递不精通类别的噪声。
- **关键设计——基于快照的专家教师**：
  - 将单次训练过程中每个 epoch 保存的模型快照直接视为多个“专家”，**无需分别训练**。
  - 为第 \(i\) 类选择**在该类上 IoU 最高的 checkpoint** 作为该类专家教师 \(E_i\)。
  - 每个专家仅对自己擅长的类别提供监督，即对其特征图中属于该类别的体素（由语义标签生成掩码 \(M_i\)）进行蒸馏。
- **集成与蒸馏**：从所有专家中提取各自擅长类别的特征，拼接成集成特征 \(F_{\text{ensemble}}\)：
  \[
  F_{\text{ensemble}} = F_0 M_0 \cup F_1 M_1 \cup \dots \cup F_{n-1} M_{n-1}
  \]
  然后用 \(F_{\text{student}}\) 与 \(F_{\text{ensemble}}\) 的 MSE 损失：
  \[
  L_{\text{ASD}} = \mathbb{E}[\|F_{\text{student}} - F_{\text{ensemble}}\|^2]
  \]

### 3. 统一训练框架
- 将 ASD 中的单模态教师替换为 IHD 建成的多模态教师，得到统一蒸馏损失 \(L_{\text{ASD-IHD}}\)。
- 最终目标：
  \[
  L = L_{\text{Seg}} + \alpha L_{\text{ASD-IHD}}
  \]
  其中 \(\alpha = 4\)。

## 实验设计

- **数据集与指标**：
  - **ScanNetV2**：1 513 个室内扫描场景（1 201 训练，312 验证），20 类语义；报告 **mIoU**。
  - **S3DIS Area 5**：271 个房间，13 类语义；报告 **mIoU、mAcc、OA**。
- **基准模型**：Point Transformer V3 (PTV3) 作为学生网络，同时作为无蒸馏的基线。
- **对比方法**：包括 PointNet++、KPConv、MinkUNet、Point Transformer V1/V2、OctFormer、Swin3D 等纯点云方法，以及 BPNet、DeepViewAgg、VMVF 等多模态方法；还对比了使用多数据集预训练的 PTV3†。
- **实现细节**：
  - 框架：基于 Pointcept。
  - 训练策略：AdamW + OneCycle LR scheduler，batch size 24，最大学习率 0.006，权重衰减 0.05。
  - 教师预训练：ScanNet 800 epoch，S3DIS 3 000 epoch；蒸馏阶段：ScanNet 160 epoch，S3DIS 1 200 epoch。
  - 体素尺寸 2 cm，数据增强包括随机丢弃、旋转、缩放、翻转、弹性变形、球面裁剪、Mix3D 等。
  - 图像分支采用简单 UNet，最多选取 12 个视图。

## 主要结论与发现

- HAS‑KD 成功在**不增加推理成本**的前提下，将多模态和多专家的知识注入单模态点云网络。
- 在 ScanNetV2 上，HAS‑KD 以 **78.6 mIoU** 超越所有非多数据集训练的纯点云方法，甚至与使用 17 倍大数据预训练的 PTV3† 持平。
- 在 S3DIS Area 5 上，HAS‑KD 将强基线 PTV3 从 71.1 提升至 **72.1 mIoU**，达到新 SOTA。
- IOF 策略通过视图筛选显著提升多模态教师，进而蒸馏出更强的学生。
- ASD 的“类别专家”集成方式优于传统的平均集成，能提供更专业的监督，从而让学生性能更高。

## 优点

- **高效实用**：完全解耦训练与推理，推理阶段仅需单模态点云网络，速度、显存与普通模型完全相同。
- **训练成本低**：快照蒸馏利用单次训练即可获得所有专家，避免了传统集成需多次训练的弊端。
- **即插即用**：方法可无缝嵌入当前前沿分割网络（如 PTV3），实现可观增益。
- **视角选择合理**：利用语义丰度指导视图选择，比随机挑选视图更利于发挥多模态优势。
- **类别级专长蒸馏**：让专家只教自己擅长的类别，避免了平庸或差教师的噪声干扰。

## 不足与局限

- **训练代价未完整披露**：虽降低了集成训练开销，但教师预训练（800/3000 epoch）本身成本不低，且未报告所需的 GPU 数量和训练时间，读者难以评估实际训练资源需求。

- **与多模态“教师”的可能差距**：蒸馏上限受限于教师性能，若教师未能很好融合多模态信息，学生收益可能有限。
