## 摘要
我们提出了Casper3D，一个轻量级的概率框架，用于将含噪的多视图2D基础模型嵌入转换为潜在的3D语义表示。我们将视图级语义特征建模为底层3D语义状态的噪声观测，并通过一个基于集合的变分模型在多视图推理过程中结合相对姿态来推断该状态。Casper3D的训练通过从新视角预测保留的语义观察进行，同时保持与视觉和文本语义空间的对齐，以实现开放词汇的3D理解。该框架不依赖于特定骨干网络，并可应用于语言对齐和自监督嵌入。实验表明，与简单的多视图池化相比，Casper3D能产生更稳定的3D语义，尤其是在模糊和噪声环境下。

## Abstract
We present Casper3D, a lightweight probabilistic framework for converting noisy multi-view 2D foundation-model embeddings into a latent 3D semantic representation. We model view-level semantic features as noisy observations of an underlying 3D semantic state and infer this state with a set-based variational model that incorporates relative pose during multi-view reasoning. Casper3D is trained by predicting held-out semantic observations from novel viewpoints, while remaining aligned with visual and text semantic spaces for open-vocabulary 3D understanding. The framework is backbone-agnostic and applies to both language-aligned and self-supervised embeddings. Experiments show that Casper3D produces more stable 3D semantics than simple multi-view pooling, especially in ambiguous and noisy settings.

## 1. 论文核心问题与研究动机
- 现有2D基础模型（如CLIP、DINO）虽能提取强大的语义特征，但这些特征**依附于图像坐标，受视角、遮挡、尺度等因素影响**，并非真正的三维语义。
- 直接将这些视图依赖的2D特征通过**简单平均或池化**映射到3D点云会丢失空间一致性，产生噪声和歧义。
- 本研究将2D→3D特征提升视为一个**贝叶斯反问题**：认为每个3D点存在一个**不可观测的、视角无关的潜在语义状态**，而各视图的2D特征只是其含噪观测，目标是从多视图观测中**推断出该隐藏状态的后验分布**。

## 2. 方法论：Casper3D 框架
- **整体思想**：对每个3D点\(p_i\)，假设存在隐变量\(z_i\)（视角稳定的语义状态），2D特征\(x_{ij}\)由该状态与视角条件\(T_j\)共同生成。通过**变分推断**，从无序的上下文观测集合\(C_i\)中估计后验\(q_\phi(z_i|C_i)\)（高斯分布），再通过解码器根据\(z_i\)和新的视角**预测该视角下应出现的2D特征**。
- **训练策略**：不使用真实的3D特征标签，而是采用**保留视图预测**的自监督方式：将观测分为上下文集和保留集，模型用上下文推断\(z_i\)，再预测保留视图的特征，迫使隐变量捕捉视角不变的信息。
- **网络架构**：
  - **编码器**：基于Set Transformer，使用**位姿感知注意力块（PAB）**，将相对位姿\(T_{C\to C}\)注入注意力计算的查询和偏置中，保证排列等变性。
  - **解码器**：接收隐变量和保留视图相对于上下文的位姿\(T_{H\to C}\)，预测该视图的2D特征。
  - 最终通过一个投影头\(h_\psi\)将隐空间映射回基础模型的原嵌入空间，得到可用于下游任务的3D特征\(e_i\)。

- **依赖准确几何**：训练和评估都需要真实的相机位姿和深度图来投影特征，无法直接应用于无精确几何的场景。文中提到未来可用现成的3D估计模型替代。

- 使用平均特征作为弱锚可能在某些极端视角变化下不够合理。
