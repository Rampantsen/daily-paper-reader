## 摘要
在内部物流环境中运行的自主移动机器人依赖几何地图进行定位和导航，但缺乏对物体及其上下文属性的语义理解。我们提出了一种上下文语义建图流水线，它结合了基于SLAM的几何建图、基于SAM的实例分割、实例聚类和VLM多视图推理，生成了编码几何结构、物体类别和物体可移动性的上下文语义地图表示。通过聚合多个视角的观测并在零样本、开放词汇设置下查询VLM，该流水线无需任务特定训练或预定义物体类别即可推断上下文物体属性——本文通过可移动性进行演示。我们在两种提示策略下评估了三种VLM，并对流水线进行了逐组件分析。所提出的流水线在语义分类上达到了98.93%的mIoU，在物体可移动性估计上达到了89.17%的mAcc。组件分析表明，VLM推理是上下文理解的主要瓶颈，而实例聚类是全景性能的主要限制。生成的语义地图支持在动态内部物流环境中进行上下文感知过滤和鲁棒导航。

## Abstract
Autonomous mobile robots operating in intralogistics environments rely on geometric maps for localization and navigation, but lack semantic understanding of objects and their contextual properties. We present a contextual semantic mapping pipeline that combines SLAM-based geometric mapping, SAM-based instance segmentation, instance clustering, and VLM multi-view reasoning to produce a contextual semantic map representation encoding geometric structure, object class, and object movability. By aggregating observations across multiple viewpoints and querying a VLM in a zero-shot, open-vocabulary setting, the pipeline infers contextual object properties--here demonstrated through movability--without requiring task-specific training or predefined object categories. We evaluate three VLMs under two prompting strategies and conduct a component-wise analysis of the pipeline. The proposed pipeline achieves 98.93 % mIoU for semantic classification and 89.17 % mAcc for object movability estimation. Component analysis identifies VLM reasoning as the primary bottleneck for contextual understanding and instance clustering as the main limitation for panoptic performance. The resulting semantic map supports context-aware filtering and robust navigation in dynamic intralogistics environments.

### 2. 方法论

流水线由五个主要模块构成：

```
多模态观测 → 几何建图 → 实例分割 → 实例聚类 → VLM推理 → 地图融合 → 上下文语义地图
```

- **传感器输入与时空对齐**：使用两个 360° 二维激光雷达和一个前视 RGB 相机，通过时间同步建立激光点与图像像素的对应关系，使后续可联合处理空间和视觉信息。
- **几何建图**：采用 GMapping 算法（一种 2D SLAM）构建环境的二维点云几何地图 \( GM \)。
- **实例分割**：对每一帧 RGB 图像独立运行 SAM（Segment Anything Model）的自动掩码生成模式，获得细粒度、类别无关的实例掩码，并利用点-像素对应将掩码投影到地图坐标系。
- **实例聚类**：在地图空间内，基于掩码投影区域间交并比（IoU > 0.3）对跨帧实例进行聚类，从而将同一物理物体的多次观测聚合为一个持久化的实例。

  - **输入表示**：对每个实例提供一张复合图像，左侧为高亮绿色掩码的全景图，右侧为裁切的物体特写图，并将该实例在所有视角下的复合图像同时送入 VLM。

- **地图融合**：将实例的几何范围、语义类别和可移动性标签回注至几何地图点云，形成六维语义地图 \( SM = \{(p_i, cls_i, ins_i, m_i)\} \)，其中 movability 取值于 \(\{ \text{immovable}, \text{movable}, \text{mobile} \}\)。

### 6. 主要结论与发现

- Gemini 3.1 Flash Lite 配合直接 JSON 提示在语义分类上表现最佳（98.93% mIoU），而 GPT‑5.1 在可移动性上表现最优（96.28% mAcc），GPT‑o3 居中。

### 7. 方法优点

- **即插即用的语义建图框架**：将 SAM 的类无关分割与 VLM 的零样本推理有机结合，无需预定义类别或额外训练，极大提升了在内部物流新环境中的部署灵活性。
- **多视角聚合与上下文推理**：通过实例聚类融合多帧观测，并让 VLM 基于全景+特写复合图像理解空间关系和物体状态，有效处理遮挡和局部歧义，并实现了超越视觉外观的推理（如区分“独立托盘”与“叉车上的托盘”）。
- **可解释的模块化设计**：流水线各模块相对独立，可替换或单独升级；输出包含文本解释和结构化 JSON，便于调试和下游任务使用。
- **系统性组件分析**：通过 oracle 替换实验清晰定位瓶颈，为后续优化（如改进实例关联或增强 VLM 推理）提供了明确方向。
- **公开数据集**：提供含标注的图像、激光扫描和提示语的数据集，支持复现和进一步研究。

- **场景单一**：仅在单个受控的内部物流环境、有限的物体实例上评估，泛化到更复杂、大面积或多楼层环境的有效性未知。
