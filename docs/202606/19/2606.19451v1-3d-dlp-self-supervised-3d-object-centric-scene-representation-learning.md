## 摘要
我们提出了3D-DLP，一种自监督的以物体为中心的表示学习模型，它将场景级别的RGB-D或体素观测分解为一组三维潜在粒子。基于深度潜在粒子（DLP）框架，每个粒子编码了解耦的属性，包括三维关键点位置、边界框尺寸和外观特征，并表示场景中的一个独立实体。该模型通过端到端的自监督重建目标学习可解释的逐粒子分割图。我们在仿真和真实世界数据集上证明，学习到的潜在空间是可解释且可控的：通过操纵粒子位置并解码，我们可以生成新颖的场景配置。此外，我们展示了利用这些紧凑的三维潜在粒子进行下游机器人操作，相比缺乏显式三维信息或依赖于无物体中心结构的密集三维输入的基线方法，性能有所提升。代码和视频可在https://eubooks3003.github.io/3d-dlp获取。

## Abstract
We introduce 3D-DLP, a self-supervised object-centric representation learning model that decomposes scene-level RGB-D or voxel observations into a set of 3D latent particles. Building on the Deep Latent Particles (DLP) framework, each particle encodes disentangled attributes, including 3D keypoint position, bounding box dimensions, and appearance features, and represents a distinct entity in the scene. The model learns interpretable per-particle segmentation maps through an end-to-end self-supervised reconstruction objective. We demonstrate on both simulated and real-world datasets that the learned latent space is interpretable and controllable: by manipulating particle positions and decoding, we can generate novel scene configurations. Furthermore, we show that leveraging these compact 3D latent particles for downstream robotic manipulation improves performance over baselines that either lack explicit 3D information or rely on memory-intensive dense 3D inputs without object-centric structure. Code and videos are available at https://eubooks3003.github.io/3d-dlp.

### 2. 方法论
- **整体框架**：基于 Deep Latent Particles (DLP)，将场景观测编码为一组解耦的 3D 潜在粒子（每个粒子含位置、尺寸、外观等属性）和一个背景潜在变量，再通过解码合成重建输入。
- **三种模态变体**：
  - **3D‑DLP‑D**：处理 RGB‑D 图像，沿用空间 softmax 关键点提议。
  - **3D‑DLP‑V**：处理占用体素（二进制），采用基于 K‑means 的几何关键点提议。
  - **3D‑DLP‑VC**：处理 RGB 体素，**核心创新点**包括：
    - **外观感知 K‑means 先验**：在 CIELAB 颜色 + 位置联合空间聚类，用亮度加权，生成贴合物体表面和边界的 3D 关键点提议。
    - **编码器**：用 3D CNN 和立体变换网络（STN）从提议位置提取局部块，输出粒子位移、尺度、透明度等属性；掩蔽已激活粒子区域后编码背景。
    - **体素合成**：每粒子解码出 RGBA 立方块，通过 alpha 混合与背景组合重建场景，无需 2D 中所需的组合顺序。
    - **色度损失**：在 MSE 基础上，对占用体素计算色度（去除亮度）误差，**防止灰度坍缩**（即仅靠亮度匹配而丧失彩色）。
  - 训练目标为 VAE 的 ELBO：`L = L_rec + β_KL L_KL + β_obj L_obj`，其中 `L_obj` 鼓励粒子稀疏使用。
- **粒子属性**：位置 `z_p`（3D 坐标）、尺度 `z_s`（3D 边框）、透明度 `z_t`（Beta 分布）、外观特征 `z_f`。

- 3D‑DLP 在多数数据集上实现了比 AE/VAE 更高的 Masked PSNR 和 IoU，消融证明 **K‑means 先验**和**色度损失**不可或缺。

### 7. 优点与亮点
- **首创性**：首个直接处理彩色体素的自监督 3D 对象中心分解方法。
- **统一框架**：支持 RGB‑D、占用体素、RGB 体素三种模态，可灵活适配不同传感器。
- **关键技术贡献**：外观感知 K‑means 先验 + 色度损失，有效解决稀疏体素下的关键点检测和颜色保真问题。
- **可解释性与可控性**：粒子的位置、尺度可直接编辑，直观改变场景，为交互式应用打开大门。
- **下游实用性**：无缝集成到模仿学习策略中，在多物体操作任务上取得显著增益。
- **实验全面**：涵盖合成到真实世界、多种基准和消融，定量与定性验证兼具。

### 8. 不足与局限
- **体素瓶颈**：体素化带来较高内存消耗，尚未直接处理原始点云。

- **计算成本**：训练需 48 GPU 小时，虽可行但若需超大规模场景训练可能仍有负担。
