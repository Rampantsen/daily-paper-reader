## 摘要
本文提出SGFormer++，一种面向三维场景图生成（SGG）的新型语义图Transformer，旨在将点云场景解析为语义结构图，其中节点表示检测到的对象实例，边编码其成对关系，核心挑战在于建模复杂的全局场景结构。现有基于图卷积网络（GCN）的方法存在过度平滑和感受野受限的问题，而SGFormer++利用Transformer层作为骨干网络来实现全局消息传递。具体而言，我们引入了两个针对三维SGG设计的关键组件：（1）Graph Embedding Layer++，以线性计算复杂度高效融合边感知的全局上下文；（2）Semantic Injection Layer++，利用来自大型语言模型（LLM）和视觉-语言模型（VLM）的语言先验丰富视觉特征，从而在不引入额外可训练参数的情况下增强语义表示。为进一步应对增量式SGG（I-SGG）的实际挑战，即新的关系类别顺序到达，我们为SGFormer++配备了一个新颖的空间引导特征适配器，利用主客体空间几何校准谓词特征以对抗尺度变化，以及一个级联二元预测头，通过任务增量分类器扩展和logit蒸馏减轻灾难性遗忘。在3DSSG基准上的大量实验表明，SGFormer++在标准和增量设定下均达到了最先进的性能：在增量设定下，谓词A@1取得了显著的4.49%的绝对提升。代码和数据可在以下链接获取：https://github.com/Andy20178/SGFormer。

## Abstract
In this paper, we propose SGFormer++, a novel Semantic Graph Transformer for 3D scene graph generation (SGG), which aims to parse point cloud scenes into semantic structural graphs, where nodes denote detected object instances and edges encode their pairwise relationships, with the core challenge lying in modeling complex global scene structure. While existing graph convolutional network (GCN)-based methods suffer from over-smoothing and limited receptive fields, SGFormer++ leverages Transformer layers as its backbone to enable global message passing. Specifically, we introduce two key components tailored for 3D SGG: (1) a Graph Embedding Layer++ that efficiently integrates edge-aware global context with linear computational complexity, and (2) a Semantic Injection Layer++ that enriches visual features with linguistic priors from large language models (LLMs) and vision-language models (VLMs), boosting semantic representation without introducing extra trainable parameters. To further address the practical challenge of incremental SGG (I-SGG), where new relationship categories arrive sequentially, we equip SGFormer++ with a novel Spatial-guided Feature Adapter, which calibrates predicate features using subject-object spatial geometry to counter scale variation, and a Cascaded Binary Prediction Head that mitigates catastrophic forgetting via task-incremental classifier expansion and logit distillation. Extensive experiments on the 3DSSG benchmark demonstrate that SGFormer++ achieves state-of-the-art performance in both standard and incremental settings: it yields a significant 4.49% absolute improvement in Predicate A@1 under the incremental setting. Code and data are available at: https://github.com/Andy20178/SGFormer.

- **整体含义**：论文提出 SGFormer++，以 Transformer 代替 GCN 作为骨干，结合语义先验注入与增量学习机制，实现更鲁棒、可演进的 3D 场景图生成。

## 2. 方法论

### 2.1 核心思想
以 **Transformer 层**构建图消息传递网络，捕获全场景长程依赖；同时利用 **大型语言模型（LLM）与视觉-语言模型（VLM）** 提供语义先验，增强特征表示；针对增量场景，设计**特征适配**与**分类器隔离**策略对抗灾难性遗忘。

### 2.2 关键技术细节

#### (a) 图嵌入层++（Graph Embedding Layer++, GEL++）
- 将**节点特征**（物体）与**边特征**（关系）投影到统一维度，通过 **多头边感知自注意力** 实现消息传递。  
- 边特征不仅作为注意力计算的调制项，还通过 **空间引导特征适配器** 显式注入几何先验（物体的 3D 包围盒中心偏移与尺寸差异），校准边表示，缓解尺度变化与特征漂移。  
- 整体计算复杂度为 **线性**，可高效处理大规模边。

#### (b) 语义注入层++（Semantic Injection Layer++, SIL++）
- 利用 **VLM**（如 Qwen3-VL）从多视角 2D 图像生成**场景特定**文字描述，并通过 CLIP 文本编码器获取语义嵌入。  
- 使用 **交叉注意力**，以节点视觉特征为 Query，语义嵌入为 Key/Value，将文本知识动态注入节点表示中，不引入额外可训练参数。  
- 相比原 SIL（仅用静态类级 LLM 描述），SIL++ 可捕捉场景上下文，尤其实用于长尾和零样本关系类别。

#### (c) 级联二值预测头（Cascaded Binary Prediction Head, CBPH）用于增量学习
- 将谓词多分类分解为一系列**独立的二值分类器**（每个谓词类别一个）。  
- 新任务到达时，仅新增的二值分类器可被更新，历史分类器参数**冻结**，实现参数隔离。  
- 对冻结的旧分类器，通过 **二元知识蒸馏损失（BKD）** 维持输出一致性；同时借助空间引导适配器**重新对齐**漂移后的特征，保证蒸馏的有效性。  
- 推理时，所有任务的二值分类器并联输出，统一判定关系类别。

### 2.3 训练与推理
- 标准 SGG：使用焦点损失（Focal Loss）处理类别不平衡，联合优化节点、谓词分类与语义对齐损失。  
- 增量 SGG：在标准损失基础上，加上**空间适配蒸馏损失** \( \mathcal{L}_{\text{BKD-A}} \)，通过冻结旧分类器与历史 logit 对齐缓解遗忘。  
- 推理时，关系置信度通过 sigmoid 并采用阈值 0.5 筛选。

## 6. 主要结论与发现

- **标准 SGG 性能最优**：SGFormer++ 在 3DSSG 上取得最佳物体、谓词、三元组准确率，尤其谓词 A@1 达到 88.59%，明显优于基于 GCN 的方法和原 SGFormer。

- **模型可堆叠且高效**：GEL++ 边感知注意力与适配器引入的计算开销极低，深度增加时仍能保持性价比较高。

## 7. 优点

- **全局建模能力强**：用 Transformer 替代 GCN，解决过度平滑与局部限制，实现全图长程消息传递。  
- **多模态语义注入巧妙**：VLM 生成的场景特定文本嵌入以无参数方式丰富视觉特征，不增加模型容量却大幅提升语义理解。  
- **增量学习方案新颖且实用**：  
  - 二值预测头实现参数级别隔离，无需回放旧样本。  
  - 空间适配器利用任务不变的几何信息缓解特征漂移，使蒸馏更稳定。  
- **实验扎实全面**：多设置、多指标、多对比，验证充分，代码开源，复现性强。

## 8. 不足与局限

- **增量范围受限**：增量学习仅针对新增**关系类别**，假设物体类别固定不变；未处理物体类别增量或跨域动态场景。
