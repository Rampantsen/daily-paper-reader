## 摘要
三维高斯泼溅（3DGS）已处于三维场景重建的前沿。将语言驱动的开放词汇理解扩展到3DGS，在具身人工智能等现实应用中备受关注。近期方法通过为每个实例学习特征属性，并将高维对比语言-图像预训练（CLIP）特征直接蒸馏到场景表示中来赋予语义。然而，这些方法的实例分组机制要么需要预定义实例数量，要么受其自底向上分组策略的噪声影响。此外，对CLIP的依赖将语义理解限制在简单名词短语，阻碍了复杂的空间推理和指称表达定位。我们提出GaussDet，一种通过利用具有指称表达能力的离散、开放词汇二维物体检测器来规避对密集CLIP特征需求的方法。我们为单个高斯学习实例特征，将场景分解为三维实例组。通过渲染这些组并从多视图二维检测中聚合语义投票，我们为每个三维实例生成一个鲁棒的视图聚合语义标签分布（VASD）。这种视图聚合策略起到了强正则化作用，减弱了由低质量实例分组引起的虚假标签。我们的方法能够直接从简单语言查询零样本扩展到复杂的指称定位。在两个关键任务——开放词汇分割（LeRF-OVS、ScanNet）和指称表达定位（Ref-LeRF）——上的广泛评估表明，GaussDet相对于现有方法取得了一致的改进。最值得注意的是，在严格的零样本设定下，我们在指称定位中实现了显著的16.7% mIoU提升。

## Abstract
3D Gaussian Splatting (3DGS) has emerged at the forefront of 3D scene reconstruction. Extending 3DGS with language-driven, open-vocabulary understanding has gained significant attention for real-world applications such as embodied AI. Recent methods achieve this by learning an instance feature attribute and assigning semantics by distilling high-dimensional Contrastive Language-Image Pretraining (CLIP) features directly into the scene representation. However, the instance grouping mechanisms of these methods either require a predefined number of instances or suffer from noise in their bottom-up grouping strategies. Furthermore, the reliance on CLIP restricts semantic understanding to simple noun phrases, preventing complex spatial reasoning and referential expression grounding. We present GaussDet, a method that circumvents the need for dense CLIP features by leveraging discrete, open-vocabulary 2D object detectors with referring expression capabilities. We learn instance features for individual Gaussians to decompose the scene into 3D instance groups. By rendering these groups and aggregating semantic votes from multi-view 2D detections, we generate a robust View-Aggregated Semantic Label Distribution (VASD) for each 3D instance. This view-aggregation strategy acts as a strong regularizer, attenuating spurious labels caused by low-quality instance grouping. Our approach enables a straightforward, zero-shot extension from simple language queries to complex referential grounding. Extensive evaluations across two key tasks -- open-vocabulary segmentation (LeRF-OVS, ScanNet) and referring expression grounding (Ref-LeRF) -- demonstrate that GaussDet achieves consistent improvements over existing methods. Most notably, we achieve a substantial 16.7% mIoU improvement in referential grounding within a strict zero-shot setting.

- **背景**：3D高斯泼溅（3DGS）已广泛用于场景重建，但将其扩展至开放词汇语义理解面临瓶颈。现有方法（如 OpenGaussian、LaGa）通过为高斯泼溅学习实例特征，并依赖从 SAM 掩膜提取的密集 CLIP 特征进行语义分配。
- **痛点**：
  - **实例分组噪声**：自底向上的分组策略（如 HDBSCAN 聚类）常导致欠分割、错误关联或背景渗入。
  - **语义表达受限**：密集 CLIP 特征仅能处理简单名词短语，难以支撑复杂的空间推理和指称表达（referring expression）定位。
  - **视图一致性问题**：孤立掩膜的 CLIP 特征丢失了周围图像上下文，导致语义描述力低。
- **整体含义**：提出 **GaussDet**，用离散、开放词汇的 2D 物体检测器取代密集 CLIP 特征，解决上述局限，并将 3DGS 的开放词汇能力无缝扩展到零样本指称表达定位。

## 2. 方法论
### 2.1 总体流程
- **场景表示**：在标准 3DGS 基础上，为每个高斯点添加可学习的实例特征（维度 6 或 32，依场景定），使用 SAM 掩膜监督的对比损失训练。
- **实例分组**：沿用 LaGa 的 2D-3D 关联聚类机制，得到 `Ns` 个 3D 实例组 `Gi`。
- **语义标签图生成**：利用开放词汇 2D 检测器（如 Qwen-VL 2.5、YOLO-World）对每帧图像生成离散语义标签图 `Sv`，其中每个检测框按尺寸加权渲染，背景像素标记为 -1。
- **视图筛选**：对每个实例组 `Gi`，基于高斯点的低透明度和显著性阈值过滤，计算掩膜可见度分数 `si,v`，选出 top-K 个最佳视图 `Ki`。
- **视图聚合语义标签分布（VASD）**：
  - 将 `Gi` 投影至 `Ki` 的各视图，提取有效像素点的语义标签序列 `ℓi` 和对应高斯显著性分数序列 `qi`。
  - 公式：`ℓi = ⋃_{v∈Ki} Sv(ˆP^v_i)`, `qi = ⋃_{v∈Ki} Qv(τv(Gi))`
  - 以加权求和的方式汇总各类别出现频次，得到该实例组的语义分布。
- **语义背景正则化（SBR）**：在 VASD 中显式保留背景标签，避免错误地将背景高斯点强制赋予前景语义，从而抑制假阳性。

### 2.2 关键设计
- **检测器驱动的语义分配**：直接使用离散的物体类别标签，而非 CLIP 连续嵌入，提高了语义精确性和指称表达能力。
- **多视图投票与聚合**：通过 top-K 视图的标签投票，天然具备对错误分组和单帧检测噪声的鲁棒性。
- **即插即用特性**：VASD 可作为模块集成到不同的实例分组方法上（OpenGaussian、LaGa），无需改动原流程。

## 7. 优点
- **摆脱对密集 CLIP 的依赖**：使用 2D 检测器，语义更精确、描述力更强，天然支持复杂短语和指称表达。
- **强大的抗噪声能力**：VASD 显式引入背景标签并利用多视图投票，对实例分组误差、检测虚警和遮挡均有强正则化效果。
- **零样本泛化**：无需任何指称表达微调或 3D-文本匹配网络，即可完成复杂定位，应用门槛极低。
- **模块化与复用性**：语义分配与实例分组解耦，可直接融入现有框架，提升所有基于 SAM+CLIP 的方法。
- **高效实现**：在单张消费级显卡上运行，推理阶段计算开销小，实用性高。

## 8. 不足与局限
- **2D 检测器天花板效应**：最终性能受限于所选检测器的能力，尤其在处理高度遮挡或视角不全时，预测错误会传播至 3D 语义。
- **实例分组基座依赖**：仍沿用 LaGa 的 2D-3D 关联聚类，当场景分解质量极差时（如严重的欠分割），VASD 虽能缓解，但无法完全纠正。
- **端到端优化缺失**：实例特征学习和语义分配为两阶段，未能联合优化以相互促进。
- **视角覆盖要求**：对于重叠严重且观测视角匮乏的物体，分布可能不均衡，导致分割不完整。
- **场景类型有限**：仅在室内的 LeRF/ScanNet 场景上评估，未涉及户外或大规模自动驾驶场景，通用性有待验证。
