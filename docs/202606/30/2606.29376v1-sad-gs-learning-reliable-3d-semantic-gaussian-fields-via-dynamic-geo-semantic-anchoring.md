## 摘要
开放词汇的三维语义高斯场学习依赖于多视角二维监督，但其语义目标和空间分配往往不可靠。在不同视角下，视角依赖的特征导致语义身份漂移，而传播的跟踪器掩码引入边界泄漏和身份切换。直接针对这些不可靠的二维目标进行优化会迫使三维表征吸收多视角矛盾，导致严重的误差累积。为解决这一局限性，我们提出 SAD-GS，一个通过动态地理语义锚定学习可靠三维语义高斯场的框架。具体而言，语义锚点蒸馏（SAD）将每个视角的视觉嵌入蒸馏为共识文本锚点，以建立视角不变的语义身份。同时，地理语义反馈循环（GSFL）利用逐步演化的三维场来主动过滤跟踪器异常，并通过保守的三门更新规则细化空间掩码分配。在 LERF-OVS、3D-OVS 和 Mip-NeRF360 上的广泛评估表明，SAD-GS 在开放词汇定位和语义分割两方面始终取得最佳整体性能。这些全面改进验证了动态地理语义锚定在可靠三维语义高斯场学习中的有效性和鲁棒性。

## Abstract
Open-vocabulary 3D semantic Gaussian field learning relies on multi-view 2D supervision, whose semantic targets and spatial assignments are often unreliable. Across varying viewpoints, view-dependent features cause semantic identity drift, while propagated tracker masks introduce boundary leakage and identity switches. Directly optimizing against these unreliable 2D targets forces the 3D representation to absorb multi-view contradictions, leading to severe error accumulation. To resolve this limitation, we propose SAD-GS, a framework for learning reliable 3D semantic Gaussian fields via dynamic geo-semantic anchoring. Specifically, Semantic Anchor Distillation (SAD) distills per-view visual embeddings into consensus text anchors to establish a viewpoint-invariant semantic identity. Concurrently, the Geo-Semantic Feedback Loop (GSFL) leverages the evolving 3D field to actively filter tracker anomalies and refine spatial mask assignments via a conservative three-gate update rule. Extensive evaluations on LERF-OVS, 3D-OVS, and Mip-NeRF360 show that SAD-GS consistently achieves the best overall performance in both open-vocabulary localization and semantic segmentation. These comprehensive improvements validate the effectiveness and robustness of dynamic geo-semantic anchoring for reliable 3D semantic Gaussian field learning.

直接将这些不可靠的二维伪标签回传优化三维场，会迫使三维表征拟合多视图间的相互矛盾，导致严重的误差累积，使下游的开放词汇定位和分割性能大幅下降。因此，**如何从充满噪声的多视角二维监督中学习出可靠的三维语义场**，是本文要解决的核心科学挑战。

### 2. 方法论：SAD-GS 框架

本文提出 **SAD-GS**，通过**动态地理语义锚定**，将语义身份和空间分配的可靠性问题**解耦**，并分别进行稳定和净化。

#### 核心思想
*   **语义锚点蒸馏 (SAD)**：将视角波动的视觉嵌入蒸馏为稳定、视角不变的共识文本锚点，解决语义身份漂移。
*   **地理语义反馈循环 (GSFL)**：利用演化中的三维场作为几何-语义校验器，对二维跟踪掩码进行动态可信度评估和选择性的保守修正，解决空间标签噪声。

#### 关键技术细节

**A. 语义锚点蒸馏 (SAD)**
1.  **共识锚点生成**：对每个被跟踪对象 `k`，采样多视角图像块，用多模态大模型 Qwen3-VL 生成文本描述。将描述编码为 CLIP 文本嵌入，过滤掉与初始均值相似度低于阈值 `τ` 的离群描述后，加权聚合得到该对象的共识文本锚点 `a_k`。
2.  **投影与蒸馏**：三维高斯的语义特征 `f_i` 为 16 维向量。渲染后，通过一个可学习的映射头 `ϕ: R^16 → R^512` 将其投影到 CLIP 空间。蒸馏损失 `L_sad` 直接最小化投影后的渲染特征 `ϕ(F_{u,v})` 与其对应跟踪标签 `M_{2D}` 的共识锚点 `a_{M_{2D}(u,v)}` 之间的 L2 距离和余弦距离，使得三维场学习到视角不变的语义。

**B. 地理语义反馈循环 (GSFL)**  
GSFL 在 SAD 完成了语义热身（Twarmup=5,000 次迭代）后启动，核心是**三门伪标签修正规则**。
1.  **地理语义一致性评分 (C)**：  
    对每个像素 `(u,v)`，计算其渲染特征与当前跟踪标签锚点的余弦相似度，再乘以一个由局部深度梯度 `|∇D|` 导出的几何平滑惩罚项 `exp(-λ|∇D|)`。该分数综合了语义和几何可信度。
2.  **三门规则**：仅对 **C < τ_conf**（门1：置信度门）的低置信度像素执行以下修正尝试。遍历所有锚点，找到相似度最高的候选标签 `k*`。只有当候选满足以下两个条件时，才用 `k*` 替换原始跟踪标签：
    *   **门2：接受门**：其绝对相似度 `cos(F̂_{u,v}, a_{k*}) ≥ τ_acc`，防止在遮挡等弱证据下强制修正。
    *   **门3：间隔门**：它与第二高相似度的 `s_{(2)}` 的差值 `s_{k*} - s_{(2)} ≥ δ`，防止在多个相近类别间的模糊性中错误修正。
3.  **锚点更新**：基于修正后的掩码，对每个锚点 `a_k` 用指数移动平均 (EMA) 进行更新，使其适配演化中的三维特征分布，但仅使用跟踪掩码区域的特征，避免自预测循环。

最终，三维场仅使用经过 GSFL 修正后的掩码 `M'` 和 `L_sad` 损失进行优化。

### 6. 主要结论与发现
*   SAD-GS 通过稳定监督信号，在三个基准数据集上的开放词汇定位和分割任务中，**一致地取得了最佳性能**，在极具挑战性的 `ramen` 场景上将定位准确率从 74.7% 提升至 **85.8%**，分割 mIoU 从 51.8% 提升至 **77.9%**。

### 7. 优点
*   **新颖的可靠性视角**：首次将开放词汇三维语义学习形式化为一个监督可靠性问题，并识别出两个结构性的故障模式，角度犀利且深刻。
*   **优效的问题解耦框架**：SAD 和 GSFL 分别解决了视觉特征漂移和空间标签噪声，结构清晰，相互配合，相比“端到端”拟合噪声伪标签的范式，是更根本的改进思路。
*   **稳健且可解释的设计**：GSFL 的三门修正规则动机明确，每个门对应一种典型的修正失败模式，并通过消融实验和语义空间可视化得到有力证明。
*   **全面的实验验证**：不仅包含多基准的横向对比，还通过收敛动态分析、解码鲁棒性分析和语义空间结构分析，对系统行为进行了深入刻画。

### 8. 不足与局限
*   **饱和性**：GSFL 的修正增益在严重空间重叠或复杂容器关系的场景下趋于饱和，因为单纯的文本锚定在此情况下提供的监督信息不完整。
*   **静态阈值的局限性**：当许多语义类别在 CLIP 空间中高度聚集时，固定的全局间隔阈值 `δ` 的区分能力下降。
*   **跟踪器依赖**：锚点更新和初始掩码仍依赖外部跟踪器（DEVA），其基础性能仍是整个流程的天花板。未来可探索融入时空先验或自适应决策边界来缓解这些问题。
