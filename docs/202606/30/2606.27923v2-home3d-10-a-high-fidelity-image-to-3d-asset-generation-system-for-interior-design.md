## 摘要
我们提出Home3D 1.0，一个模块化的图像到3D生成系统，能够从单张参考图像生成高质量3D资产，面向室内设计和电子商务应用。给定家具或装饰品的照片，该系统输出一个带有基于物理渲染（PBR）材质的网格，并且该网格可以分解为特定于材质的组件。该流程组织为四个紧密耦合的模块：几何模块通过几何VAE和由粗到细的流匹配DiT进行隐式SDF建模，重建出水密网格；纹理模块预测多视角反照率观测，将其重投影到网格上，并用3D纹理场补全未观察到的表面区域；材质模块使用MatWeaver通过基于视频的分割和UV空间投票获得组件掩码，然后通过分层多模态匹配从精选的材质库中检索并烘焙PBR贴图；部件模块利用PartVAE和PartDiT生成可编辑材质的语义部件网格，一次性解码多头部件特定的SDF场。每个模块均使用专用指标独立评估，既展示了当前系统的能力，也指出了通往更广泛部署仍存在的差距。

## Abstract
We present Home3D 1.0, a modular image-to-3D generation system that produces high-quality 3D assets from a single reference image, targeting interior design and e-commerce applications. Given a photograph of a furniture or decor item, the system outputs a mesh with physically-based rendering (PBR) materials, and the mesh can be decomposed into material-specific components. The pipeline is organized into four tightly coupled modules: Geometry reconstructs a watertight mesh through latent SDF modelling with a geometry VAE and a coarse-to-fine flow-matching DiT; Texture predicts multiview albedo observations, reprojects them onto the mesh, and completes unseen surface regions with a 3D texture field; Material uses MatWeaver to obtain component masks through video-based segmentation and UV-space voting, then retrieves and bakes PBR maps from a curated material library through hierarchical multi-modal matching; and Parts generates material-editable semantic part meshes with a PartVAE and PartDiT, decoding multi-head part-specific SDF fields in one pass. Each module is evaluated independently with dedicated metrics, highlighting both the current system capability and the remaining gaps toward broader deployment.

### 2. 方法论
系统由四个前后耦合且部分并行的模块组成：
- **2.1 几何重建（Geometry）**  
  - **核心思想**：基于隐式SDF的VAE + 由粗到细的流匹配DiT，恢复紧凑且全局一致的形状并进一步细化高频细节。
  - **关键技术**：
    - 几何VAE：编码器以点云（含位置、法线、锐边指示）为输入，通过Perceiver式交叉注意力聚合全局与局部几何特征，得到VecSet潜变量 \(Z\)；解码器将潜变量 \(Z\) 映射为连续SDF，用Marching Cubes提取网格。
    - 流匹配DiT：第一阶段（粗）基于图像条件生成粗潜变量，解码后体素化并嵌入RoPE位置编码作为空间先验；第二阶段（精）结合图像特征和空间先验预测精细潜变量，最终解码出水密网格。
  - **数据**：混合ObjaverseXL和自有的天猫设计家供应商生产级家具模型，分大规模预训练集和高质量微调集。
- **2.2 纹理生成（Texture）**  
  - **核心思想**：从参考图像和几何条件同时预测多视角反照率图，经重投影到网格后，用3D纹理场补全遮挡区域。
  - **流程**：多视角Albedo预测 → 按相机投影贴回网格（获得不完整纹理） → 3D纹理补全模块直接在表面上色，屏蔽UV拓扑和视角局限。
  - **训练**：采用多视角训练、光照增强、扩散损失、一致性正则和梯度正则等，以分离固有色与光照。
- **2.3 部件感知PBR材质检索（Material）**  
  - **核心思想**：不直接生成PBR贴图，而是将网格分割为材质区域，从预构建的专家材质库中分层检索高精度PBR材质并烘焙成引擎就绪纹理。
  - **MatWeaver分割**：将3D部件分割形式化为视频跟踪任务。利用SAM2在合成视频序列中分割材质组件，通过UV−Face Atlas和共识投票将2D掩码映射到3D面，获得拓扑清洁的分割结果。
  - **多层多模态检索**：
    1. 用RoMA匹配输入图像与合成渲染进行姿态估计，实现像素级2D-3D对齐。
    2. 对每个部件，先用VLM（视觉语言模型）推理材质类别和描述。
    3. 通过Qwen3-VL跨模态嵌入在特定类别材质库中检索Top-5候选。
    4. 用另一个冻结VLM作为专家裁判重排序，选出最佳材质。
    5. 物理烘焙：将每个部件的金属度、粗糙度、法线等PBR属性合并为统一的、经过Draco压缩的生产级纹理贴图。
  - **材质数据库构建**：专家策划的多级分类，通过GPT-5.2生成初始描述，Qwen3-Embedding聚类，Gemini进行对比重写，使每个材质条目的描述高区分度且语义一致。
- **2.4 材质感知部件生成（Parts）**  
  - **核心思想**：用一个固定大小的共享潜变量一次性生成多个部件的SDF场，避免部件数量增多时解码开销线性增长。
  - **PartVAE**：编码器输入带有部件ID的点云（位置+法线+部件嵌入），经点交叉注意力瓶颈输出共享潜变量；解码器为多头SDF，在单次前向传播中输出多个部件专属SDF通道，通过Marching Cubes提取各部件网格。
  - **PartDiT**：以DINOv2-Large图像token和完整物体的几何潜变量为条件，用流匹配从噪声生成潜变量，解耦出材质可编辑的部件。
  - **训练**：三阶段（大规模预训练→继续训练CT改善视觉-材质映射→监督微调SFT提升边界质量和一致性）。

### 3. 实验设计
- **基准数据集**：一个精心整理的100个家具案例集合，每个案例包含参考图像、设计师制作的3D真实模型（源自3D-FRONT同源数据）。
- **对比方法**：三个闭源图像到3D系统——Hunyuan3D-3.0、Seed3D 1.0、Tripo 3.0。
- **评价指标**：
  - 几何：Chamfer Distance（CD×10⁻³，↓）、Earth Mover’s Distance（EMD×10⁻²，↓）、F1@0.01（↑）。
  - 纹理与材质：CLIP-I（↑）和LPIPS（↓），基于固定视角PBR渲染。

### 6. 主要结论与发现
- Home3D 1.0 通过几何、纹理、材质、部件四阶段紧密协作，实现了从单张家具照片到高保真、材质可编辑3D资产的端到端生成。
- 隐式SDF的由粗到细生成策略和3D纹理补全可显著改善弱可见区域的几何完整性和表面细节。
- 基于检索的PBR材质方案比直接生成立即展现出更高的物理准确性和生产就绪性；视频分割与UV投票实现了无需标注的语义材质分割。
- 多头SDF部件解码以固定成本生成语义部件，比逐步生成方案更高效，便于设计人员组件级材质替换。

### 7. 优点
- **模块化解耦设计**：每个模块集中解决家具生成中的特定失败模式，便于独立优化和升级。
- **工业级生产考虑**：输出为水密网格、引擎就绪PBR贴图并包含减面压缩，直接适配电商与室内设计管线。
- **材质保真与可编辑性**：放弃直接生成所有PBR通道，改用检索 + 烘焙，保证材质身份精确且不依赖高成本生成模型。
- **部件生成的创新**：共享潜变量 + 多头SDF解码，避免部件数量增加带来的时序开销，同时实现材质感知的语义分解。
- **无监督材质分割**：MatWeaver利用视频SAM2跟踪和UV投票，无需人工标注即可获得高质量3D面级分割。

### 8. 不足与局限
- **实验规模有限**：评估仅使用100个家具案例，未能在大规模、多类别产品数据集上验证泛化性。
- **缺乏离线工业性能指标**：未报告生成一张图到完整资产的平均耗时、内存占用上限、大规模生产吞吐量等工程指标。
- **模块间独立评估未能揭示串联误差**：各模块单独评估，缺少流水线端到端误差传播分析（如几何偏差如何影响后续材质检索）。
- **关闭源对比偏差风险**：对比系统来自不同团队，训练数据、模型规模未对齐，可能会有不公平因素；且自身组件也部分依赖预先训练的闭源模型（如VLM）。
- **鲁棒性与扩展性受限**：文中指出未来需解决跨阶段反馈、铰接和可变形物体扩展，当前系统可能难以处理高透明度、高反光或结构复杂（如复杂灯具、藤编）的家具。
- **部件分解固定槽位限制**：PartVAE使用固定数量（8个）部件槽，可能不适用于零件数目差异极大的物体，多槽位浪费或不足。
- **材质检索仍依赖手工数据库**：若偏离预设材料类别（如非常规合成材料），可能检索失败或误匹配。
