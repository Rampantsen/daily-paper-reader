## 摘要
我们提出了MoVerse，一种实时视频世界模型，可从单张窄视场图像创建交互式可导航场景。该设置极具挑战性，因为输入仅观察到环境的一小部分，而交互式漫游需要完整的周围世界、持久的几何结构、可控的相机运动以及时间连贯的高保真观测。MoVerse通过将世界构建与观测渲染分离来解决此问题。它首先利用拓扑感知扩散将输入扩展为重力对齐的360°全景图，在三维推理前闭合缺失的视场。然后，使用全景几何感知残差预测将全景图提升为持久的三维高斯脚手架，产生密集且可直接渲染的空间记忆。最后，高斯条件的视频渲染器将沿用户指定相机轨迹的脚手架渲染转化为照片级真实感视频。为使该渲染器在交互中实用，我们训练了一个双向扩散教师模型用于高质量条件渲染，并将其蒸馏为一个因果自回归学生模型以实现有界延迟流式传输。该设计结合了显式三维表示的控制性和长期连贯性以及生成式视频模型的感知质量。MoVerse在单个NVIDIA RTX 4090 GPU上支持每秒8帧的实时场景漫游，展示了实现交互式视频输出的单图像世界创建的可行路径。

## Abstract
We present MoVerse, a real-time video world model that creates an interactively navigable scene from a single narrow-field-of-view image. This setting is challenging because the input observes only a small fraction of the environment, while interactive roaming requires a complete surrounding world, persistent geometry, controllable camera motion, and temporally coherent high-fidelity observations. MoVerse addresses this problem by separating world construction from observation rendering. It first expands the input into a gravity-aligned 360$^\circ$ panorama with topology-aware diffusion, closing the missing field of view before 3D reasoning. It then lifts the panorama into a persistent 3D Gaussian scaffold using panoramic geometry-aware residual prediction, yielding a dense and directly renderable spatial memory. Finally, a Gaussian-conditioned video renderer translates scaffold renderings along user-specified camera trajectories into photorealistic video. To make this renderer practical for interaction, we train a bidirectional diffusion teacher for high-quality conditional rendering and distill it into a causal autoregressive student for bounded-latency streaming. This design combines the controllability and long-range consistency of explicit 3D representations with the perceptual quality of generative video models. MoVerse supports real-time scene roaming at 8~FPS on a single NVIDIA RTX~4090 GPU, demonstrating a practical path toward single-image world creation with interactive video output.

## 2. 方法论

MoVerse 采用三阶段流水线，各阶段角色分明。

- **阶段一：几何感知全景生成（I → P）**  
  将输入窄视场图像扩展为重力对齐的 360° 等距柱状全景图。  
  - **重力对齐与规范空间定义**：通过**可微分自动水平校准**估计二维密集对应场，并用**刚性变换瓶颈**（可微 soft-argmin 求解器）限制为全局相机旋转，消除失调，将输入映射到水平的规范全景坐标系中。  
  - **拓扑感知补全**：在潜在扩散模型中采用**圆形潜在编码**和**水平移位等变生成**，确保全景图左右边界在 S¹ 拓扑上无缝闭合；训练时引入移位一致性损失。  
  - **数据集与训练**：使用 **Horizon360** 数据集，其中原始全景图先规范化（使地平线与赤道对齐，垂线保持铅垂），再对每张全景图采样不同姿态的透视视图，并以输入视图为中心滚动全景图目标，使已知区域远离周期边界。训练目标结合标准去噪损失、移位损失和光流辅助损失。

- **阶段二：全景高斯脚手架生成（P → G）**  
  将全景图提升为可直接渲染的 3D 高斯泼溅（3DGS）场景。  
  - **全景高斯初始化**：利用估计的深度图，按球形反投影初始化高斯中心，并使用**纬度感知尺度修正**（尺度 ∝ cos φ），使高斯足迹符合等距柱状投影的变形特性。  
  - **残差预测与组合**：在角度 – 逆深度空间中预测高斯参数的残差（∆θ, ∆ϕ, ∆z 等），应用残差后重新投影至三维空间，保证更新与 ERP 几何及水平闭合一致。预测头采用零初始化，确保训练从物理合理的初始解开始只学习局部修正。  
  - **训练目标**：在 **HM3D** 数据集上训练，同时优化目标视图的 L1、LPIPS 和 SSIM 损失，并加入源视图重建损失、逆深度损失以及位置残差的 Hinge 正则化。

- **阶段三：高斯条件视频渲染与实时流式传输（ˆV → V）**  
  将沿用户相机轨迹渲染的脚手架图像流，转化为照片级真实感视频。  
  - **双向教师模型**：基于 **Wan2.1-T2V-1.3B** 微调，以高斯渲染帧作为条件，拼接目标与条件 VAE 隐变量，使用**共享旋转位置编码（RoPE）** 使条件与目标在 Transformer 空间中位置对齐，训练流匹配损失。教师可看到完整窗口，产生高质量视频。  
  - **因果自回归学生模型**：将教师蒸馏为学生，采用 **Self‑Forcing** 和 **DMD（RAVEN）** 策略，学生以自回归方式逐块生成，仅依赖当前条件帧和历史 KV 缓存。使用 **MemRoPE** 管理缓存（一个 Sink 帧、一个长程滑动平均 token、三个局部帧窗口），保证无限长轨迹的实时推演。  
  - **训练数据混合策略**：设计三阶段课程：1) 单视图深度投影合成的轨迹对齐代理渲染；2) 稀疏关键帧重建的高斯渲染视频；3) 全景条件高斯脚手架渲染视频。逐级提供更具部署真实性的条件信号。  
  - **实时推理**：采用两阶段去噪和 **TAEHV** 快速解码器，最终在 RTX 4090 上实现 8 FPS 的端到端漫游。

## 6. 主要结论与发现

- 将显式三维几何（全景高斯脚手架）与因果生成式视频渲染分离，是实现单图像实时世界导航的有效范式。
