## 摘要
从单目视频重建动态场景仍是计算机视觉的根本挑战。现有前馈式方法为每帧逐像素预测3D高斯，面临高斯重复和视角依赖偏差，阻碍了场景运动的有效学习。我们提出C4G，一个基于紧致时间戳条件可学习高斯查询令牌的前馈式4D重建框架。每个令牌在完整时间上下文中聚合相应特征，并解码一个位置受目标时间戳调制的3D高斯，从而无需逐场景优化即可实现全局一致的运动建模。为捕捉细粒度细节，我们进一步引入基于视频扩散模型的渲染增强模块。由于我们的框架能有效将特征聚合到高斯中，我们将此能力扩展到特征提升，产生支持点跟踪和动态场景理解的4D特征场。C4G在使用显著更少的高斯且无需相机位姿的情况下，实现了较强的新视角合成性能，同时展现出更强的运动建模能力与对大时间间隔的鲁棒性。

## Abstract
Dynamic scene reconstruction from monocular video remains a fundamental challenge in computer vision. Existing feed-forward methods predict 3D Gaussians pixel-wise for each frame, suffering from duplicated Gaussians and view-dependent biases that hinder effective learning of scene motion. We present C4G, a feed-forward 4D reconstruction framework built upon a compact set of timestamp-conditioned learnable Gaussian query tokens. Each token aggregates corresponding features across the full temporal context and decodes a 3D Gaussian whose position is modulated by the target timestamp, enabling globally coherent motion modeling without per-scene optimization. To capture fine-grained details, we further introduce a video diffusion model-based rendering enhancement module. Since our framework effectively aggregates features into Gaussians, we extend this capability to feature lifting, producing a 4D feature field that supports point tracking and dynamic scene understanding. C4G achieves strong novel-view synthesis performance using significantly fewer Gaussians and without requiring camera poses, while exhibiting stronger motion modeling and robustness to large temporal gaps.

## 1. 论文的核心问题与整体含义
- **研究背景**：从单目视频中重建动态 4D 场景是计算机视觉的基本挑战，对于 AR/VR、机器人和内容生成具有重要价值。
- **现有方法的痛点**：当前主流的**前馈式动态高斯泼溅（4DGS）** 方法大多采用**逐像素高斯预测**，即每一帧的每个像素都独立生成高斯。这导致两个严重问题：
  - **高斯重复与鬼影**：在时间插值处，来自相邻输入帧的高斯会重复出现，造成视觉重影。
  - **视角依赖偏差**：渲染时过度依赖时间上最接近的输入帧，无法利用远帧信息填补遮挡区域，难以学习真正的全局场景运动。
- **核心含义**：本文通过**设计上的根本改变**——用一组紧凑的可学习查询令牌取代逐像素预测，迫使模型学习**全局连贯的运动表示**，同时保持极高的效率。

## 2. 论文提出的方法论
- **总体框架**：C4G（Compact 4D Representation with Gaussians），一种**不需要相机位姿**、前馈式的 4D 重建框架。
- **核心模块一：紧凑 4D 高斯重建**
  - **视觉特征提取**：使用预训练的几何基础模型 VGGT 提取多帧图像特征 `Ft`。
  - **时间戳条件化**：为每个输入帧的特征注入归一化的**时间戳嵌入**，同时使可学习的查询令牌 `Q` 以**目标时间戳** `tb` 为条件。
  - **查询式高斯解码器** `DG`：
    - 固定数量 `N=2048` 的可学习查询令牌与所有帧的特征拼接，通过仅 2 层的 Transformer（全自注意力）进行交互。
    - 每个查询令牌聚合全局时间上下文后，通过一个 MLP 头解码为一个 3D 高斯（位置 `μ`、不透明度 `σ`、协方差 `Σ`、颜色 `c`）。
    - 通过改变目标时间戳 `tb`，同一组查询可瞬间生成任意时刻的紧凑高斯集合，实现动态重建。
  - **损失函数**：结合光度损失（MSE+LPIPS）、伪真值深度/法线损失（来自 MoGe-2）以及点跟踪损失（来自 CowTracker），以弥补动态场景下多视角一致性监督的缺失。
- **核心模块二：VDM 渲染增强**
  - 采用基于 **Wan2.1-VACE-1.3B** 的 ControlNet 架构，以渲染视频为目标、原始输入帧为参考，通过扩散模型修复细节、提升画质。
- **扩展模块：任意特征 4D 提升**
  - **观察到的特性**：解码器 `DG` 的注意力会自动形成“第一层找空间对应、第二层关注时间邻近帧”的**互补模式**。
  - **特征解码器** `DF`：复用 `DG` 的查询令牌和注意力模式，仅训练值投影层，可将任意 2D 基础模型（如 DINOv3、LSeg）的特征提升为附着在高斯上的 4D 特征场，用于点跟踪和语义理解。

## 6. 论文的主要结论与发现
- **范式优势**：用 **0.007× 甚至更少的高斯数量**，C4G 在无需相机位姿的条件下，在多个动态数据集上实现了**最优或极具竞争力的新视角合成性能**。
- **运动建模**：相较于逐像素方法，C4G 能够学习到**全局场景运动**，因此对**大时间间隔的输入极为鲁棒**，渲染质量下降更少。
- **涌现特性**：紧凑查询机制迫使 Transformer 自发出空间-时间互补的注意力模式，这种模式可被复用，实现了**首个前馈式 4D 特征场提升**，在点跟踪和场景理解上超越已有方法。

## 7. 优点
- **架构创新**：从逐像素预测转向**紧凑令牌预测**，从根本上解决了高斯重复和视角依赖偏差，重建出的场景几何更统一、运动更连贯。
- **高效率**：仅用 2048 个高斯即可表示完整动态场景，计算和存储成本极低。
- **通用性与扩展性**：框架天然支持无需位姿的重建，并可轻易扩展为包含任意语义特征的 4D 特征场，打通了从重建到理解的桥梁。
- **分析深刻**：通过对注意力模式的深入分析，不仅解释了模型为何有效，还为特征提升模块的设计提供了理论依据。

## 8. 不足与局限
- **重建范围受限**：
  - **空间上**：只能重建输入帧中**至少出现过一次**的区域，极端新视角下会出现空洞，依赖扩散模型填补（可能导致幻觉）。
  - **时间上**：模型**无法进行外推**，不能预测未来帧。
- **VDM 增强模块的局限**：虽然提升了画质，但扩散模型的训练仅基于合成数据集（Kubric），在真实场景的复杂纹理或超大运动中，提升效果和一致性可能会受限。
- **点跟踪的缺陷**：由于高斯非逐像素对齐，查询点可能与最近的高斯存在初始位置偏差，这在某些静态点密集的基准测试中会稍稍拉低分数。
