## 摘要
单目三维形状恢复是几何理解的基础，然而在任意视角和未见过的物体类别上实现鲁棒泛化仍是一个重大挑战。本文提出了一个可泛化的变形学习框架，通过显式地变形类别级形状模板以匹配目标观测，从而重建三维物体。为解决模板与目标之间的复杂形状变化，我们引入了几何引导的特征建模机制。该过程首先利用模板拓扑丰富基础特征以获得几何感知表示，然后将其与目标观测显式关联以引导精确变形。此外，为弥合固定模板与任意目标视角之间的差异，我们提出了视角自适应特征聚合模块。该模块利用多视图模板特征及其对应的相机姿态来丰富规范模板表示，确保无论目标视角如何都能实现鲁棒的特征对齐。大量实验表明，我们的方法在处理大形状变化和多样化视角方面显著优于现有最优方法，展现出对新类别的强大泛化能力，并有效支持下层的真实世界灵巧机器人操作任务。项目主页：https://GODeform.github.io/

## Abstract
Monocular 3D shape recovery is fundamental to geometric understanding, yet achieving robust generalization across arbitrary viewpoints and unseen object categories remains a significant challenge. In this paper, we present a generalizable deformation learning framework that reconstructs 3D objects by explicitly deforming a category-level shape template to match the target observation. To address complex shape variations between the template and the target, we introduce a geometry-guided feature modeling mechanism. This process first enriches foundation features with template topology to yield a geometry-aware representation, which is then explicitly correlated with the target observation to guide precise deformation. Furthermore, to bridge the disparity between the fixed template and arbitrary target views, we propose a view-adaptive feature aggregation module. This module leverages multi-view template features and their corresponding camera poses to enrich the canonical template representation, ensuring robust feature alignment regardless of the target's perspective. Extensive experiments demonstrate that our approach significantly outperforms state-of-the-art methods in handling large shape variations and diverse viewpoints, exhibiting strong generalization to novel categories and effectively supporting downstream real-world dexterous robotic manipulation tasks. # 论文提出的方法论
## 总体框架
- 将重建建模为**条件流匹配问题**：给定目标单目图像 \( I_T \) 和类别级三维模板 \( S \)，学习一个连续的变形映射 \( \Phi \)，将模板几何分布输运为目标形状 \( \hat{T} = \Phi(S | c) \)，其中条件 \( c = \{I_T, I_S\} \)。
- 采用整流流（rectified flow）的线性插值路径，训练网络预测速度场，推理时直接预测变形场 \( D = v_\theta(S, 0, c) \)，单步即可完成变形。

## 几何引导的基础特征建模
- **特征提升与传播**：先从模板可见点提取 DINOv3 的 2D 基础特征，再通过 3D 点云编码器获取几何嵌入，计算几何相似度矩阵，将可见点的语义特征加权扩散到整个模板点云（包括被遮挡区域），得到完备特征 \( F_{complete} \)。
- **语义感知的特征对齐**：将完备模板特征作为 query，目标图像的 2D 基础特征作为 key/value，通过**交叉注意力**建立模板点与目标图像局部之间的语义对应，生成对齐后的特征 \( F_{aligned} \)，并经自注意力精炼形成最终条件。

## 视角自适应特征聚合
- **主视图选择**：从多视角模板图像集中，选择与目标图像基础特征余弦相似度最高的视图作为主视图，其余为辅视图。
- **姿态感知特征调制**：计算辅视图相对于主视图的相机位姿变换，编码为相对位姿嵌入，并与对应视图的 2D 基础特征相加，实现特征空间的视角解耦。
- **跨视图特征融合**：以主视图的位姿调制特征为 query，所有视图调制特征为 key/value，通过交叉注意力融合互补信息，再经残差连接得到视角不变的局部特征，随后馈入几何引导的特征传播模块。

## 训练目标
- 总损失为：\( L = \lambda_{FM}L_{FM} + \lambda_{CD}L_{CD} + \lambda_{Lap}L_{Lap} + \lambda_{ARAP}L_{ARAP} + \lambda_{reg}L_{reg} + \lambda_{sil}L_{sil} \)，各项分别为：

- **视角自适应特征聚合**通过融合多视图模板特征和姿态信息，明显增强了模型对视角变化的鲁棒性。

# 优点
- **利用 2D 基础模型特征**，突破传统变形方法对类别泛化的限制，实现零样本式的新类别变形。
- **几何引导的特征传播**巧妙地将 2D 语义扩散到 3D 空间，处理遮挡区域。
- **视角自适应聚合**解耦外观与几何，大幅提升单目推断时的视角鲁棒性。
- **单步流匹配推理**，计算高效（变形约 0.67 秒），无需目标深度信息，适合机器人实时应用。
- **应用驱动**：直接保留稠密点对应，自然赋能下游任务（接触转移、抓取优化）。

# 不足与局限
- **单视图输入的固有病态性**：目标关键部位完全遮挡时，不可见部分的几何只能依赖模板结构，可能导致与真实形状的偏差。

- **实验范围**：训练集中在 ShapeNetv2 的 7 个类别，未见类别测试仅 OakInk 的 4 类，更广泛的开放世界物体和复杂背景下的表现有待验证。
- **真实抓取实验**：仅在手部、手掌和单一的人形机器人上测试，任务为简单的提升，未涉及更复杂的操作（如按、推等）。
- **数据集偏差**：依赖 DINOv3 特征的分布，可能受预训练数据偏差影响。
