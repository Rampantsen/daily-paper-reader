## 摘要
语义三维占用为自动驾驶和机器人决策提供了体素化的世界状态，但物体和稀有类别错误会影响自由空间解释、碰撞检测和时间状态传播。我们发现一种常见的视觉语言模型策略，即将三维体素或物体特征与裁剪-描述文本嵌入对齐，能提高文本空间相似性，但并未可靠地提升闭集占用的平均交并比。受此不匹配现象的推动，我们提出VISA，一种针对现有占用世界模型的训练时语义审计方法。VISA在离线视觉语言模型上查询每个物理物体实例的代表性裁剪图像，获得包含类别假设、合理混淆、可靠性、属性和证据的结构化审计，并沿物体轨迹传播。该审计被锚定到匹配的三维物体体素，并通过可靠性加权的类别层次、属性因子和场景级审计图损失蒸馏为语义logits，而推理过程保持不变且无需视觉语言模型。在nuScenes数据集上，三次运行的平均结果显示，VISA将OccWorld的mIoU从19.06提升至20.05，GaussianWorld从21.36提升至21.91；在GaussianWorld上，物体mIoU从18.18提高到19.16，稀有类别mIoU从15.60提高到16.79。这些结果表明，视觉语言模型更适合作为可靠性感知的语义审计器用于闭集占用任务，而非用作通用的描述-嵌入目标。

## Abstract
Semantic 3D occupancy provides a voxelized world state for autonomous driving and robot decision making, but object and rare-class errors can affect free-space interpretation, collision checking, and temporal state propagation. We show that a common VLM strategy, aligning 3D voxel or object features with crop-caption embeddings, improves text-space similarity without reliably improving closed-set occupancy mIoU. Motivated by this mismatch, we propose VISA, a training-time semantic auditing approach for existing occupancy world models. VISA queries an offline VLM on a representative crop of each physical object instance, obtains a structured audit with class hypotheses, plausible confusions, reliability, attributes, and evidence, and propagates it along the object track. The audit is grounded to matched 3D object voxels and distilled into semantic logits through reliability-weighted taxonomy, attribute-factor, and scene-level audit graph losses, while inference remains unchanged and requires no VLM. On nuScenes, averaged across three runs, VISA improves OccWorld from 19.06 to 20.05 mIoU and GaussianWorld from 21.36 to 21.91 mIoU; on GaussianWorld, object mIoU improves from 18.18 to 19.16 and rare-class mIoU from 15.60 to 16.79. These results suggest that VLMs are better suited to closed-set occupancy as reliability-aware semantic auditors than as generic caption-embedding targets.

- **现有 VLM 策略的缺陷**：常见的对齐方法（将 3D 体素/对象特征与裁剪图像的描述嵌入对齐）能够提升文本空间相似度，但并未可靠地提升**闭集占用 mIoU**。这揭示了**文本对齐与密集体素语义决策之间的弱耦合**。

## 2. 方法论
- **核心思想**：VISA 在训练时利用离线 VLM 对**每个物理对象的代表性裁剪图像**进行结构化审计，并将审计结果蒸馏到 3D 体素 logits，**推理时完全不使用 VLM**。
- **审计元组**：
  - `A_i = (cvlm_i, q_i, C_i, u_i, r_i, e_i)`
  - `cvlm_i` 为闭集类别假设，`q_i` 为置信度，`C_i` 为可能混淆类别集合，`u_i` 为预设的视觉属性向量（如“大型车辆”“两轮车”），`r_i` 为视觉可靠性，`e_i` 为证据字符串。
- **接地与传播**：
  - 审计仅应用于**匹配的 3D 目标体素**（`V_{i,t}`：落在 3D 框内且标签与对象类别一致的体素）。
  - 跟踪同一实例（通过 token）将审计沿时间轴广播。
- **损失函数**：
  - **可靠性加权的分类层次蒸馏** (`L_tax`)：构建由 VLM 类别假设+混淆集构成的软标签，与 one‑hot 真值混合 ，并用 `r_i * max(q_i,0.25)` 加权。
  - **属性因子损失** (`L_attr`)：将预测的对象类别概率映射到预定义的属性矩阵，与 VLM 给出的 `u_i` 进行加权 BCE 对齐。
  - **场景级审计图损失** (`L_graph`)：基于属性 Jaccard 相似度和审计兼容性确定正/负边，惩罚属性预测不一致。
  - 总监督：`L_VISA = λ_tax L_tax + λ_attr L_attr + λ_graph L_graph`，附加在原有占用损失之上。

## 7. 优点与亮点
- **训练–推理分离**：VLM 仅用于离线预处理，推理零额外开销，具有很强的部署友好性。
- **审计结构丰富**：利用 VLM 输出的**类别假设、混淆集、视觉属性和可靠性**，比单一伪标签或嵌入对齐更贴合闭集任务。
- **接地与过滤机制**：仅对匹配体素施加监督，并结合可靠性阈值和加权，极大降低噪声影响。
- **全面实验验证**：覆盖多个骨干、消融、非 VLM 控制基线、鲁棒性分析、未来预测与规划诊断，论证严谨。
- **可解释性**：审计元组可直接检查，诊断表格清晰展示了 VLM 的强项与弱项。

## 8. 不足与局限
- **离线成本与 VLM 偏差**：审计生成需耗费约 111 GPU 小时，并可能继承 VLM 自身的识别偏差（例如对遮挡或小目标的判别误差）。
- **监督适应性受限**：VISA 当前仅审计**可见物体实例**，对背景/静态（stuff）区域（道路、人行道等）的改善有限，增益集中于对象语义。
- **依赖标注数据**：训练中需要**准确的 3D 框、实例跟踪和密集体素标签**进行接地，限制了其在无高质量时间标注数据集上的直接应用。
- **规划安全解耦**：虽能改善开环轨迹仿真的 L2 误差，但**未能降低碰撞率**，表明未直接优化安全关键指标。
- **失败边界**：当裁剪存在前景干扰、上下文偏差或极端低分辨率时，VLM 审计可能产生错误引导，尽管通过可靠性加权缓解，仍可能导致区域性的误分（如错误连贯的假阳性区域）。
