## 摘要
几何形状对视角具有不变性，这使得任意图像集合成为同一三维状态的冗余编码。现有的前馈重建模型未能利用这一点：逐视图方法输出重叠且未对齐的点图，其数量随输入视图数量线性增长；而全局隐变量方法则固定于低分辨率输出。我们提出Surflo，它将可变数量的未定位RGB视图压缩为K个隐变量标记（即一个全局状态），并通过流匹配将点独立地从噪声传输到表面，从而解码出有向三维表面点。这使输出摆脱了任何固定网格或标记预算的限制：同一隐变量在一次前向传递中可产生几千到上百万个点。为抑制独立逐点解码固有的局部不一致性，我们在常微分方程积分过程中注入光度梯度，通过推理时引导项关联邻近点。Surflo在表面度量上达到或超越了前馈基线方法，运行速度比需要数百个视图的基于优化的方法快一个数量级，并且是唯一将全局隐变量与任意分辨率解码相结合的前馈方法。

## Abstract
Geometry is invariant to viewpoint, which makes any collection of images a redundant encoding of a single 3D state. Existing feed-forward reconstruction models fail to exploit this: per-view methods emit overlapping, unaligned pointmaps that grow linearly with input count, while global-latent methods commit to a fixed, low-resolution output. We introduce Surflo, which compresses a variable number of unposed RGB views into K latent tokens-one global state-and decodes oriented 3D surface points by independently transporting them from noise onto the surface via flow matching. This frees the output from any fixed grid or token budget: the same latent yields from a few thousand to a million points in a single forward pass. To suppress the local inconsistencies inherent to independent per-point decoding, an inference-time guidance term correlates nearby points by injecting a photometric gradient during ODE integration. Surflo matches or surpasses feed-forward baselines on surface metrics, runs an order of magnitude faster than optimization-based methods that require hundreds of views, and is the only feed-forward approach to combine a global latent with arbitrary-resolution decoding.

*   **核心问题**：现有的前馈3D重建模型未能有效利用“几何对视角不变”这一基本原理。它们主要分为两类，各有缺陷：
    *   **逐视图模型**：为每张输入图像生成独立的点云图，导致输出随视图数量线性增长、数据高度冗余且难以融合成一致的全局网格。
    *   **全局隐变量模型**：虽然能压缩场景信息，但输出分辨率（如点数）是固定的，缺乏灵活性。
*   **整体含义**：论文提出Surflo模型，旨在解决上述矛盾。其核心设计是**一个全局状态**，该状态能将任意数量的输入视图压缩为固定大小的表示，并支持从该表示中**解码任意数量、全局一致的有向表面点**。这实现了前馈重建模型在效率和灵活性上的统一。

### 2. 论文提出的方法论

Surflo方法由三个关键部分组成，协同工作实现其设计目标：

*   **核心思想**：将多视图3D重建视为一个“将场景压缩为全局隐变量，然后通过流匹配从该隐变量恢复任意分辨率几何”的过程。

*   **关键技术细节与算法流程**:
    1.  **编码器：从图像到固定大小的全局状态**
        *   **主干网络**：采用一个冻结的预训练VGGT模型作为基础特征提取器，获取具有强几何感知能力的多层级图像块标记和相机标记。
        *   **3D位置编码**：为增强空间感知，将VGGT预测的点云坐标通过傅里叶特征进行编码，并加到图像块标记上。解码器使用相同的编码方式，以建立查询点与场景特征的空间关联。
        *   **压缩**：使用一个类感知器的交叉注意力模块，将来自所有视图的图像块标记和相机标记，分别压缩成固定尺寸的隐变量\(z_p\)和\(z_c\)，最终拼接成全局状态\(z\)。其尺寸恒为\(K \times D\)，与输入视图数\(N\)无关。

    2.  **解码器：基于流匹配的独立查询点生成**
        *   **逐点流**：将表面解码视为一个传输问题。在给定全局状态\(z\)的条件下，使用流匹配框架学习一个速度场\(v_{\theta}\)，将每个查询点\(x_t \in \mathbb{R}^3 \times \mathbb{S}^2\)（三维坐标和法向）从噪声分布\(p_0\)传输到目标表面分布\(p_1\)。
        *   **架构**：解码器是一个多层Transformer，它通过交叉注意力与隐变量\(z\)交互，并独立地为每个查询点预测速度。其计算受到时间嵌入和相机隐变量\(z_c\)的调节。
        *   **训练目标**：最小化标准流匹配损失函数 \(\mathcal{L}_{FM}\)。
        *   **源分布**：为提高效率，初始噪声点并非从纯高斯分布采样，而是从以VGGT预测的点云为中心的高斯混合分布中采样，使得初始点更接近真实表面。
        *   **推理**：从源分布采样\(P\)个查询点，通过常微分方程求解器从\(t=0\)到\(t=1\)积分速度场\(v_{\theta}\)。由于所有点共享隐变量\(z\)，可以并行解码任意数量（数千到数百万）的点。

    3.  **引导：通过通信实现一致性**
        *   **动机**：独立的逐点解码缺乏对全局表面一致性的约束，可能导致点散布在不同表面层或产生漂移。
        *   **机制**：在ODE积分的后期（\(t \ge 0.95\)），引入一个推理时引导项。具体做法是：
            1.  基于当前估计的目标点\(\hat{x}_1\)，将其视为一个微型有向高斯飞溅并进行可微渲染，得到预测图像。
            2.  计算预测图像与输入图像之间的光度损失\(\mathcal{L}_{\text{render}}\)。
            3.  执行多步梯度下降来调整目标点，即\(\hat{x}_g = \hat{x}_1 - \nabla \mathcal{L}_{\text{render}}\)。
            4.  用调整后的目标点重新计算“引导速度”\(v_g\)，并用此速度进行实际的ODE步进。
        *   **效果**：这个梯度将输入图像的光度信息注入到生成过程中，使得独立采样的点能够通过共享的全局渲染信号进行“通信”和协调，从而抑制噪声，保持与输入视图的几何一致性。可选地，还可加入单目深度引导。

### 6. 论文的主要结论与发现

*   Surflo成功构建了一个**全局、固定尺寸的场景表示**，它不受输入视图数量影响，且能解码出**任意分辨率的全局一致表面**。
*   在多个少视图3D重建基准测试中，Surflo在**表面几何度量（CD, F1）上达到或超越了**现有前馈基线，并且**比优化方法快一个数量级以上**。
*   **推理时引导**是解决独立逐点解码不一致性问题的关键，它能有效利用输入图像信息提升细节和全局一致性。
*   作为一项辅助贡献，首次构建并公开了**大规模场景级水密网格数据集（网格化DL3DV）**，这将有益于未来该领域的研究。

### 7. 优点

*   **架构新颖性**：首次将“全局隐变量压缩”与“任意分辨率流匹配解码”成功结合用于场景级重建，在灵活性和一致性之间取得了良好平衡。
*   **高效率与可扩展性**：解码成本与输出分辨率解耦，一次编码可多次查询，适合需要不同细节层次的应用。
*   **推理时引导巧妙**：通过可微渲染注入光度梯度，在不重新训练模型的情况下，有效解决了独立解码的固有缺陷，提升了重建质量。
*   **重要数据贡献**：发布网格化DL3DV数据集，填补了该领域大规模高质量表面数据集的空白。
*   **实验扎实**：评估全面，消融充分，与多种主流方法进行了公平且令人信服的比较。

### 8. 不足与局限

*   **依赖主干网络**：方法继承了其主干VGGT的失败模式。当输入视图极度过少或基线过大导致VGGT预测不准时，Surflo的点云源分布和标记特征将不可靠，进而影响解码效果。
*   **非端到端一致性**：当前模型仅表示几何，不包含外观（颜色）信息。将解码器扩展至可以预测视角相关的辐射属性是下一步的自然方向。
*   **引导开销**：虽然总体效率高，但光度引导在推理中引入了额外计算成本，在极致速度优先的场景下可能构成瓶颈。
*   **训练监督来源**：其真值来源于Gaussian Wrapping算法，该算法本身对透明或弱纹理结构处理不佳，这些瑕疵可能会传递到Surflo的学习中。
*   **域外泛化潜力**：尽管在多个数据集上表现优异，但对于与训练数据（主要来自DL3DV）特性差异极大的场景（例如，极端空旷的户外或高度反光的室内），其泛化能力未在文中充分探索。
