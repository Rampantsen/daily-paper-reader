## 摘要
单参考未见物体6D姿态估计通过仅用一个参考视图来估计任意新物体的姿态，减少了物体上线负担。近期基于对应关系的流水线利用视觉基础模型（VFM）特征取得了稳健的性能。然而，这些方法通常将这些特征视为视图内描述符，导致密集的视觉语义线索（包括外观、结构和上下文）在几何解码之前无法跨视图充分交换。因此，解码后的点特征可能缺乏联合的语义和几何判别力，使得在挑战性情形下的对应估计仍然困难。我们没有独立处理特征，而是围绕早期的跨视图语义先验来构建对应流水线。具体而言，跨视图语义交互（CVSI）使密集的查询和参考VFM token能够交换语义上下文并形成跨视图先验。然而，直接的CVSI可能会扰乱VFM token结构，而生成的语义先验仍然需要3D表示一致性以用于刚性对应。为了使该CVSI先验对3D对应学习可靠，我们引入了两个互补的训练时约束：视图内结构保持（IVSP）损失在交互过程中保持原始的视图内token亲和力结构，而参考锚定的几何一致性（RAGC）损失则强制解码点特征的空间表示一致性。最终姿态通过加权SVD从学习到的对应关系中恢复。我们进一步从BOP挑战数据集YCB-V和TUD-L构建了一个挑战性视图对协议，以评估在困难匹配场景下的鲁棒性。在六个基准上不同视图对设置下的大量实验表明，我们的方法在保持可比推理速度的同时达到了最先进的性能。

## Abstract
Single-reference unseen object 6D pose estimation reduces object onboarding by estimating poses of arbitrary novel objects from only one reference view. Recent correspondence-based pipelines have achieved robust performance with vision foundation model (VFM) features. However, they typically treat these features as intra-view descriptors, leaving dense visual-semantic cues, including appearance, structure, and context, insufficiently exchanged across views before geometric decoding. Consequently, the decoded point features may lack joint semantic and geometric discriminability, making correspondence estimation still difficult in challenging cases. Instead of processing features independently, we build the correspondence pipeline around an early cross-view semantic prior. Specifically, cross-view semantic interaction (CVSI) enables dense query and reference VFM tokens to exchange semantic context and form a cross-view prior. Nevertheless, direct CVSI may disturb the VFM token structure, while the resulting semantic prior still needs 3D representation consistency for rigid correspondence. To make this CVSI prior reliable for 3D correspondence learning, we introduce two complementary training-time constraints: the intra-view structure preservation (IVSP) loss preserves the original intra-view token affinity structure during interaction, while the reference-anchored geometric consistency (RAGC) loss enforces spatial representation consistency of decoded point features. The final pose is recovered from learned correspondences through weighted SVD. We further construct a challenging view-pair protocol from the BOP Challenge datasets YCB-V and TUD-L to evaluate robustness in difficult matching scenarios. Extensive experiments on six benchmarks under different view-pair settings show that our method achieves state-of-the-art performance while maintaining comparable inference speed.

### 1. 论文的核心问题与整体含义

- **研究背景**：单参考未见物体 6D 姿态估计旨在仅利用一张目标物体的 RGB‑D 参考视图，估计该物体在任意新观测中的 6 自由度姿态，从而大幅降低对新物体的上线成本。
- **现存问题**：当前主流的基于对应关系的流水线使用视觉基础模型（VFM，如 DINOv2/DINOv3）提取密集特征，并将其独立地作为各视图的内部描述符使用。几何解码前，跨视图的密集视觉‑语义线索（外观、局部结构、上下文关系）未能得到充分交换，导致在以下挑战场景中对应估计的语义‑几何判别力不足：
  - 分割掩膜存在噪声或包含干扰物；
  - 视角变化极大，可见重叠区域稀疏且模糊。
- **本文动机**：在几何解码前构建**早期跨视图语义先验**，使点云匹配能同时受到参考视图条件的视觉语义引导，从而提升对应可靠性。

### 2. 论文提出的方法论

#### 2.1 整体框架
- 采用**由粗到精的两阶段流水线**，每阶段均基于采样后的前景点云（粗阶段 196 点，精阶段 2048 点），通过对应矩阵估计相对姿态并进行加权 SVD 求解。
- 核心创新在于在特征投影到点云并输入几何解码器**之前**，对密集 VFM 图像 token 进行**跨视图语义交互**（CVSI），形成跨视图先验。

#### 2.2 跨视图语义交互（CVSI）
- 输入：查询图像 token \(X_q\) 和参考图像 token \(X_r\)。
- 经过 \(L\) 个交互块，每块执行双向共享权重的多头交叉注意力。以查询分支为例：
  - 线性投影至低维空间并归一化得到 \(\bar{X}_{q,l}, \bar{X}_{r,l}\);
  - 计算查询、键、值：\(Q_{q,l} = \bar{X}_{q,l} W^Q\), \(K_{r,l} = \bar{X}_{r,l} W^K\), \(V_{r,l} = \bar{X}_{r,l} W^V\);
  - 注意力响应：\(\Delta X_{q,l} = \text{softmax}(Q_{q,l} K_{r,l}^\top / \sqrt{d}) V_{r,l}\);
  - 残差连接与 FFN 更新：\(X_{q,l+1} = X_{q,l} + \Delta X_{q,l} + \text{FFN}(\text{LN}(X_{q,l} + \Delta X_{q,l}))\); 参考分支对称更新。
- 输出跨视图增强 token \(\tilde{X}_q, \tilde{X}_r\)，随后上采样并投影为逐点语义特征。

#### 2.3 训练时的两个辅助约束

- **视图内结构保持损失（IVSP）**
  - 目的：防止 CVSI 过度平滑 VFM token 的原始空间组织。
  - 方式：将交互前的 token 作为教师，交互后的作为学生，通过去中心化余弦相似矩阵对学生进行正则化：
    \[
    L_q^{\text{IVSP}} = \frac{1}{M^2} \sum_{i,j} -\bar{A}_q^D(i,j) \, A_q^S(i,j),
    \]
    其中 \(\bar{A}^D\) 为中心化后的教师相似度，\(A^S\) 为仅保留非负部分的学生相似度。最终损失为 \(L_{\text{IVSP}} = (L_q^{\text{IVSP}} + L_r^{\text{IVSP}})/2\)。

- **参考锚定几何一致性损失（RAGC）**
  - 目的：强制解码后的点特征在统一参考帧内具有空间一致性，以利于刚性对应学习。
  - 方式：基于参考点云计算归一化变换 \(T_{\text{rac}}^r\)，将参考点云和地面真实对齐后的查询点云映射至参考锚定坐标（RAC）空间。将融合后的点特征 \(F_q, F_r\) 经过 MLP 预测 RAC 坐标，并用 SmoothL1 损失监督：
    \[
    L_{\text{RAGC}} = \frac{1}{2} \left( \frac{1}{N} \sum_i \rho(\hat{u}_q^i, u_q^i) + \frac{1}{N} \sum_j \rho(\hat{u}_r^j, u_r^j) \right).
    \]
  - 该损失与对应匹配损失共同优化，但仅在训练时使用。

#### 2.4 总训练目标
\[
\mathcal{L} = \mathcal{L}_{\text{match}} + \sum_{s \in \{c, f\}} (\lambda_{\text{RAGC}} L_s^{\text{RAGC}} + \lambda_{\text{IVSP}} L_s^{\text{IVSP}}),
\]
其中 \(\mathcal{L}_{\text{match}}\) 包含粗、精阶段的 InfoNCE 对应损失和加权二元交叉熵重叠损失。

### 6. 论文的主要结论与发现

- 早期跨视图语义交互（CVSI）显著提升了单参考未见物体姿态估计的性能，在所有六个基准和自行构建的挑战性视图对协议下均达到最优结果。
