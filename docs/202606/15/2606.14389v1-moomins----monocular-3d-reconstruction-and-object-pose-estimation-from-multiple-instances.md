## 摘要
从单张单目图像同时进行三维重建和六自由度物体位姿估计本质上是一个不适定问题。然而，在工业场景中，同一物体的多个实例常被随机放置在料箱中，从而在单张图像中隐式地提供了同一物体的多个视图。我们展示了可以利用这种隐式的多视图几何，同时重建物体的三维模型并估计每个可见物体实例的六自由度位姿。我们提出MooMIns，一种基于高斯泼溅的新方法，它反转了原始的高斯泼溅范式：不是从多个相机渲染单个场景，而是从单个相机渲染多个物体实例。我们的方法使用SAM3实例分割掩码和经过修改的运动恢复结构（SfM）流程进行初始化。与基于学习的单目深度估计不同，我们根据图像证据执行真正的基于几何的重建，避免了训练数据先验导致的幻觉。我们在合成和真实的料箱抓取场景上评估了MooMIns，并展示了对未见过的物体进行精确重建以及对单个实例进行可靠位姿估计的能力。

## Abstract
Simultaneous 3D reconstruction and 6D object pose estimation from a single monocular image is an inherently ill-posed problem. In industrial settings, however, multiple instances of an object are often randomly arranged in bins, implicitly providing several views of the same object within a single image. We show that this implicit multi-view geometry can be exploited to simultaneously reconstruct the object in 3D and estimate the 6D pose of each visible object instance. We present MooMIns, a new Gaussian-splatting-based approach that inverts the original Gaussian splatting formulation: instead of rendering a single scene from multiple cameras, we render multiple object instances from a single camera. Our method is initialized with SAM3 instance segmentation masks and a modified Structure from Motion (SfM) pipeline. In contrast to learned monocular depth estimation, we perform true geometry-based reconstruction from image evidence, avoiding hallucinations caused by training data priors. ## 方法论（MooMIns）
MooMIns 的核心思想是**反转高斯泼溅渲染范式**：经典3DGS从多个相机渲染单个场景，MooMIns则从单个相机渲染规范物体的多个实例。整个流程分为初始化和改进的高斯泼溅优化两部分。

### 1. 初始化阶段
- **实例分割**：使用SAM3对输入图像生成可见实例掩码\(M_{\text{ref}}^{v,j}\)，借助多模态大语言模型提供物体描述作为提示。
- **结构重建**：将每个掩码区域裁剪为独立图像，输入修改后的COLMAP SfM流程，获得物体稀疏点云和各实例对应的虚拟相机位姿\(O_{\text{CC},j}\)。相机内参假设已知。
- 该阶段输出初始规范物体稀疏点云和初始实例位姿，为后续优化提供初值。

### 2. 改进的高斯泼溅优化管道
- **规范高斯集**：用2D高斯泼溅表示规范的物体几何与辐射属性\(G_{\text{canon}}\)（包含位置、旋转、缩放、透明度、漫反射率、粗糙度）。采用基于球面沃罗诺伊（Spherical Voronoi）的反射模型分离几何、材质与光照，光照参数\(U\)包括远场立方体贴图和近场光探针。
- **场景合成与渲染**：在每次训练迭代中，将\(G_{\text{canon}}\)通过当前估计的实例位姿\(^C P_O\)变换，组成多实例场景\(G_{\text{multi}}\)（公式(2)）。使用固定的真实相机外参渲染，同时优化照明。
- **关键技术创新**：
  - **实例丢弃（Instance Dropout）**：随机丢弃60%的实例，仅渲染保留的子集，并利用掩码分离前景与背景，防止模型将多个实例的外观过拟合到同一个规范物体上。
  - **修改的自适应密度控制（ADC）**：考虑到每个高斯在单个图像中可能因多实例而被多次可见，调整梯度阈值计算方式（公式(3)）。
  - **鲁棒实例移除**：基于每个实例的中值损失，用鲁棒标准差剔除渲染质量明显较差的异常实例（最多移除20%）。
  - **损失函数**：总损失由蒙版光度损失（L1+SSIM）、背景二元交叉熵损失（促进透明度分离）、2D高斯正则项（法线、距离）组成，并采用高斯随机丢弃（Dropout）防止稀疏视图过拟合。
  - **位姿参数化**：使用6D连续表示优化旋转，所有可学习参数（\(G_{\text{canon}}\)、位姿\(^C P_O\)、光照\(U\)）通过Adam优化器更新。

## 实验设计
### 数据集与基准
- **SYN1**：合成数据集，12面骰子，61个实例，无明显遮挡，纹理丰富。用于验证方法在有纹理、无遮挡条件下的表现。
- **REAL1**：真实拍摄的糖果包装盒（22个无遮挡实例），用于验证完整管道（含自动初始化）在真实图像上的可行性，仅定性评估。
- **XYZ1-5**：选自XYZ-IBD工业料箱数据集，包含强反射、弱纹理金属件，存在大量遮挡。为评估管道，使用了伪初始值（位姿加噪5°/30mm，几何初始化为球体）。

### 评估指标
- 位姿：平均平移误差（MTE，单位mm）、平均旋转误差（MRE，单位°），经过相似变换对齐后使用evo工具计算。
- 重建：提取网格后计算与真值之间的Chamfer距离\(d_c\)（单位mm）。

### 对比方法
- 主要与初始化结果进行对比，展示优化后的提升幅度。无其他学习方法直接对比，因为这是新定义的任务。
- 消融实验：移除背景损失\(L_m\)、远/近场光照、实例丢弃、随机背景色、鲁棒实例移除、高斯丢弃共7种变体，在SYN1和XYZ5上评估。

## 主要结论与发现
- MooMIns 能够从单张多实例图像中**真实、几何驱动地**重建物体三维形状和估计实例位姿，避免学习式方法的幻觉。
- 在纹理清晰且无遮挡的情况下（如SYN1），即使使用完全自动的SfM初始化，也能取得高精度结果。
- 对于工业常见的无纹理、强反射和遮挡物体，只要提供**合理初值**，方法依然可以收敛并显著提升位姿和重建精度。
- 消融实验证明，背景损失、实例丢弃、光照建模等模块对于处理遮挡和复杂外观至关重要。

## 优点
- **范式创新**：首次定义并解决了“从多实例单目图像联合重建与估计”这一新任务，开辟了无需先验模型的工业视觉新路线。
- **几何纯推理**：完全基于图像证据进行重建，无训练数据依赖，零样本泛化性极强。
- **巧妙的GS反转设计**：通过“单相机多场景”保持真实光照，自动引入多视图约束，比“多虚拟相机”方案更符合物理原理且收敛更稳定。
- **工程技巧完善**：实例丢弃、鲁棒移除、自适应密度控制等设计有效处理了多实例重叠、过拟合、异常值等固有问题。
- **实验验证扎实**：从合成到真实、从简单到复杂，并辅以详尽的消融研究，证明各组件的必要性。

## 不足与局限
- **初始化依赖强**：需要已知相机内参；SfM初始化在无纹理物体上失败，仍需借助额外先验；无法自动获取无模态（amodal）掩码。
- **容噪范围有限**：鲁棒性实验显示，当旋转噪声超过15°、平移噪声超过150 mm时，位姿估计质量会下降，说明在极差初值下可能发散。
- **尚未完全自动化**：当前无模态掩码需外部提供，可能导致实际应用中遮挡处理受限；未来需探索联立估计无模态掩码。
- **场景适用范围**：该方法假设所有实例共享完全相同的几何与外观，制造公差、纹理变化或非刚性形变可能引起异常实例被剔除，且要求场景中除物体外背景较易分离。
- **计算资源未量化**：未说明每个场景的训练耗时，难以评估实时性。
