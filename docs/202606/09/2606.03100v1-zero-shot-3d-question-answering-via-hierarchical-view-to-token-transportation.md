## 摘要
近期，通过二维视觉-语言模型（VLMs）实现零样本三维场景理解因其在空间推理方面展现出的潜力而备受关注。通常，从三维点云中采样多个二维视图并输入预训练的VLMs以回答给定问题。这一范式凸显了输入上下文质量的关键作用，并带来了在有限输入预算下尽可能保留任务相关三维细节的挑战。我们提出\texttt{KeyVT}，一种在视图和令牌两个层面进行输入上下文收集的层次化方法。具体而言，我们将像素特征与相机参数相结合，并基于语义内容和几何位置评估视图重要性，从而获得空间一致且任务相关的视图。此外，我们在最优传输（OT）框架下通过识别代表性令牌来解决所选视图间图像块冗余的问题，将视图令牌和关键令牌视为嵌入空间中的两个离散分布。通过最小化OT距离，这些关键令牌有望覆盖所有视图特征。我们在三个广泛使用的基准上评估了我们的框架，结果表明其相较于现有免调优方法有显著提升，性能可与基于训练的方法相媲美。

## Abstract
Recently, zero-shot 3D scene understanding via 2D Vision-Language Models (VLMs) has gained increasing research interest due to their promising spatial reasoning capabilities. Typically, multiple 2D views are sampled from a 3D point cloud and fed into pre-trained VLMs to answer a given question. This paradigm highlights the critical role of input context quality and raises the challenge of retaining as many task-relevant 3D details as possible under a limited input budget. We propose \texttt{KeyVT}, a hierarchical approach for input context collection at both the view and token levels. Specifically, we combine pixel features with camera parameters and assess view importance based on both semantic content and geometric position, resulting in spatially consistent and task-relevant views. Furthermore, we address redundancy among patches across selected views by identifying representative tokens under the optimal transport (OT) framework, where view tokens and key tokens are formulated as two discrete distributions in the embedding space. These key tokens are expected to cover all view features by minimizing the OT distance. We evaluate our framework on three widely used benchmarks, demonstrating significant improvements over existing tuning-free methods and performance comparable to training-based approaches.

### 2. 论文提出的方法论：核心思想、关键技术细节

- **核心思想**：提出一个名为 **KeyVT** 的免训练、即插即用的层次化框架，通过“先选关键视图（KeyV），再选关键令牌（KeyT）”的两阶段策略，在固定的 VLM 输入预算内填充尽可能多的任务相关三维信息。

- **关键技术细节（KeyV：几何感知的关键视图选择）**：
  - **目标**：选择既与问题语义相关，又在空间上形成连贯子场景的视图，以提供关键物体及其周围环境信息。
  - **流程**：
    1.  **计算视图相对距离**：结合相机的位置（相机中心 C）和朝向（R）信息，通过公式 `D(V₀, Vᵢ) = ||C₀ - Cᵢ|| + θ(R₀, Rᵢ)` 计算当前视图 Vᵢ 与第一个视图 V₀ 之间的空间距离，其中 θ 是角度距离。
    2.  **划分子场景**：基于计算出的视图距离 D，使用一个预定义的窗口大小 δ 进行滑动窗口分割，将整个三维场景划分为一系列空间上连续的子场景序列 {L₁, L₂, …, L|L|}。
    3.  **评估子场景相关性**：利用预训练的多模态编码器（如 BLIP2），计算子场景 Lₗ 内所有视图与问题 Q 的语义相似度分数，然后结合其中的最大值和平均值作为该子场景的整体相关性得分 `rₗ`。
    4.  **自适应分配关键视图**：根据子场景的重要性（`rₗ`）及其大小（`|Lₗ|`）进行加权，按比例将预定的关键视图总数 K 动态分配给各子场景。在每个子场景内，选择语义相似度得分最高的 `Nₗ` 张视图作为最终的关键视图集 M*。

- **关键技术细节（KeyT：基于最优传输的关键令牌选择）**：
  - **目标**：从选出的关键视图令牌集合 P 中，学习一个数量更少（M < N）的“虚拟令牌”集合 Q，使其在嵌入空间中能最大程度地保留 P 的语义信息，以此压缩冗余。
  - **核心思想**：将此问题建模为一个最优传输（OT）问题，其中 P 和 Q 被视为嵌入空间中的两个离散分布。
  - **算法流程**：
    1.  **定义分布**：令 P = Σ αₙeₙ， Q = Σ βₘcₘ，其中 eₙ 是真实视图令牌嵌入，cₘ 是待学习的虚拟令牌嵌入，α 和 β 是两个概率单纯形向量（如均设为均匀分布）。
    2.  **构建成本与优化**：定义成本矩阵 C，其中 `Cₙ,ₘ = 1 - cos(eₙ, cₘ)`，即令牌间的余弦距离。优化的目标是找到最优传输计划 T，使得将分布 P “搬运”成分布 Q 的 Sinkhorn 距离最小化：`d_C,γ(P, Q) = min_{T∈U(α, β)} <T, C>`。通过梯度下降法（如 Adam 优化器，迭代 10-15 步）高效更新虚拟令牌 cₘ。
    3.  **生成最终输入**：VLM 无法直接处理虚拟令牌。因此，对于每个学习到的虚拟令牌 cₘ，根据其对应的传输计划列 `T·, m` 选取概率最高的 `S_M` 个真实视图令牌作为其“邻居”。最终将所有选中的真实令牌拼接起来，构成 VLM 的最终输入上下文 I。

- **创新的层次化设计**：首次在 3D 问答任务中，将空间感知的关键视图选择和基于最优传输的令牌压缩相结合，从两个层面优化输入上下文。
- **几何与语义融合**：KeyV 方法巧妙地将相机参数提供的几何信息与 BLIP2 提供的语义相关性评分融合，解决了单纯语义检索忽略空间连续性的问题。
- **原理优雅的压缩方法**：KeyT 利用最优传输理论，将令牌压缩转化为寻找能够覆盖原始令牌分布的代表性子集问题，方法有坚实的理论基础，且学习过程仅需极少量迭代。
- **强大的即插即用能力与免训练特性**：KeyVT 无需任何训练或微调，仅需对虚拟令牌进行几步无监督学习即可适配任何 VLM，极大地降低了应用门槛和成本。

### 8. 不足与局限：包括实验覆盖、偏差风险、应用限制等

- **对初始视图的依赖**：整个流程的质量高度依赖于第一阶段关键视图（KeyV）的选择。如果一开始就遗漏了包含关键信息的视图，后续的压缩（KeyT）也无法弥补。
- **相机参数限制**：KeyV 的有效性依赖于准确的相机参数。对于没有提供相机参数的视频（如 VSI-Bench），需要额外使用工具（如 FastVGGT）来估计，这会引入额外的处理步骤和潜在的估计误差。
- **虚拟令牌的微调需求**：虽然主要流程免训练，但 KeyT 模块中虚拟令牌的优化仍需要数次梯度迭代。对于极端实时性要求的应用，这可能仍比完全的无迭代方法慢。
- **token 可解释性较弱**：最后输入的拼接令牌是经过压缩和选择后的抽象表示，其具体的空间含义比原始视图更难以解释，可能在需要精确 3D 定位的下游任务中存在局限。
- **任务覆盖面有限**：当前评估仅限于 3D 问答任务，该方法在 3D 视觉定位、导航等其他 3D 理解任务上的通用性还有待验证。
