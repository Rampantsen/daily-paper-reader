## 摘要
高保真3D高斯头部化身生成对于AR/VR、远程呈现和数字人等应用至关重要。现有方法依赖多视图数据集、3D捕捉或中间2D视图合成。相比之下，我们仅从随机采样的2D图像中学习条件及无条件3D头部模型，无需使用多视图数据、3D监督或中间视图生成。我们提出MVCHead，一种单次状态空间模型，在回归3D高斯的同时直接在3D表示中施加多视图一致性约束。其核心是分层状态空间（HiSS）模块，该模块逐步从粗到细细化高斯，并捕获长程依赖关系。在每个HiSS块中，我们通过提出的分层双向状态扫描（HiBiSS）修改了Mamba的标准单向扫描，使递归与多视图不一致性最强的轴对齐。最后，我们设计了一个SE(3)多视图评论家，判断一组自渲染是否源于单一底层3D配置，在未观测真实多视图对的情况下奖励跨视图像素对齐。MVCHead实现了最先进的感知质量，在纹理和几何一致性上超越了先前方法，并保持了相当的形状一致性。为证明可扩展性，我们发布了FaceGS-10K，这是首个大规模即用型3D高斯头部资产数据集，用于训练和评估3D头部模型。项目页面与代码：https://humansensinglab.github.io/MVCHead/

## Abstract
High-fidelity 3D Gaussian head avatar generation is critical for applications such as AR/VR, telepresence, and digital humans. Existing methods depend on multi-view datasets, 3D captures, or intermediate 2D view synthesis. In contrast, we learn both conditional and unconditional 3D head models from randomly sampled 2D images alone, without using multi-view data, 3D supervision, or intermediate view generation. We introduce MVCHead, a single-shot state space model that enforces multi-view consistency (MVC) directly in the 3D representation while regressing 3D Gaussians under these constraints. At its core, we propose a Hierarchical State Space (HiSS) block that progressively refines Gaussians from coarse to fine, while capturing long-range dependencies. Within each HiSS block, we modify Mamba's standard unidirectional scan with the proposed Hierarchical Bi-directional State Scan (HiBiSS) that aligns recurrence with the axes along which multi-view inconsistencies are strongest. Finally, we design an SE(3) Multi-view Critic that judges whether a set of self-renders arises from a single underlying 3D configuration, rewarding cross-view pixel alignment without observing real multi-view pairs. MVCHead achieves state-of-the-art perceptual quality, surpasses prior methods in both texture and geometric consistency, and maintains comparable shape consistency. To demonstrate scalability, we release FaceGS-10K, the first large-scale dataset of ready-to-use 3D Gaussian head assets for training and evaluation of 3D head models. - 核心挑战：如何在这种设定下**强制模型学习跨视图一致性**，避免几何漂移和身份不一致。

### 2. 论文提出的方法论：核心思想、关键技术细节
MVCHead 是一个端到端的单次状态空间模型，从潜码直接回归完整3D高斯属性集。通过**结构化设计**和**自监督一致性奖励**取代了对多视图监督的依赖，主要包括三个创新组件：

#### 2.1 分层状态空间（HiSS）块
- 将3D头部表示为高斯点的层次化组织，从粗到细逐级细化。较粗层的高斯作为**锚点（anchor）**，后续更细的高斯被参数化为相对锚点的偏移量，从而防止细节漂移。
- 每个 HiSS 块内采用 **Dual‑Mixer** 架构：
  - **自注意力**负责全局语义和长程依赖（如整体身份）；
  - **状态空间混合器**沿图像轴向进行局部连贯性建模，强制沿行和列的结构一致性。
- 通过 AdaIN 实现外观条件调制，且**不引入相机条件**，避免模型坍缩为视角特定的2D映射。

#### 2.2 分层双向状态扫描（HiBiSS）
- 基于一个关键洞察：多视图不一致性主要集中在对准直立头部的**水平方向（偏航）和垂直方向（俯仰）**，即左右移动主要影响水平位移，上下移动主要影响垂直位移。
- 将标准的 Mamba 单向扫描扩展为**沿行/列的四个方向双向扫描**（→, ←, ↓, ↑），形成层次化双向递归路径，有效对齐了主要的不一致漂移轴。
- 扫描作用于上采样前的特征网格，使外观和几何信息能够沿轴传播，提升跨视图一致性。

#### 2.3 SE(3) 多视图评论家
- 基于另一个关键洞察：从同一3D模型渲染的多个视图（自渲染）天然几何一致，而来自不同潜在身份的渲染不一致。
- 设计一个 ViT 风格编码器 \(E_\psi\)，通过**几何变换注意力（GTA）**注入相对相机外参，学习判断一组图像/位姿对是否来自单一3D配置，输出一致性分数 \(s\)。
- 训练时：正样本为同一化身的不同视角渲染，负样本为不同化身但相同视角渲染；采用二分类交叉熵训练 \(E_\psi\)。然后将 \(E_\psi\) 作为可微奖励项，生成器被优化以最大化正样本得分，从而**显式提升多视图一致性**。
- 最终损失：\(\mathcal{L}_{total} = \lambda_{mvc} \mathcal{L}_{mvc} + \mathcal{L}_{adv} + \lambda_{knn}\mathcal{L}_{knn} + \lambda_{ctr}\mathcal{L}_{ctr}\)，其中 \(\mathcal{L}_{adv}\) 为相机条件对抗损失，\(\mathcal{L}_{knn},\mathcal{L}_{ctr}\) 为高斯点云的间距和中心偏移正则。

### 3. 实验设计：数据集、Benchmark 与方法对比
- **数据集**：FFHQ（512×512）和 FFHQ‑C，均仅使用2D图像。
- **评估指标**：
  - **感知真实感**：FID、FID₃D（随机视角下的FID）。
  - **多视图一致性**（首次在3D头部化身中全面量化）：
    - 形状一致性：Chamfer Distance (CD)、深度误差 (depth)。
    - 纹理一致性：cPSNR、cSSIM、cLPIPS（从不同视角子集重建后渲染对比）。
    - 几何一致性：MEt3R（利用 MASt3R + DINO + FeatUp 评估无对应的大视角变化一致性）。
- **对比方法**：StyleSDF、EpiGRAF、VoxGRAF、GMPI、StyleNeRF、EG3D、Mimic3D、GSGAN、GSM、GGHead、CGSGAN（全部在相同的数据约束下）。

### 4. 资源与算力
- **训练配置**：使用 4 块 NVIDIA H100 GPU，训练 10M 步，耗时约 3 天。
- 推理为单次前向过程，即时生成 240K 个高斯点，渲染速度接近实时。

### 5. 实验数量与充分性
- **主实验**：在两个数据集上分别与超过10种基线方法对比，报告FID、FID₃D及5项多视图一致性指标。
- **消融实验**：在 FFHQ‑C 上对损失项（去掉 \(\mathcal{L}_{adv}\)、\(\mathcal{L}_{mvc}\)）、HiSS 块中的状态空间部分、HiBiSS 单向扫描替代进行了对比，验证了各组件的贡献。
- **扩展实验**：利用训练好的模型生成 FaceGS‑10K 数据集（10K 个即用型3D高斯头资产），并展示单图反演的条件生成能力。
- 实验设置公开、公平，对比标准与先前工作一致，消融充分，评估维度全面且前人未系统衡量本任务的多视图一致性。

### 6. 论文的主要结论与发现
- MVCHead 在极小资源设定下达到**最先进的感知质量（最低 FID）**，并在纹理和几何一致性上全面超越现有最佳方法 CGSGAN，形状一致性持平。
- 分层双向状态扫描与 SE(3) 多视图评论家是实现多视图一致性的关键：二者分别通过**架构偏置**和**学习奖励**迫使模型产生跨视角自洽的3D表示。
- 无需中间视图合成或任何3D真值，即可生成具有复杂纹理和精细面部微结构的高保真头部化身。
- 面向社区开源 FaceGS‑10K 数据集，可推动3D头部模型的规模化训练与评估。

### 7. 优点：方法或实验设计亮点
- **新颖的设定与建模**：首次将状态空间模型引入3D高斯头部生成，实现了真正的极简监督（纯2D）一致性生成。
- **精巧的组件设计**：
  - HiBiSS 巧妙利用了头部旋转导致的轴对齐漂移特性，通过四向扫描内建多视图一致性。
  - SE(3) 评论家利用“自渲染天然一致”的事实，自监督地学习跨视图对齐信号，避免了真实多视图数据。
- **全面的评估体系**：不仅使用传统 FID，还引入 CD、cPSNR、MEt3R 等指标首次定量分析了生成头部化身的多视图一致性。
- **实用贡献**：开源大规模3D高斯数据集 FaceGS‑10K，并提供条件生成 inversion，拓展了下游应用。

### 8. 不足与局限
- **视角覆盖不足**：仅训练在前、侧视图区域，无法生成后脑和完整360°化身，限制了在需要全方位渲染场景中的应用。
- **几何先验弱**：所有结构信息完全从2D对抗训练中习得，缺乏任何显式对称性、拓扑等强几何约束，可能导致偶尔的几何异常。
- **评论家负面样本简单**：负样本来自不同身份，评论家可能尚未学会细粒度的几何/纹理一致性（如同一身份在不同视角下的微小错位），未来可引入更难的负样本进一步提升一致性。
- **未涉及表情/姿态控制**：当前为无条件生成，未探索动态驱动或可控表情的扩展。
- **训练资源需求不低**：虽然推理高效，但训练需要 4 块 H100 三天，对某些研究团队仍有一定门槛。
