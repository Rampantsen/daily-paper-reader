## 摘要
我们提出了一种以图像为条件、将图像作为主要几何源而非辅助引导的点云补全方法。为此，我们引入了一个图像到点云（I2P）模块，该模块可直接从单张RGB图像重建完整点云，无需三维输入。此外，我们还引入了一个基于Transformer的点到点（P2P）细化模块，利用点标记与图像特征之间的自注意力和交叉注意力，对粗略的I2P输出进行迭代细化。I2P模块使图像编码器能够学习丰富的几何表征，而P2P模块则逐步恢复细粒度细节。与依赖辅助损失或融合模块的现有多模态方法不同，我们明确的I2P任务仅基于图像就提供了强大的几何感知先验。在ShapeNet-ViPC数据集上的大量实验表明，我们的方法实现了最先进的补全性能，相比先前方法，倒角距离相对改善了12.3%。代码开源在：https://github.com/AzharSindhi/I2PRef.git

## Abstract
We present an image-conditioned point cloud completion approach that treats images as the primary geometric source rather than a secondary guide. To this end, we introduce an Image-to-Point (I2P) module that can reconstruct complete point clouds directly from a single RGB image, with no need for 3D inputs. Additionally, we introduce a transformer-based Point-to-Point (P2P) refinement module that uses self- and cross-attention between point tokens and image features to iteratively refine the coarse I2P output. The I2P module enables the image encoder to learn rich geometric representations, while the P2P module progressively recovers fine-grained details. Unlike existing multimodal methods that rely on auxiliary losses or fusion modules, our explicit I2P task provides a strong, geometry-aware prior based on images alone. Extensive experiments on ShapeNet-ViPC demonstrate state-of-the-art completion performance with a 12.3% relative Chamfer Distance improvement over prior methods. Code is available at: https://github.com/AzharSindhi/I2PRef.git

## 2. 方法论

### 2.1 核心思路
- 两条支路：（1）**Image-to-Point (I2P) 模块**从单张RGB图像直接生成粗糙完整点云；（2）**Point-to-Point (P2P) 细化模块**利用Transformer的注意机制，结合图像特征对粗糙点云进行迭代优化，最终输出精细点云。

### 2.2 Image-to-Point (I2P) 模块
- 采用类U-Net的层级编码器（起始7×7卷积 + 多次下采样，每个阶段含ResNet块、RMSNorm、SiLU激活、注意力层），最终输出特征图 \(F \in \mathbb{R}^{H' \times W' \times C}\)。
- 特征图展平为 \(N = H'W'\) 个特征向量，送入点生成器（类似XMFNet的设计），全局聚合后解码为多个点云分支，产生新点并与部分点云拼接，得到粗糙点云 \(P_0 \in \mathbb{R}^{N \times 3}\)。
- 参数仅约6M，轻量且有效。

### 2.3 Point-to-Point (P2P) 细化模块
- 将粗糙点坐标通过线性投影得到初始嵌入 \(X_0 \in \mathbb{R}^{N \times D}\)。
- 进行 \(L\) 次迭代（本文 \(L=4\)），每次迭代包含：
  - **自注意力**：点嵌入之间交换几何上下文。
  - **交叉注意力**：点嵌入为查询，图像特征为键和值，注入图像指导。
  - 前馈网络和残差连接。
- 每次迭代输出偏移量 \(\Delta P^{(l)}\)，累加更新点云：\(P^{(l+1)} = P^{(l)} + \Delta P^{(l)}\)，最终得到精确点云 \(P_c\)。

### 2.4 损失函数
- 主损失为**Chamfer Distance (CD)**，同时监督粗糙I2P输出和最终细化结果：
  \[
  \mathcal{L}_{\text{cd}} = \text{CD}(P_c, P_{\text{gt}}) + \alpha \, \text{CD}(P_0, P_{\text{gt}})
  \]
- 权重 α 从0.7线性退火至0.1，使训练重心从粗结构逐渐转向精细优化。
