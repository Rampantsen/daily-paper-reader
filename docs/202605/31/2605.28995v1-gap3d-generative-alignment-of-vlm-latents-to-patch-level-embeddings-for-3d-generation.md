## 摘要
近期将视觉-语言模型（VLMs）作为提示编码器以调节生成模型的方法通常依赖昂贵的端到端训练，或将特征映射为压缩表示，丢弃了几何感知任务（如3D资产生成）所需的密集空间结构。为解决此问题，我们提出GAP3D，这是一种模块化的、基于扩散的方法，它将VLM生成的潜在表示直接对齐到预训练图像编码器的完整块级特征空间，使得冻结的下游生成模型能够将VLM用作提示编码器，同时保持空间结构化的条件信号。在3D资产生成上的评估显示，我们的方法主要通过通用领域的图像-文本对进行训练，绕开了对大规模3D数据的需求。尽管仅在文本输入上训练，该方法也展现出了对多模态提示的新兴零样本能力。最后，虽然目前优先考虑高层语义而非细粒度细节，GAP3D表明VLM与图像编码器特征空间之间的表示差距可以通过基于扩散的对齐部分弥合，朝着通过生成式对齐到密集嵌入空间实现基础模型模块化集成迈出了第一步。

## Abstract
Recent approaches integrating vision-language models (VLMs) as prompt encoders for generative model conditioning typically rely on expensive end-to-end training or map features to compressed representations, discarding the dense spatial structure required for geometry-aware tasks like 3D asset generation. To address this, we propose GAP3D, a modular, diffusion-based approach that aligns VLM-generated latents directly to the complete, patch-level feature space of a pre-trained image encoder, enabling a frozen downstream generative model to utilize a VLM as prompt encoder while maintaining a spatially structured conditioning signal. Evaluated on 3D asset generation, our method bypasses the need for large-scale 3D data by training mainly on general-domain image-text pairs. It also exhibits emergent zero-shot capabilities for multimodal prompts, despite being trained exclusively on text input. Finally, while currently prioritizing high-level semantics over fine-grained detail, GAP3D demonstrates that the representation gap between VLM and image-encoder feature spaces can be partially bridged through diffusion-based alignment, taking the first steps towards a modular integration of foundation models through generative alignment to dense embedding spaces.

## 1. 研究动机与核心问题
当前生成模型需要丰富的条件信号来实现精准控制,但主流文本编码器（如 CLIP、T5）因输入长度限制、全局语义偏重和空间理解缺失,难以处理长提示、细粒度属性绑定与复杂组合结构。视觉‑语言模型（VLMs）具备强大的表示容量与多模态推理能力,然而将其集成到生成流程中存在两种困境：

- **端到端训练方案**：计算开销大,需大量领域数据,且与已有预训练生成模型不兼容。
- **模块化压缩映射方案**：将 VLM 特征对齐到冻结生成模型的输入空间,但通常只映射为**池化全局向量或文本 token**,丢弃了**密集的空间结构**,这严重限制了几何感知任务（如 3D 物体生成）的性能。

因此本文研究一个根本性问题：**能否将 VLM 表示直接映射到冻结图像编码器的完整、高维、块级（patch‑level）特征空间,从而为下游生成模型提供兼具语义推理与空间细节的条件信号？** 这一思路有望将文本‑to‑3D 转化为伪图像‑to‑3D 问题,使冻结的 3D 生成模型专注“3D 提升”,同时保留模块化、可插拔的优势。

## 2. 方法论

### 2.1 核心思想
GAP3D 基于**扩散式生成对齐**,通过流匹配（Rectified Flow）训练一个扩散变换器（DiT）,将 VLM 产生的潜在语义特征直接生成为目标图像编码器 DINOv2 的完整嵌入序列（CLS token、register token 以及全部空间 patch 特征）,再将该密集表示注入冻结的 TRELLIS 图像‑to‑3D 管道,实现文本‑to‑3D 生成。

### 2.2 VLM 编码与可学习软标记
- 在冻结的 VLM（Qwen2.5‑VL‑3B‑Instruct）输入文本前追加 $s$ 个**可训练软标记**。
- 前向传播后提取这些软标记对应的隐藏状态,得到条件信号 $\mathbf{C} \in \mathbb{R}^{s \times D_{\text{VLM}}}$。
- 训练期间 VLM 主干冻结,仅优化软标记,保留 VLM 的推理能力与多模态先验。

### 2.3 目标表示空间
选用**自监督图像编码器 DINOv2（dinov2‑vitl14‑reg）**的完整特征空间,输入图像尺寸 $518 \times 518$,包含：
- 密集 2D patch 网格 $\mathbf{x}^p \in \mathbb{R}^{h \times w \times D_{\text{img}}}$,其中 $h = w = 37$,$D_{\text{img}}=1024$；
- 全局 CLS token $\mathbf{x}^{\text{cls}} \in \mathbb{R}^{1 \times 1024}$；
- 一组 register token $\mathbf{x}^{\text{reg}} \in \mathbb{R}^{4 \times 1024}$。

三种成分联合生成,以保证全局语义与局部空间细节的一致性。

### 2.4 扩散架构与条件注入
- 使用基于 **Lumina‑Next** 的 DiT,36 层,隐藏维度 1792。
- 输入：噪声化的 $\mathbf{x}_t^p, \mathbf{x}_t^{\text{cls}}, \mathbf{x}_t^{\text{reg}}$。
- 对 2D patch 采用线性 patch 嵌入 + 2D RoPE 位置编码；对 CLS 和 register 采用独立线性投影 + 可学习位置嵌入,并赋以恒等旋转因子（不引入空间偏置）。
- **VLM 条件 $\mathbf{C}$ 通过交叉注意力注入** DiT,引导去噪过程生成与语义对齐的视觉特征。

### 2.5 训练目标：Rectified Flow
对每个成分 $k \in \{p, \text{cls}, \text{reg}\}$,定义线性插值路径：
$$\mathbf{x}_t^k = t\mathbf{x}_1^k + (1-t)\mathbf{x}_0^k,\quad t\in[0,1]$$
其中 $\mathbf{x}_0^k \sim \mathcal{N}(0,I)$,$\mathbf{x}_1^k$ 为真实 DINOv2 嵌入。网络 $v_\theta$ 同时预测三种成分的速度场,目标为加权流匹配损失：
$$\mathcal{L}(\theta) = \mathbb{E}_{t,\mathbf{x}_0,\mathbf{x}_1}\left[\sum_k \lambda_k \left\| v_\theta^k(\mathbf{x}_t, t, \mathbf{C}) - (\mathbf{x}_1^k - \mathbf{x}_0^k) \right\|^2 \right]$$
权重设置：$\lambda^p = 0.4$,$\lambda^{\text{cls}} = 0.3$,$\lambda^{\text{reg}} = 0.3$。

### 2.6 下游集成与域自适应
- 直接将生成的 DINOv2 嵌入注入冻结的 TRELLIS **图像‑to‑3D** 管道,替代原编码器,实现文本‑to‑3D。
- 训练分两阶段：
  - **预训练**：在 BLIP3‑o 数据集（约 3100 万通用图文对）上建立语义到视觉特征的基础映射；
  - **微调**：在从 Objaverse‑XL 渲染的 6 万张 3D 资产单视图图像（纯黑背景、单物体）上微调,以缓解自然图像与 3D 渲染之间的分布偏移。

## 6. 主要结论与发现
- **密集空间对齐可行**：通过流匹配,可将 VLM 潜在特征成功映射到 DINOv2 的完整块级空间,且生成的嵌入在方向上接近真实嵌入（patch 余弦相似度 0.67 在 Toys4K 上）,优于 BLIP3‑o 的范数一致性。
- **3D 生成有效**：微调后的模型在 Toys4K 上的生成质量接近 TRELLIS 原生文本‑to‑3D 管道,显著优于直接使用预训练模型或以文本约束替代微调。
- **域自适应至关重要**：从自然图像到 3D 渲染的分布偏移会引入严重伪影（如悬浮背景平面）,微调通过匹配编码空间分布有效缓解。
- **涌现零样本多模态能力**：尽管训练仅用文本,冻结 VLM 的联合表示允许直接接受图像+文本输入进行 3D 编辑,但当前主要扮演语义转换器角色,结构保留有限。
- **细粒度不足**：模型偏向高层语义,低层视觉细节（精确颜色、形状比例）的保留仍存在明显差距。

## 7. 优点
- **首次将 VLM 表示对齐到完整、高维、块级视觉编码空间**,而非压缩语义向量,为下游提供空间结构条件。
- **完全模块化,冻结下游 3D 模型**,无需重新训练生成器,只需训练 VLM 软标记和对齐 DiT。
- **主要依赖通用图文对训练**,避免了昂贵的大规模 3D 数据标注,复用性强。
- **展示了生成式对齐的强大潜力**,即使目标空间未经过跨模态预训练（如 CLIP）,仍能建立有意义的映射。
- **域自适应策略**：两阶段训练有效克服分布偏移,且实验证明单纯提示工程无法替代。

## 8. 不足与局限
- **细粒度几何与外观细节不足**,patch 对齐质量明显低于 CLS,生成资产常缺失提示中的精确属性。
- **微调带来灾难性遗忘**：对自然图像的表示质量大幅下降（MS‑COCO 上余弦相似度从 0.43 降至 0.28）,文中未给出缓解方案。
- **生成质量尚不及原生文本‑to‑3D 基线**,尤其在几何保真度上仍有差距（FD_PointNet++ 44.37 vs 26.54）。
- **多模态能力仅剩语义翻译**,无法严格保留输入图像的结构,训练中未加入图像重建或编辑目标。
- **VLM 软标记容量可能成为瓶颈**,目前使用的可训练 token 数量有限,难以编码足够精细的空间信息。
- 对比基线仅限于压缩对齐的代表性方法（BLIP3‑o）,**未与其他模块化文本‑to‑patch 对齐方法直接比较**（可能因缺乏同类工作）。
