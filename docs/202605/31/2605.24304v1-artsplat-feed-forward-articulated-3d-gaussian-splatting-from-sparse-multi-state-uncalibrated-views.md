## 摘要
从稀疏视图图像重建关节物体是一个不适定问题，需要同时推断几何形状和底层关节结构。现有的基于 NeRF 和三维高斯溅射（3DGS）的关节物体重建方法通常依赖密集视图或强先验（如深度图、关节类型、预定义关节数量），并需要昂贵的逐物体优化。本文提出 ArtSplat，首个用于关节三维高斯溅射的前馈式框架。它能在单次前向传递中，从多个关节状态下的稀疏多视图图像同时重建几何形状和关节参数。为了解决单次前向关节重建的挑战，我们引入了一种逐像素关节图表示，能够将关节参数估计集成到前馈流水线中。我们还提出了一种带有状态令牌的跨状态注意力（CSA）机制，能有效捕获输入状态间的离散运动。在 PartNet-Mobility 数据集上的 68 个关节物体（包括单关节和多关节配置）实验表明，ArtSplat 在几何形状和关节估计方面均取得了有竞争力的性能，同时比基线方法快 400 倍以上。

## Abstract
Articulated object reconstruction from sparse-view images is an ill-posed problem that requires simultaneous inference of geometry and underlying articulation structure. Existing methods for articulated object reconstruction based on NeRF and 3D Gaussian Splatting (3DGS) typically rely on dense views or strong priors (e.g., depth maps, joint types, predefined number of joints) and require costly per-object optimization. In this paper, we propose ArtSplat, the first feed-forward framework for articulated 3D Gaussian Splatting. It reconstructs both geometry and joint parameters from sparse multi-view images across multiple articulation states in a single forward pass. To address the challenges of single-pass articulated reconstruction, we introduce a per-pixel joint map representation that enables the integration of joint parameter estimation into the feed-forward pipeline. We further propose a Cross-State Attention (CSA) mechanism with state tokens, which effectively captures discrete motion across input states. Experiments on 68 articulated objects from PartNet-Mobility, including both single- and multi-joint configurations, demonstrate that ArtSplat achieves competitive performance in both geometry and joint estimation, while being over 400 times faster than baselines.

# ArtSplat：前馈式关节三维高斯溅射重建总结

## 1. 研究动机与核心问题

- **问题定义**：关节物体（如带抽屉的柜子、可开合笔记本）由刚性部件通过运动副连接。要从稀疏多视角图像中同时恢复物体的三维几何（形状）和关节参数（轴、角度等），这是一个高度不适定问题。
- **现有方法局限**：目前基于 NeRF 或 3DGS 的方法通常需要密集视图（每个状态约50+）与强先验，并采用**逐物体测试时优化**，重建一个物体需数十分钟到数小时，难以快速推广到大量新物体。
- **前馈式方法的挑战**：静态场景的前馈式重建无法处理部件运动，而动态场景方法假设连续小位移，不适合关节物体的大幅度、离散状态变化。
- **本文目标**：实现首个**前馈式关节物体重建**，从无标定的稀疏 RGB 图像中，在**单次前向传递**中同时输出三维高斯表征和关节参数，无需逐物体优化。

## 2. 方法论

### 2.1 整体框架
ArtSplat 基于 VGGT 的 Transformer 架构扩展，接收来自两个关节状态（State 0 和 State 1）的共 N = V×S 张无标定 RGB 图像，前向输出相机参数、深度图、高斯参数以及**逐像素关节图 (Joint Map)**，再通过可微关节变换将高斯变换到任意关节状态并渲染。

### 2.2 逐像素关节图表示
- 传统方法依赖部件分割并回归每个部件的关节参数，需要不可微的操作。ArtSplat 提出**11维逐像素关节图**，每个像素预测一组关节参数，直接端到端可微。
- 11通道构成（见表I）：
  - **不变组 (9维)**：关节类型 logits（静态/旋转/移动）、旋转轴方向、轴枢位置。
  - **变分组 (2维)**：状态相关的旋转角度 θ 和位移距离 d。
- 这种方式将关节估计与几何估计统一为密集预测任务。

### 2.3 关节预测模块
由三部分组成：
- **状态令牌 (State Tokens)**：每个关节状态附加一个可学习令牌 `z_s`，与 VGGT 的摄像机令牌类似，通过 backbone 的自注意力和交叉注意力聚合该状态的全图特征。
- **跨状态注意力 (Cross-State Attention, CSA)** ：将 backbone 输出的状态令牌 `z_s^bb` 作为查询，**另一状态**的 patch 特征图作为键和值，计算出增强的状态摘要 `tilde{z}_s`，从而显式捕获两状态间的部件运动。
- **双分支 DPT 解码器**：
  - 不变分支：使用对称池化 `c_inv = (tilde{z}_0 + tilde{z}_1)/2` 作为 FiLM 条件，预测关节类型、轴、枢轴。
  - 变分分支：使用各自 `c_var_s = tilde{z}_s`，预测该状态下的 θ 或 d。
  - 同时将点云 `P_i,s` 通过 MLP 注入特征图，使解码器感知三维几何。

### 2.4 关节变换与部件分组
- **训练时**使用真实部件标签进行监督。
- **推理时**采用 **HDBSCAN** 聚类算法**自动推断部件数量**：旋转关节在 Plücker 线坐标 (a, a×p) 上聚类，移动关节在轴方向 a 上聚类。
- 得到部件级平均参数后，利用 Rodrigues 公式对高斯中心点 μ 和旋转四元数 q 施加刚性变换，将高斯从参考状态变换到目标关节状态。

### 2.5 训练策略
由于直接端到端训练不稳定，采用**两阶段训练**：
- **第一阶段**（80k 迭代）：仅使用直接监督损失：
  - 相机位姿损失 `L_pose`
  - 深度损失 `L_depth`
  - 关节图损失 `L_joint`（交叉熵、L1 轴、垂直距离枢轴、Huber角度/位移）
  - 不变性一致性损失 `L_consist`（强制同一部件在不同状态下预测不变）
  - 部件感知平滑损失 `L_smooth`
- **第二阶段**（40k 迭代）：在此基础上加入渲染损失（RGB MSE + LPIPS），使几何与关节参数联合微调。
- 实验证明两阶段训练明显优于直接联合优化。

## 3. 实验设计

- **输入协议**：所有方法均使用 8 张输入视图（2 状态 × 4 视角），在 12 张新视角和关节状态的目标视图上评估。
- **对比方法**：
  - 逐物体优化方法：PARIS（RGB+关节数）、DTA（RGB-D+关节数）、ArtGS（RGB-D+关节数）、ReArtGS（RGB+关节数）、ScrewSplat（纯 RGB，无关节数）

- **联合重建可行**：ArtSplat 首次证明可在单次前馈推理中从无标定稀疏视图中同时恢复物体几何和关节参数，完全去除逐物体优化。

## 7. 优点与亮点

- **首创性**：第一项将关节参数估计完整融入前馈式 3DGS 框架的工作。
- **优雅的表示**：逐像素关节图避免了不可微的部件分割，实现端到端训练。
- **跨状态注意力设计**：巧妙利用注意力机制建模离散状态间的运动对应，有效提升轴估计精度。
- **部件自动化**：推理时通过 HDBSCAN 聚类自动推断部件数量，无需预定义。
- **两阶段训练**：弥补了直接联合训练的不稳定性，同时第一阶段省去渲染管路，训练更高效。
- **坚实的实验**：多指标、多基线、详细消融，论证充分。

## 8. 不足与局限

- **运动链限制**：当前模型仅处理连接到静态基座的单自由度（转动或移动）关节，不能处理串联运动链（如机器人手臂）。
- **仿真数据依赖**：实验完全基于 PartNet-Mobility 合成渲染数据，未验证在真实世界拍摄图像中的泛化能力。
- **网格提取质量**：由于 3DGS 的离散性和非表面性，重建的高保真网格仍有挑战，文中提及了部分失败案例。
- **小幅度运动模糊**：当两个输入状态间关节运动很小时，关节轴预测容易偏离真实值，导致渲染错误。
- **前馈式固有局限**：重建质量受限于输入视图的覆盖度，对于从未见过的视角或过度遮挡区域可能产生退化。
