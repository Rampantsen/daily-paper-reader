## 摘要
纯视觉占据预测需要从标定的环视图像中恢复语义3D占据场，其中每个视图提供的沿相机射线的观测具有不明确的深度。现有方法已从密集的结构化表示发展到稀疏的高斯图元，从而提高了3D场景表示的效率。然而，高斯学习仍然主要依赖图像域特征，这为体积推理提供了有限的显式几何信息。我们的关键观察是，有效的高斯占据建模不仅需要稀疏图元，还需要更丰富的几何和语义学习线索。本文中，我们提出VGOcc，它从基础模型中学习视觉和几何线索用于高斯建模。VGOcc将这些线索融入图元初始化和细化中，产生一种称为视觉-几何高斯的表示，专为语义占据预测量身定制。具体来说，我们提出视觉-几何高斯生成，从射线深度假设中形成空间平衡的高斯中心，而视觉语义特征初始化图元属性。接下来，我们设计姿态感知特征学习，将基础标记与相机嵌入和标定射线信息相结合。然后，在每个高斯细化阶段，邻域视图的特征在投影的3D位置聚合。最后，高斯解码器利用姿态感知特征细化生成的高斯，并将其渲染为语义占据。在nuScenes上的实验表明，VGOcc在纯视觉3D占据预测中实现了最先进的性能。代码将开源在 https://github.com/JHLin42in/VGOcc。

## Abstract
Vision-only occupancy prediction requires recovering a semantic 3D occupancy field from calibrated surround-view images, where each view provides observations with ambiguous depth along camera rays. Existing methods have progressed from dense structured representations to sparse Gaussian primitives, improving the efficiency of 3D scene representation. However, Gaussian learning still relies primarily on image domain features, which provide limited explicit geometric information for volumetric reasoning. Our key observation is that effective Gaussian occupancy modeling requires not only sparse primitives, but also richer geometric and semantic learning cues. In this paper, we propose VGOcc, which learns visual and geometric cues from foundation models for Gaussian modeling. VGOcc incorporates these cues into primitive initialization and refinement, yielding a representation termed Visual-Geometric Gaussians tailored to semantic occupancy prediction. Specifically, we propose Visual-Geometric Gaussian Birth to form spatially balanced Gaussian centers from ray depth hypotheses, while visual semantic features initialize primitive attributes. Next, we design Pose-Aware Feature Learning to combine foundation tokens with camera embeddings and calibrated ray information. Features from neighboring views are then aggregated at projected 3D locations for each Gaussian refinement stage. Finally, Gaussian decoder refines birth Gaussians with pose-aware features and renders them into semantic occupancy. Experiments on nuScenes demonstrate that VGOcc achieves state-of-the-art performance in vision-only 3D occupancy prediction. Codes will be available at https://github.com/JHLin42in/VGOcc.

## 2. 方法论：核心思想、关键技术细节与算法流程
- **整体框架**（图2）：冻结的VGGT提取多级DINO、frame、global标记；相机参数提供标定射线和姿态。经由可训练的DPT头得到射线深度假设，通过**视觉-几何高斯生成（VGBirth）** 得到初始高斯查询，再通过**姿态感知特征学习（PFLearn）** 产生四个细化阶段的特征，驱动**高斯占据解码器**迭代优化，最终渲染为语义占据场。
- **视觉-几何高斯生成（VGBirth）**：
  - *射线深度假设构造*：对每个图像补丁，DPT头从冻结几何特征预测深度后验（含空类别），每个非空深度仓生成一个3D候选点 \(x_{cib}=o_c + d_b r_{ci}\)，置信度 \(w_{cib}\)，并绑定同一射线的DINO token \(\xi_{ci}\)，保证几何位置与视觉语义的对应。
  - *快速体素稀疏采样*：将候选点分入粗体素，先均匀选择非空体素，再在每个选中体素内按置信度采样，得到 \(M_g\) 个几何引导的高斯中心，避免仅依赖置信度排名导致的过度集中于可见表面的问题；另增加 \(M_p\) 个可学习的回退查询覆盖弱观测区域。
  - *姿态感知属性初始化*：对每个查询中心，聚合其来源射线对应的DINO特征（经PoseGate注入相机射线信息），通过MLP预测不透明度、语义和潜在特征等属性的残差，与可学习基参数相加，完成非空间属性的初始化。
- **姿态感知特征学习（PFLearn）**：
  - *补丁局部姿态调节*：为每个补丁编码射线原点、方向及图像坐标，得到区分不同射线的姿态嵌入 \(t_{ci}\)。
  - *阶段式三流融合*：在每一细化阶段，将DINO、frame、global三种标记分别通过PoseFiLM以 \(t_{ci}\) 调节后，用软门控动态融合，保留互补信息并自适应不同阶段的需求。
  - *姿态引导的跨视图聚合*：在源射线上的度量采样点投影至邻域相机，采样特征时引入射线方向兼容性（点积）作为偏置项，通过交叉注意力聚合特征，显式建模跨视图几何一致性，并消解射线深度模糊。
- **高斯占据解码器**：保持标准稀疏高斯细化流程不变，仅用VGBirth得到的初始查询 \(Q\) 和PFLearn提供的四阶段特征 \(F\) 替换原有初始化与采样特征，最终渲染为语义占据。

## 6. 主要结论与发现
- VGOcc在nuScenes上取得SC IoU 34.07%和SSC mIoU 21.75%，显著超过先前最优的SurroundOcc和GaussianFormer-2，在16类中14类取得最高分数。
- 视觉-几何高斯生成提供的**几何引导中心**与**视觉语义属性初始化**，大幅改善了场景空间覆盖和语义先验质量；快速体素稀疏采样有效避免了高斯过于集中在近景可见表面的问题。
- 姿态感知特征学习通过**补丁级射线编码**和**跨视图几何一致性聚合**，为高斯细化注入了显式的深度与多视角关系信息，尤其提升了遮挡区域和远区的预测完整性。
- 定性结果证实，VGOcc在不同光照和天气条件下均能保持更完整、更清晰的道路、人行道和物体边界，证明了视觉-几何表示对复杂驾驶场景的鲁棒性。

## 7. 优点
- **方法创新**：首次在稀疏高斯占据框架中系统性地融合预训练基础模型的**视觉语义**与**几何深度**线索，提出视觉-几何高斯概念。
- **组件设计精巧**：VGBirth通过射线深度假设和两层采样策略，同时实现了几何合理的高斯分布与稀疏性带来的高效率；PFLearn通过姿态编码和射线兼容性引导的跨视图注意力，显式增强了多视图几何推理。
- **高效且兼容**：高斯解码器保持不变，仅改善输入查询和阶段特征，验证了“更好的初始化与更丰富的特征”相比于纯粹稀疏化能带来显著提升，范式具有良好的可扩展性。
- **实验扎实**：消融实验设计细致，各个组件的增益清晰分离，并展现了补丁级姿态调节和三流融合的至关重要性。

## 8. 不足与局限
- **对精确相机标定的依赖**：方法高度依赖相机内外参用于射线构建和跨视图投影，标定误差可能严重影响性能。
- **计算开销较大**：虽然稀疏高斯降低了场景表示的复杂度，但冻结的VGGT及多级特征提取本身计算量高，限制了在资源受限平台上的实时应用。
- **缺少时序建模**：当前框架仅处理单帧输入，无法利用历史信息来提高对动态物体和被遮挡结构的预测稳健性。

- **未探索其他基础模型与训练策略**：仅使用了VGGT和DINO，且完全冻结，未研究微调或整合其他几何基础模型（如MASt3R）的潜力。
