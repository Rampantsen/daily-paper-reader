## 摘要
开放词汇3D场景理解旨在通过从视觉语言模型迁移语义知识，分割超出预定义类别的3D场景。现有方法通过将语言对齐的2D特征提升到3D推进了这一任务，但它们通常依赖于上下文独立的语义表示，使得对象关系在上下文细化方面未得到充分探索。我们提出RelGraphOV，一个利用3D场景图增强开放词汇3D理解的关系感知框架。我们的方法从多视图观测构建关系场景图，利用视觉语言推理推断对象关系并修剪几何上不合理的连接，无需手动关系标注。为了聚合关系上下文同时避免特征干扰，我们引入了自适应门控双流上下文GAT，它分离密集几何特征和语义CLIP嵌入，执行边引导的消息传递，并自适应地融合互补语义。层次对比目标进一步促进了实例级一致性和类别级区分。在ScanNetV2、ScanNet200、ScanNet++和Replica上的实验展示了强大的性能和泛化能力。项目页面：https://cxavireh.github.io/relgraphov-projectpage

## Abstract
Open-vocabulary 3D scene understanding aims to segment 3D scenes beyond predefined categories by transferring semantic knowledge from vision-language models. Existing methods have advanced this task by lifting language-aligned 2D features into 3D, yet they often rely on context-independent semantic representations, leaving object relationships underexplored for contextual refinement. We propose RelGraphOV, a relationship-aware framework that uses 3D scene graphs to enhance open-vocabulary 3D understanding. Our method constructs relational scene graphs from multi-view observations by leveraging vision-language reasoning to infer object relationships and prune geometrically implausible connections, without manual relationship annotations. To aggregate relational context while avoiding feature interference, we introduce an Adaptive Gated Dual-Stream Contextual GAT that separates dense geometric features and semantic CLIP embeddings, performs edge-guided message passing, and adaptively fuses complementary semantics. A hierarchical contrastive objective further promotes instance-level consistency and category-level discrimination. Experiments on ScanNetV2, ScanNet200, ScanNet$++$, and Replica demonstrate strong performance and generalization ability. Project Page: https://cxavireh.github.io/relgraphov-projectpage

开放词汇 3D 场景理解旨在让模型在无预定义类别限制下，对 3D 场景进行语义分割。现有方法大多将 2D 视觉‑语言特征（如 CLIP、LSeg）提升到 3D，但它们普遍将场景中的物体视为独立实体，忽略了物体之间的结构化关系。这导致在视觉外观相似但上下文不同的对象（例如普通窗帘与浴帘）上容易出现语义混淆。论文的核心思想是：显式建模对象间的关系上下文，可以显著提升开放词汇 3D 语义理解的准确性与鲁棒性。

### 2. 方法论核心思想与技术细节
整体框架 RelGraphOV 由三部组成：关系感知 3D 场景图构建、VLM 驱动的节点标注、以及自适应双流图网络进行上下文特征学习。

- **场景图构建**  
  1. 基于多视图 RGB‑D 序列聚合 2D 掩码得到类别无关的 3D 物体提议（节点），并根据点数和几何属性（体积、法线）滤除背景结构。  
  2. 通过统一评分函数选取每个节点的最优关键帧，提取 LSeg 密集特征 \( f_l \) 和 CLIP 实例特征 \( f_c \) 作为节点初始特征。  
  3. 基于空间邻近性（质心距离、包围盒邻接）初始化候选边。  
  4. 利用视觉‑语言模型（VLM）对候选边进行关系推理，使用 Set‑of‑Mark 提示和思维链要求模型输出非对称描述（如“人坐在椅子上”），无关系则剪枝，并为保留边提取主‑谓‑宾三元组，用 CLIP 文本编码器生成结构化的边特征 \( F_e \)，得到最终场景图 \( SG=(V,E_{refine},F_v,F_e) \)。

- **VLM 节点标注引擎**  
  从多视图分别生成物体描述，用轻量 LLM 裁决核心类别一致性，冲突时以多视图视觉内容为准，最终为每个节点产出高精度文本标注 \( f_i^{anno} \)，作为软监督信号。

- **自适应门控双流上下文图注意力网络（GAT）**  
  为避免密集 LSeg 特征与实例 CLIP 特征在传播中相互干扰，设计双流结构：  
  - 主流：融合几何编码与 LSeg 特征，作为主要上下文聚合通道。  
  - 辅助流：融合几何编码与 CLIP 特征，保留开放词汇先验。  
  网络包含以下关键模块：  
  - **全局上下文引入**：加入虚拟超节点，通过标准 GAT 获取全局特征，再以交叉注意力注入主流。  
  - **边特征引导的消息传递**：在主流和辅助流中，以邻居节点状态为 Value，将边特征拼接至 Key 中，通过交叉注意力实现关系感知的邻居信息筛选。  
  - **自适应门控融合**：主流聚合后，通过可学习的门控机制（公式 3‑4）动态注入辅助流的互补语义，而不污染辅助流本身。  
  - 多层迭代后，主流输出最终上下文感知特征 \( F_{out} \)，辅助流输出 \( F_{aux} \) 仅用于训练。

- **层次对比损失与双对齐学习**  
  - \( \mathcal{L}_{hie} \)：鼓励节点特征与自身标注一致，同时与语义相似的实例聚合，与无关类别分离。通过标注相似度构建软正样本掩码，并引入自对 margin。  
  - 正则损失 \( \mathcal{L}_{reg} \) 和辅助损失 \( \mathcal{L}_{aux} \) 均采用广义双对齐形式（公式 8），在吸收 VLM 标注监督的同时，约束输出不远离初始 LSeg/CLIP 特征，防止灾难性遗忘。总损失为三项的加权和。

- RelGraphOV 在 ScanNetV2 上以 58.4% mIoU、73.4% mAcc 达到最优，在 ScanNet200 上也以 14.5% mIoU 和 26.1% mAcc 明显领先。

- **损失函数设计**：层次对比损失和双对齐机制有效维持类别区分度和跨模态先验。
